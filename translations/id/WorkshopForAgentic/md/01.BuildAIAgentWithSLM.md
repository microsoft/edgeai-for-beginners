## Perlengkapan Anda

### Ollama: Manajer Model AI Anda

[Ollama](https://ollama.com/) seperti Steam untuk model AI. Unduh, jalankan, dan kelola model dengan perintah sederhana.

**Yang membuatnya keren:**
- Satu perintah untuk mengunduh dan menjalankan model apa saja
- Berfungsi di Mac, Windows, Linux
- Otomatis menggunakan GPU Anda jika ada
- Sangat efisien dalam penggunaan memori

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework) adalah tempat Anda membangun agen AI yang dapat:

- ğŸ¤ Bekerja sama dengan agen lain sebagai tim

**Komponen Utama:**
- **Agen**: Asisten AI Anda dengan tugas tertentu
