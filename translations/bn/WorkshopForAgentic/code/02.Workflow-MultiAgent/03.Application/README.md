# পডকাস্ট অ্যাপ্লিকেশন

এআই এজেন্ট ব্যবহার করে পডকাস্ট স্ক্রিপ্ট তৈরি করার জন্য একটি কনসোল অ্যাপ্লিকেশন।

## ব্যবহারের পদ্ধতি

```bash
python podcast_app.py
```

## ওয়ার্কফ্লো

1. **স্বাগতম** - অ্যাপ্লিকেশন ব্যবহারকারীর সাথে অভিবাদন জানায়
2. **বিষয় ইনপুট** - ব্যবহারকারী পডকাস্টের জন্য একটি বিষয় প্রদান করে
3. **সার্চ এজেন্ট** - প্রাসঙ্গিক তথ্য অনুসন্ধান করে
4. **স্ক্রিপ্ট তৈরি এজেন্ট** - একটি পডকাস্ট স্ক্রিপ্ট তৈরি করে
5. **পর্যালোচনা** - ব্যবহারকারী স্ক্রিপ্ট পর্যালোচনা করে অনুমোদন/প্রত্যাখ্যান করে
6. **সংরক্ষণ** - অনুমোদিত স্ক্রিপ্ট `podcast.md` এ সংরক্ষণ করা হয়
