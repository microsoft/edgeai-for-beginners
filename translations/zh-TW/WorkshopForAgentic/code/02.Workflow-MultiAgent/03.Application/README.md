# Podcast Application

一個使用 AI 代理生成播客腳本的主控台應用程式。

## 使用方法

```bash
python podcast_app.py
```

## 工作流程

1. **歡迎** - 應用程式向使用者致意
2. **主題輸入** - 使用者提供播客的主題
3. **搜尋代理** - 搜尋相關資訊
4. **生成腳本代理** - 創建播客腳本
5. **審核** - 使用者審核並批准/拒絕腳本
6. **保存** - 已批准的腳本被保存到 `podcast.md`

## 需求

- Python 3.12+
- agent_framework
- 來自 02.WorkflowDevUI 的所有依賴項
