# Programu ya Podcast

Programu ya consola ya kutengeneza maandishi ya podcast kwa kutumia mawakala wa AI.

## Matumizi

```bash
python podcast_app.py
```

## Mtiririko wa Kazi

1. **Karibu** - Programu inamtambulisha mtumiaji
2. **Ingiza Mada** - Mtumiaji hutoa mada kwa ajili ya podcast
3. **Mwakala wa Utafutaji** - Huutafuta habari zinazofaa
4. **Mwakala wa Kutengeneza Script** - Hutengeneza maandishi ya podcast
5. **Kagua** - Mtumiaji anapitia na kukubali/kukatisha script
6. **Hifadhi** - Script iliyokubaliwa huhifadhiwa kwenye `podcast.md`

## Mahitaji

- Python 3.12+
- agent_framework
- Mategemeo yote kutoka 02.WorkflowDevUI
