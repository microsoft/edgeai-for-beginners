# Podcast 应用程序

一个使用 AI 代理生成播客脚本的控制台应用程序。

## 使用方法

```bash
python podcast_app.py
```

## 工作流程

1. **欢迎** - 应用程序向用户问候
2. **主题输入** - 用户提供播客主题
3. **搜索代理** - 搜索相关信息
4. **生成脚本代理** - 创建播客脚本
5. **审核** - 用户审核并批准/拒绝脚本
6. **保存** - 已批准的脚本保存到 `podcast.md`

## 需求

- Python 3.12+
- agent_framework
- 以及 02.WorkflowDevUI 中的所有依赖项
