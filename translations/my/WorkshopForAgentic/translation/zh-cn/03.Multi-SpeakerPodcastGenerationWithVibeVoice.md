# အပိုင်း ၃: သင့် Podcast ကို အသက်သွင်းပါ 🎤

![lab3](../../../../../translated_images/my/lab3.b009c55c4442250b.png)

## အဆုံးသတ်

သင့်ဟာ ခေါင်းစဉ်ကို လေ့လာပြီးပြီ။ သင့်ရဲ့ စာသားကို ရေးသားပြီးပြီ။ အခုတော့ ထပ်မံ အလှဆင်မယ့် အချိန်ပါ: သင့်စာသားကို သဘာဝအသံဖြင့် တကယ့် Podcast အသံသို့ ပြောင်းပါ!

**VibeVoice** မှာ ဝင်ပါ —— Microsoft Research ရဲ့ open source TTS (စာသားမှအသံသို့ ပြောင်းသော စနစ်) ကိုယ်ပိုင်မျှင့်၊ တည်ဆောက်သည်မှာ -
- 🎭 သဘာဝနှင့် စီးဆင်းမှုရှိသော စကားပြောဆိုခြင်း
- 👥 စကားပြောသူများ များစွာ (အများဆုံး ၄ ဦး!)
- ⏱️ ရှည်လျားသော အသံဖိုင် (အများဆုံး ၉၀ မိနစ်!)
- 🎵 ထူးခြားသော နှစ်သက်ဖွယ် ဖော်ပြမှု (စက်ရုပ်အသံ မဟုတ်ပါ!)

ဒီဟာက podcast ဖန်တီးပုံနောက်က ကွန်ပျူတာနည်းပညာ ဖြစ်ပါတယ်။ သင့် podcast ကို တွဲဖက်လုပ်ကြမယ်။

## VibeVoice ဆိုတာ ဘာလဲ? (အေးဆေးသော အရာ)

VibeVoice သည် Microsoft Research မှ ကမ္ဘာ့အတွက် လက်ဆောင်တစ်ခု ဖြစ်ပါတယ်။ ဒါဟာ podcast စတိုင် ရှိသော စကားသံ အသံဖန်တီးရန် အထူးဒီဇိုင်းပြုလုပ်ထားသည်။

### ဘာကြောင့် ကောင်းတာလဲ 🔥

* **⏱️ မရဏရက်စွဲ-တူ စကားပြောခြင်း**: အများဆုံး ၉၀ မိနစ် ချို႕ယွင်းမှု မရှိဘဲ အသံပြုလုပ်နိုင်သည် (ဒါဟာ podcast အပိုင်း တစ်ခုလုံးပါ!)
* **👥 စကားပြောသူစုံ မော်ဂျစ်**: မတူညီသော အသံ ၄ ခုပေါင်းထဲ အကြောင်းအရာ တစ်ခုထဲ ဆက်လက်ထားရှိသည်
* **⚡ အလွန်အကျွံ ထိရောက်မှု**: 7.5 Hz frame rate ကို သုံးပြီး ကွန်ပျူတာ စွမ်းဆောင်ရည် လျှော့ချသည်
* **🧠 တဟုန်ညှိအသံ**: LLM (context နားလည်မှု) နှင့် diffusion model (သဘာဝအသံဖန်တီးမှု) ပေါင်းစပ်ထားသည်
* **🎭 သဘာဝနှင့် စိမ့်သော စကားပြောခြင်း**: လှည့်ကြားပြောခြင်း၊ ရပ်နားခြင်း နှင့် စကားရဲ့ ထိပ်ထားမှုကို အလိုအလျောက် စီမံသည်

**ဤဆာ: VibeVoice ဟာ သင့်စာသားကို ဖတ်ပေးတာသာ မဟုတ်ပါဘူး —— အစစ်အမှန် လူတစ်ယောက်အတိုင်း *ဇာတ်ကောင်ပြုလုပ်တယ်*။

---

## စမှာ 🚀

**လိုအပ်တာတွေ**:

* 🐍 **Python 3.10+** (သင်သည် အပိုင်း ၁ နှင့် ၂ မှာ ရယူပြီးပြီ)
* 🚀 **uv** (မြန်ဆန်သော Python package များကို စီမံခန့်ခွဲဖို့ — ၎င်းကို တပ်ဆင်ပါမည်)
* 📝 **သင့်စာသား**: အပိုင်း ၂ မှ ထွက်ရှိလာသော `podcast.txt` ဖိုင် (`../03.Application/` တွင်ရှိသည်)

**အထူးအကြံပေးချက်**: ဒီအဆင့်မှာ ပံ့ပိုးထားပြီး အင်တာနက်ကောင်းမွန်မှု လိုအပ်သည်။ ကော်ဖီ တစ်ခွက် မောင်းပါ! ☕

---

## စလိုက်ပါ! အလွယ်တကူ နည်းလမ်း 🎬

ကျွန်တော်တို့ ဒီအချက်ကို အလွန်လွယ်ကူအောင် ပြုလုပ်ထားပါတယ်။ shell script တစ်ခုနဲ့ အားလုံးလုပ်ပါတယ်။

### လုပ်ဆောင်ပုံ

1. **ဖိုင်ကို အကောင်အထည်ဖော်ခွင့်ပြုပါ**:
```bash
chmod +x run_vibe_voice.sh
```

2. **ဖိုင်ကို လည်ပတ်ပါ**:
```bash
./run_vibe_voice.sh
```

3. **ကောလဟာလ စောင့်ဆိုင်းပါ** (ပထမဆုံး အကြိမ်လည်ပတ်မှု အချိန်ပိုတတ်နိုင်သည်)

### မျက်နှာပြင်နောက်ကွယ်မှာ ဖြစ်ပျက်တာများ 🎭

script ဟာ သင့်အလိုအမခံ အသံအင်ဂျင်နီယာလို ဖြစ်နေပါတယ် -

1. **📥 VibeVoice ကို ဒေါင်းလုပ်လုပ်ပါ**: GitHub မှ အရည်အသွေးရှိ repository ကို clone လုပ်သည်
2. **📦 လိုအပ်ချက်တွေအား ထည့်သွင်းပါ**: `uv pip` ဖြင့် အလျင်မြန်ဆုံး package များ တပ်ဆင်သည်
3. **🎬 အသံထုတ်လုပ်ခြင်း**: inference script ကို အသုံးပြုပြီး အောက်ပါအတိုင်း လည်ပတ်သည် -
   * `--model_path`: pre-trained VibeVoice-7B မော်ဒယ်
   * `--txt_path`: သင့် `podcast.txt` စာသားဖိုင်
   * `--speaker_names`: အသံပေးသူများ သတ်မှတ်ခြင်း (မူရင်းမှာ Xinran နဲ့ Anchen ပါ)

**ရလဒ်**: သင့်စာသား ဟာ တကယ့် podcast အပိုင်း ဖြစ်လာပါပြီ! 🎉

---

## သင့်တာဝန် 🎯

ဒါကို ပျော်စရာခြယ်မှုခံစေပါစို့:

### တာဝန် ၁: အကြောင်းအရာ ဖန်တီးပါ
လူ ၂ ယောက်ကြား စကားပြောနှစ်လမ်းသွယ် `../03.Application/podcast.txt` ကို စီမံရေးသားပါ။ ဘာဘာမဆို ရနိူင်ပါသည် - နည်းပညာ၊ သက္ကရာဇ်၊ ဟိုဘီ ဖြစ်ပါစေ! အဓိက စကားပြောပုံ ဖြစ်စေရန်။

**ပုံစံ ဥပမာ**:
```
说话人 1：嘿！你听说新的 AI 模型了吗？
说话人 2：不会吧！告诉我更多！
说话人 1：它叫...
```

### တာဝန် ၂: အသံထုတ်ပါ
script ကို run လုပ်ပါ။ မယုံကြည်မယုံကြည် စကားသံ ဖြစ်လာသည်ကို ကြည့်ပါ။ ပထမဆုံး run လုပ်မှာ အချိန်ပိုလိုစေပါသည် (မော်ဒယ်ဒေါင်းခြင်း အတွက်)။

### တာဝန် ၃: နားထောင်ပြီး ခွဲခြမ်းစိတ်ဖြာပါ
- သဘာဝ အသံဖြစ်လား?
- အသံပေးသူတွေ မတူညီပါသလား?
- စကားပြောလှပစွာ အပြန်အလှန်ပြောပါသလား?
- စက်ရုပ်အသံ ကြုံတွေ့ရသလား?

### တာဝန် ၄: စမ်းသပ်ခြင်း (သတ္တိပြုသူများအတွက်)
`run_vibe_voice.sh` ဖိုင်ကို ပြင်ပြီး `--speaker_names` ကို မတူညီသော အသံ စပ်ဖတ်နည်းများ စမ်းသပ်ပါ။ VibeVoice မှာ မော်ဒယ်ပြင်ပြီး အသံမျိုးစုံ ရှိသည်!

**ဆုမြှင့်စိန်ခေါ်မှု**: စကားပြောသူ ၃ ဦးဖြင့် စကားပြောခြင်း စမ်းကြည့်ပါ! 🎆

## နောက်ထပ်အချက်အလက် 📚

* **🏠 ပရောဂျက် စာမျက်နှာ**: [VibeVoice အဖွင့်ဆိုက်](https://microsoft.github.io/VibeVoice/)
* **🤗 pre-trained models**: [Hugging Face - VibeVoice-7B](https://huggingface.co/vibevoice/VibeVoice-7B)
* **📖 သုတေသနစာတမ်း**: နည်းပညာ အသေးစိတ်လေ့လာလိုသူများအတွက်

> **⚠️ တာဝန်ရှိသော AI အသိပေးချက်**: VibeVoice သည် မာကျည်းတောက်သော နည်းပညာဖြစ်သည်။ ယဉ်ကျေးစွာ အသုံးပြုပါ! Deepfake သို့မဟုတ် မှားယွင်းသည့် အကြောင်းအရာ မဖန်တီးပါနှင့်။ လူများအတွက် အကူအညီဖြစ်စေမည့် အရာ များကို ဖန်တီးပါ။🙏
