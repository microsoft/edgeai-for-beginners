# Act 1: မိတ်ဆက်ပါ သင်၏ AI သုတေသန အကူအညီယူစက် 🤖

![lab1](../../../../translated_images/lab1.1dd5210a7d114e2e.my.png)

## စိန်ခေါ်မှု

သင်သည် သင်၏နောက်တော်ပေါ် နေသော နည်းပညာပေါ့ဒ်ကက် "Future Bytes" ကို စတင်မိတ်ဆက်ချိန်းဆိုနေသည်။ ပထမဆုံးအပိုင်းမှာ နောက်ဆုံး AI တီထွင်မှုအသစ်များအကြောင်း ဖြစ်ပြီး ၂၄ နာရီအတွင်း သင်လုပ်ဆောင်ရမည့်အချက်များမှာ-
1. ခေါင်းစဉ်ကို သုတေသနပြုရန်
2. ယုံကြည်ရသော အချက်အလက်များ ရှာဖွေရန်
3. စိတ်ဝင်စားဖွယ် စာတမ်းရေးရန်
4. သဘာဝကျစွာ အသံထွက်ရန်

**ဇာတ်လမ်းလှည့်ပြောင်းချက်**: သင်တစ်ယောက်တည်း မလုပ်ရပါဘူး။ သင်၏ပထမဆုံး AI အကူအညီယူစက်ကို တည်ဆောက်တော့မည်ဖြစ်ပြီး ၎င်းသည် ညအိပ်ချိန်မလိုအပ်သော သင်၏သမားရေ သုတေသနတွင် မပြတ်တောက်မှ မပျော်ပါ။

## ဘာကြောင့် Small Language Models? (အဖြေ: အလွန်မိုက်သည်)

Small Language Models (SLMs) ကို သင်၏ကွန်ပျူတာပေါ်ရှိ ကိုယ်ပိုင် AI အဖြစ်စဉ်းစားပါ။ မိုဃ်းတောင်မှာမရှိ၊ လစဉ်ကြေးမရှိ၊ အတက်အကျ ဒေတာမျှဝေရေးမရှိပါ။

**SLMs များဘာကြောင့် 🔥:**
- **🏠 သင့်ကွန်ပျူတာပေါ်တွင် တွဲဆောင်နိုင်သည်**: လပ်တော့၊ ဒက်စတော့၊ Raspberry Pi ကြီးတွင်ပါ
- **💸 လစဉ်ကုန်ကျစရိတ်မရှိ**: API ကိစ္စအတွက် ငွေစားမည်မဟုတ်
- **🔒 ကိုယ်ရေးအချက်အလက်ပထမဦးဆုံး**: သင်၏ဒေတာသည် သင့်စက်ကို မပြေးလွှားပါ
- **⚡ အလင်းအမြန်**: အင်တာနက်ချိန်မီမှန်ရာမရှိ၊ ချက်ချင်းဖြေကြားမှု
- **🪦 အလေးချိန်နည်း**: 1B-10B parameter များ၊ ကြီးမားသည့် 100B+ မော်ဒယ်များ နှိုင်းယှဥ်

**ရေပန်းစား SLMs များ**: Qwen 3, Phi-4, Gemma 3 (ဒီအလုပ်ရုံမှာ Qwen ကို သုံးသည်)

## သင်၏ကိရိယာပစ္စည်းများ

### Ollama: သင့် AI မော်ဒယ် မန်နေဂျာ

[Ollama](https://ollama.com/) သည် AI မော်ဒယ်များအတွက် Steam လိုပါ။ မော်ဒယ်များဒေါင်းလုပ်ဆွဲ၊ လည်ပတ်၊ စီမံခန့်ခွဲနိင်သည် အရမ်းလွယ်ကူသော ကမ္မဏီဖြင့်။

**ဘာကြောင့်အရမ်းကြိုက်လဲ:**
- မော်ဒယ်တစ်ခုကို တစ်ချက်သာနဲ့ ဒေါင်းလုပ်၊ သွားဆောင်နိုင်သည်
- Mac, Windows, Linux ပေါ်တွင် အလုပ်လုပ်သည်
- GPU ရှိပါက အလိုအလျောက် အသုံးပြုသည်
- မေမရီအလွန်ထိရောက်သုံးစွဲ

### Microsoft Agent Framework: မိုဃ်းလုံးတွင် လုပ်ဆောင်ချက်များဖြစ်ပေါ်ရာနေရာ

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework) သည် သင်၏ AI အိုင်ဂျင့်များ ဖန်တီးရာနေရာဖြစ်ပြီး -

- 💬 စကားပြောပြီး ပြောဆိုမှုကို မှတ်မိသည်
- 🛠️ အသုံးပြုသူထုတ်ကိရိယာများကို အသုံးပြုသည် (ဝက်ဘ်ရှာဖွေခြင်း၊ မိုးလေဝသ စစ်ဆေးခြင်း)
- 🧠 ပြဿနာခက်ခဲများကို အဆင့်လိုက်တွေးခေါ်ကိုင်တွယ်သည်
- 🤝 အခြားအိုင်ဂျင့်များနဲ့ အဖွဲ့လိုက် အလုပ်လုပ်နိင်သည်
- 🔌 ကွဲပြားသော AI ပံ့ပိုးပေးသူများ (OpenAI, Ollama, Azure) နှင့် ချိတ်ဆက်တယ်

**ဖွဲ့စည်းမှု အခြေခံပစ္စည်းများ:**
- **အိုင်ဂျင့်များ**: အထူးတာဝန်ရှိသည့် သင်၏ AI ကူညီသူများ
- **ကိရိယာများ**: သင်ပေးသည့် အထူးစွမ်းရည်များ
- **မှတ်ဉာဏ်**: စကားပြောခဲ့သည်ကို မမေ့ခြင်း
- **စဉ်းစားတိုင်ပင်ခြင်း**: ကိုယ်တွေးခေါ်နည်း သင်ကြားခြင်း၊ မဟုတ်မှ ဖြေဆိုခြင်းသာမဟုတ်တာ

## သင်၏ လေ့ကျင့်မှု အစီအစဉ်: မစ်ရှင် ၄ ခု

### မစ္ရှင် ၁: ပထမဆုံး အိုင်ဂျင့် ဖန်တီးပါ

📓 [Open Notebook](../code/01.BasicAgent/00.BasicAgent-agent.ipynb)

**ခရီးစဉ်**: သင်၏ပေါ့ဒ်ကတ် စာတမ်းရေးသူ AI တစ်ယောက် Alex ကို တည်ဆောက်မည်။ Alex က နည်းပညာအကြောင်းနှစ်ဦးဆွေးနွေးနေသော တင်ဆက်သူနှစ်ဦးအကြား စကားပြောဆွေးနွေးမှု ဖန်တီးဖို့လိုသည်။

**သင်ယူမည့်အရာများ**:
- AI အိုင်ဂျင့်တစ်ယောက်ကို မနက်ဖြန်မနက်ဖြန်ထရန် လွယ်ကူသည်
- ပုဂ္ဂိုလိကလက္ခဏာနှင့် ညွှန်ကြားချက်ပေးနည်း
- တကယ့် podcast စာတမ်းတွေ ဖန်တီးနိင်ရန်
- ပြန်ပြောစာသားကို နားလည်ခြင်း

**အောင်မြင်မှု ကိန်းတိုင်**: Alex သည် သင့် "Future Bytes" ပထမဆုံး အပိုင်းအတွက် AI အကြောင်း စကားပြောစာတမ်းတစ်ခု ဖန်တီးနိုင်ရန် 🎯

### မစ္ရှင် ၂: Alex ကို စွမ်းအားအထူးပစ္စည်းများပေးပါ (ကိရိယာများ!)

📓 [Open Notebook](../code/01.BasicAgent/01.BasicAgent-tools.ipynb)

**ခရီးစဉ်**: Alex သွင်ပြင်မြင်ကွင်းများမရှိသေး၊ မိုးလေဝသဖြစ်သောအချိန်သိဖို့ မရှိတယ်။ ကိရိယာများ ပေးပြီး ဤပြဿနာကို ဖြေရှင်းကြမည်။

**သင်ယူမည့်အရာများ**:
- ကိုယ်ပိုင် Python function များကို "ကိရိယာ" အဖြစ် ဖန်တီးနိင်ခြင်း
- Alex သည် ဘယ်အချိန် ဘယ်ကိရိယာကို သုံးရမယ် ဆုံးဖြတ်နိုင်ဖို့ ခွင့်ပြုခြင်း
- အလိုအလျောက် ပြဿနာဖြေရှင်းမှု ကြည့်ခြင်း
- နည်းလမ်းပေါင်းစည်းပြီး အခက်အခဲများ ဖြေရှင်းခြင်း

**အောင်မြင်မှု ကိန်းတိုင်**: "Tokyo ရဲ့ မိုးလေဝသ ဘယ်လိုလဲ?" ဘက်မေးလိုက်ရင် Alex မွ အလိုတော်တော် သိနိုင်ရန် ☁️

### မစ္ရှင် ၃: Alex ကို စဉ်းစားတတ်ကောင်းအောင် သင်ကြားပါ

📓 [Open Notebook](../code/01.BasicAgent/02.BasicAgent-reasoning.ipynb)

**ခရီးစဉ်**: Alex အလုပ်လုပ်နည်စနစ် ပြသပါစေ။ ပြဿနာဖြေရှင်းစဉ် ပုံမှန် ဖြေချက်တင်မဟုတ်ပဲ စဉ်းစားနည်းကို မြင်ရပါစေလို့။

**သင်ယူမည့်အရာများ**:
- "စဉ်းစားမှု မုဒ်" ကို ဖွင့်နိုင်ပါမည် (သင်သင်သည့် သင်္ချာတန်းလေးမှ မြင်သလို)
- Alex ၏ အဆင့်လိုက် စဉ်းစား နည်းလမ်းကို ကြည့်ရှုနိင်
- စဉ်လေးတက်ပြောခြင်းနည်းလမ်း နားလည်မှု
- Alex မှားယွင်းတယ်ဆိုရင် ထောက်ပံ့ပြင်ဆင်နည်း

**အောင်မြင်မှု ကိန်းတိုင်**: ခက်ခဲသော သင်္ချာမေးခွန်းတစ်ခွန်း ပေးပြီး Alex ရဲ့ စဉ်းစားမှု ကြည့်ခြင်း 🧠

### မစ္ရှင် ၄: Alex ကို အင်တာနက်သို့ ချိတ်ဆက်ပါ

📓 [Open Notebook](../code/01.BasicAgent/03.BasicAgent-websearch.ipynb)

**ခရီးစဉ်**: Alex ၏ သိမြင်မှုတွင် အချိန်ကန့်သတ်ချက်ရှိသည်။ အမှန်တကယ်အချိန်နှင့်တပြေးညီ အင်တာနက်မှ သတင်းရယူစေလိုက်မည်။

**သင်ယူမည့်အရာများ**:
- ကိုယ်ပိုင် ဝက်ဘ်ရှာဖွေကိရိယာတစ်ခု ဖန်တီးခြင်း
- ပြင်ပ API များ သုံးခြင်း
- ကွန်ယက် အမှားများကို ပိုမိုအဆင်ပြေစွာတုံ့ပြန်နည်း
- Alex ၏သင်ကြားမှု ဒေတာအပြင်က သတင်း ရယူခြင်း

**အောင်မြင်မှု ကိန်းတိုင်**: ဒီနေ့ရဲ့ နည်းပညာသတင်းအကြောင်း မေးပြီး စစ်တမ်းနောက်ဆုံးရလဒ်များ ရယူနိုင်ရန် 📰
