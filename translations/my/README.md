## နိဒါန်း

EdgeAI အစပြုသူများအတွက် သင်တန်းသို့ ကြိုဆိုပါသည် — Edge Artificial Intelligence၏ ပြောင်းလဲစေနိုင်သော လောကထဲသို့ သင့်စုံလင်သော ခရီးစဉ်ဖြစ်သည်။ ဤသင်တန်းသည် အင်အားမြင့် AI အင်အားများနှင့် edge ကိရိယာများပေါ်တွင် လက်တွေ့ သတ်မှတ်ထားသော deployment အတွင်း အတောအတွင်း ဆက်စပ်ပေးကာ ဒေတာ ထွက်ပေါက်ရာနှင့် ဆုံးဖြတ်ချက် ထုတ်ပေးရမည့် အနေတွင် AI ၏ စွမ်းရည်ကို တိုက်ရိုက် အသုံးချနိုင်ရန် အားပေးပါသည်။

### သင် ကျွမ်းကျင်မည့် အရာများ

ဤသင်တန်းသည် အခြေခံ ရှင်းလင်းချက်များမှ စ၍ ထုတ်လုပ်ရေး အသင့် အကောင်အထည် ဖော်နိုင်သော အကောင့်တင်မှုများအထိ သင်ယူပေးပါသည်၊ အဓိကအကြောင်းအရာများမှာ-
- **Small Language Models (SLMs)** - edge ပေါ်တွင် တပ်ဆင် အသုံးပြုရန် အလိုက်အသင့် အဆင့်သတ်မှတ်ထားသော မော်ဒယ်များ
- **ပစ္စည်းအခြေပြု ချိန်ညှိမှု (Hardware-aware optimization)** - မတူညီသည့် ပလက်ဖောင်းများပေါ်တွင် အကောင်းဆုံး အကျိုးထည့်ရန် ချိန်ညှိခြင်း
- **အချိန်နှင့် တပြေးညီ ခန့်မှန်းချက် (Real-time inference)** - ကိုယ်ရေးအချက်အလက် သိုလှုံရာ လုံခြုံမှု ထိန်းသိမ်းထားနိုင်ခြင်းနှင့်တကွ
- **ထုတ်လုပ်ရေး အသင့် ထည့်သွင်းခြင်း (Production deployment)** - စီးပွားရေး အပလီကေးရှင်းများအတွက် ထည့်သွင်းရေး မဟာဗျူဟာများ

### EdgeAI ရဲ့ အရေးပါမှု

Edge AI သည် အရေးပါသော ချိန်လှုပ်ရှားမှုတစ်ခုကို ကိုယ်စားပြုသည်၊ ၎င်းသည် မော်ဒယ် များကို cloud ပေါ် မူတည်စေဘဲ ဒေတာထွက်ပေါက်ရာ နီးစပ်သည့် hardware ပေါ်တွင် တိုက်ရိုက် လည်ပတ်စေခြင်းဖြင့် အောက်ပါပြဿနာများကို ဖြေရှင်းပေးသည်။
- **privacy နှင့် လုံခြုံရေး**: အထူးသက်သာနိုင်သော ဒေတာများကို cloud ထဲ မပို့ဘဲ ဒေသတွင်းတွင် ဆက်လုပ်နိုင်ပါသည်
- **အချိန်နှင့်တပြေးညီ အကောင်အထည်ဖော်မှု**: အချိန်တိုအခါတွင် network ညှင်းစွာဖြစ်သော ကြာချိန်ကို ဖယ်ရှားပေးနိုင်သည်
- **ကုန်ကျစရိတ် သက်သာခြင်း**: bandwidth နှင့် cloud ကွန်ပျူတင်း အသုံးစရိတ် လျှော့ချပေးနိုင်သည်
- **တောင့်တင်းခံနိုင်မှု**: network ဖြတ်တောက်ချိန်များတွင်လည်း လုပ်ဆောင်ချက်များကို ဆက်လက်ထိန်းသိမ်းနိုင်သည်
- **စည်းမျဉ်းစည်းကမ်း လိုက်နာမှု**: ဒေတာ ပိုင်နက်စည်းမျဉ်းစည်းကမ်း တတ်နိုင်စေသည်

### Edge AI

Edge AI သည် AI အယ်လ်ဂိုရဆိုရီများနှင့် ဘာသာစကားမော်ဒယ်များကို cloud ကို မူတည်ဘဲ ဒေတာ ထွက်ပေါက်ရာ နီးစပ်သည့် hardware ပေါ်တွင် တိုက်ရိုက် ပြေးဆောင်ခြင်းကို ရည်ညွှန်းသည်။ ၎င်းသည် latency ကို လျှော့ချပေး၊ privacy ကို တိုးတက်စေပြီး အချိန်နှင့်တပြေးညီ ဆုံးဖြတ်ချက်ချနိုင်စေသည်။

### အခြေခံ နိယာမများ:
- **On-device inference**: AI မော်ဒယ်များကို edge ကိရိယာများ (ဖုန်းများ၊ router များ၊ microcontrollers၊ စက်မှု ကွန်ပျူတာများ)ပေါ်တွင် ရှေ့နေ၍ ပြေးဆောင်သည်
- **Offline capability**: အမြဲတမ်း အင်တာနက် ချိတ်ဆက်မှု မလိုအပ်ဘဲ လည်ပတ်နိုင်သည်
- **Low latency**: အချိန်နှင့်တပြေးညီ စနစ်များအတွက် ကိုက်ညီသော ချက်ချင်း တုံ့ပြန်မှုများ ရရှိစေသည်
- **Data sovereignty**: အထူးသက်သာသည့် ဒေတာများကို ဒေသတွင်းတွင် ထိန်းသိမ်း၍ လုံခြုံရေးနှင့် စည်းကမ်းလိုက်နာမှုကို တိုးတက်စေသည်

### Small Language Models (SLMs)

Phi-4, Mistral-7B, Gemma ကဲ့သို့ SLMs များသည် အကြီးစား LLM များ၏ အကောင်းဆုံး အားသာချက်များကို edge သုံးအတွက် သင့်လျော်အောင် သင်ကြားထားသို့မဟုတ် distilled ပြုလုပ်ထားသော အပုံစံများဖြစ်သည်၊ ၎င်းများသည်-
- **မှတ်ဉာဏ် အသုံးချမှု လျော့နည်းခြင်း**: အကန့်အသတ်ရှိသော edge ကိရိယာမှတ်ဉာဏ်များကို ထိထိရောက်ရောက် အသုံးချနိုင်စေသည်
- **တွက်ချက်မှု တာဝန်လျော့ချပ်ခြင်း**: CPU နှင့် edge GPU ဘက်တွင် အကောင်းဆုံး လည်ပတ်စေရန် အလုပ်လျှော့ချထားသည်
- **စတင်လည်ပတ်ချိန် မြန်ဆန်ခြင်း**: တုံ့ပြန်မှုမြန်သော အပလီကေးရှင်းများအတွက် အမြန် စတင်အလုပ်လုပ်နိုင်ခြင်း

၎င်းတို့သည် အောက်ပါ အကျဉ်းချုပ် အကန့်အသတ်များနှင့် ကိုက်ညီပြီး အင်္ဂါရပ်များ ပြုလုပ်နိုင်စေသည်-
- **Embedded systems**: IoT ကိရိယာများနှင့် စက်မှု ထိန်းချုပ်ရေး ကိရိယာများ
- **Mobile devices**: အော့ဖ်လိုင်းအင်္ဂါရပ်များပါဝင်သည့် စမတ်ဖုန်းများနှင့် တက်ဘလက်များ
- **IoT Devices**: ကန့်သတ်အရင်းအမြစ်ရှိ စနစ်များနှင့် စမတ် စင်ဆာများ
- **Edge servers**: GPU အရင်းအမြစ် ကန့်သတ်ချက်ရှိ ဒေသတွင်း အင်အားပေးယူနစ်များ
- **Personal Computers**: ဒက်စ်တော့ပ်နှင့် လက်ပ်တော့များတွင် တပ်ဆင်အသုံးပြုနိုင်သည့် အခြေအနေများ

ထုတ်လုပ်ရေး အသင့် အကောင်အထည်ဖော်ထားသည့် လက်တွေ့ အလုပ်ရုံ ပစ္စည်းများ အပြည့်အစုံပါဝင်သည်-

- **[အလုပ်ရုံ လမ်းညွှန်](./Workshop/Readme.md)** - သင်ယူရမည့် ရည်ရွယ်ချက်များ၊ မျိုးစုံ အကျိုးရလဒ်များနှင့် အရင်းအမြစ် လမ်းကြောင်းများ အပြည့်အစုံ
- **Python နမူနာများ** (၆ အစည်းအဝေး) - အကောင်းဆုံး လက်တွေ့ကျင့်စဉ်များ၊ error handling နှင့် အသေးစိတ် ရှင်းလင်းချက်များဖြင့် အပ်ဒိတ်လုပ်ထားသည်
- **Jupyter Notebooks** (၈ ခု အပြန်အလှန်) - ပိုင်းခြားလမ်းစဉ်နည်းလမ်းများ၊ အရှိန်နှိုင်းယှဉ်ခြင်းနှင့် စွမ်းဆောင်ရည် ကြီးကြပ်မှုအပါအဝင် အဆင့်ဆင့် လမ်းညွှန်များ
- **အစည်းအဝေး လမ်းညွှန်များ** - အလုပ်ရုံ အစည်းအဝေးတစ်ခုချင်းစီအတွက် အသေးစိတ် markdown လမ်းညွှန်များ
- **အတည်ပြုခြင်း ကိရိယာများ** - ကုဒ်အရည်အသွေး စစ်ဆေးရန် script များနှင့် smoke test များ စစ်ဆေးရန် script များ

**သင် တည်ဆောက်မည့် အရာများ:**
- စတရင်းပြေးသော stream รองမှုပါရှိသည့် ဒေသခံ AI chat အပလီကေးရှင်းများ

## သင် တည်ဆောက်မည့် အရာများ

### 🎯 အဓိက ကျွမ်းကျင်မှုများ
- **Edge AI Architecture**: cloud ပေါင်းစပ်မှုနှင့်အတူ ဒေသအရင်မြှင့်ထားသော AI စနစ်များကို ဒီဇိုင်းဆွဲပါ
- **မော်ဒယ် အကောင်းပြုပြင်ခြင်း**: Edge သို့ တပ်ဆင်ရန် မော်ဒယ်များကို quantize နှင့် compress ပြုလုပ်ခြင်း (အရှိန် 85% တိုးမြင့်, အရွယ်အစား 75% လျော့ချ)
- **ပလက်ဖောင်း မျိုးစုံ တပ်ဆင်ခြင်း**: Windows, mobile, embedded, နှင့် cloud-edge hybrid စနစ်များ
- **ထုတ်လုပ်မှု လည်ပတ်မှုများ**: ထုတ်လုပ်ရေးတွင် Edge AI ကို ကြီးကြပ်ခြင်း၊ တိုးချဲ့ခြင်း နှင့် ထိန်းသိမ်းခြင်း

### 🏗️ လက်တွေ့ ပရောဂျက်များ
- **Foundry Local Chat Apps**: Windows 11 native application နှင့် မော်ဒယ်ပြောင်းလဲနိုင်မှု
- **Multi-Agent Systems**: ချိတ်ဆက်လုပ်ဆောင်မှုများအတွက် အထူးအလုပ်ဆောင်နိုင်သော အေးဂျင့်များနှင့် ညှိနှိုင်းသူ
- **RAG Applications**: local စာရွက်များကို vector search ဖြင့် ဆန်းစစ်လုပ်ဆောင်ခြင်း
- **Model Routers**: အလုပ်တာဝန် အခြေခံ၍ မော်ဒယ်များကို ထက်မြက်စွာ ရွေးချယ်ပေးခြင်း
- **API Frameworks**: streaming နှင့် health monitoring ပါရှိသည့် ထုတ်လုပ်ရေးအသင့် client များ
- **Cross-Platform Tools**: LangChain/Semantic Kernel တွဲဖက်အသုံးပြုနည်း ပုံစံများ
