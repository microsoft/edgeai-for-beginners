## ទិដ្ឋភាពទូទៅ

សម័យនេះស្រាវជ្រាវពីរបៀបយកមូឌែលបើកចំហទៅកាន់ Foundry L## Part 6: Hands-On Exercise

### កីឡាការណ៍: ស្វែងរកមូឌែល និងប្រៀបធៀប

Create your own model evaluation script based on Sample 03:

```cmd
REM create_model_test.cmd
@echo off
echo Model Discovery and Testing Script
echo =====================================

echo.
echo Step 1: List available models
foundry model list

echo.
echo Step 2: Check what's cached
foundry cache list

echo.
echo Step 3: Start phi-4-mini for testing
foundry model run phi-4-mini --verbose

echo.
echo Step 4: Test with a simple prompt
curl -X POST http://localhost:8000/v1/chat/completions ^
  -H "Content-Type: application/json" ^
  -d "{\"model\":\"phi-4-mini\",\"messages\":[{\"role\":\"user\",\"content\":\"Hello, please introduce yourself.\"}],\"max_tokens\":100}"

echo.
echo Model test complete!
```

### ភារកិច្ចរបស់អ្នក

1. **រត់ស្គ្រីប Sample 03**: `samples\03\list_and_bench.cmd`
2. **សាកល្បងមូឌែលខុសៗគ្នា**: សាកល្បងយ៉ាងហោចណាស់ 3 មូឌែលផ្សេងគ្នា
3. **ប្រៀបធៀបប្រសិទ្ធភាព**: កត់សម្គាល់ភាពខុសគ្នានៅលើល្បឿន និងគុណភាពការឆ្លើយតប
4. **រាយការណ៍ករណីសិក្សា**: បង្កើតតារាងប្រៀបធៀបសាមញ្ញ

```cmd
REM Check service status
foundry service status
```

**អង្គចងចាំមិនគ្រប់គ្រាន់:**
- ចាប់ផ្តើមជាមួយមូឌែលតូច (`phi-4-mini`)

**ប្រសិទ្ធភាពយឺត:**

- ពិចារណាផ្ទុកទិន្នន័យលឿនជាង (SSD)

### អនុវត្តល្អបំផុត

1. **ចាប់ផ្តើមតូច**: ចាប់ផ្តើមជាមួយ `phi-4-mini` ដើម្បីផ្ទៀងផ្អែកការតំឡើង

3. **ត្រួតពិនិត្យធនធាន**: តាមដានការប្រើប្រាស់អង្គចងចាំ

## ផ្នែកទី 1: ស្វែងរកមូឌែលជាមួយ Foundry CLI

### បញ្ជារគ្រប់គ្រងមូឌែលមូលដ្ឋាន

foundry CLI ផ្តល់នូវពាក្យបញ្ជា​គ្រោះ​ងាយស្រួលសម្រាប់ស្វែងរកនិងគ្រប់គ្រងមូឌែល:

```cmd
REM List all available models in the catalog
foundry model list

REM List cached (downloaded) models
foundry cache list

REM Check cache directory location
foundry cache ls
```

### រត់មូឌែលដំបូងរបស់អ្នក

ចាប់ផ្តើមជាមួយមូឌែលដែលមានប្រជាប្រិយ និងបានសាកល្បងល្អ ដើម្បីយល់ពីលក្ខណៈប្រតិបត្តិការ:

```cmd
REM Run Phi-4-Mini (lightweight, fast)
foundry model run phi-4-mini --verbose

REM Run Qwen 2.5 7B (larger, more capable)
foundry model run qwen2.5-7b --verbose

REM Run DeepSeek (specialized for coding)
foundry model run deepseek-r1-7b --verbose
```

**Note:** The `--verbose` flag provides detailed startup information, including:
- Model download progress (on first run)
- Memory allocation details
- Service binding information
- Performance initialization metrics

### យល់ដឹងពីប្រភេទមូឌែល

**Small Language Models (SLMs):**
- `phi-4-mini`: លឿន ប្រកශសន៍ ប្រសិទ្ធសម្រាប់ការជជែកទូទៅ
- `phi-4`: កំណែមានសមត្ថភាពខ្លាំងជាង មានការសន្និដ្ឋានល្អជាង

**Medium Models:**
- `qwen2.5-7b`: មានសមត្ថភាពសន្និដ្ឋានល្អ និង context វែង
- `deepseek-r1-7b`: បង្កើតមកអោយសម្រួលសម្រាប់ការបង្កើតកូដ

**Larger Models:**
- `llama-3.2`: មូឌែលបើកចំហចុងក្រោយពី Meta
- `qwen2.5-14b`: សមត្ថភាពសម្រាប់ reasoning ក្រុមហ៊ុន

## ផ្នែកទី 5: ការប៊េន្មាកមូឌែលសាមញ្ញ

### ការសាកល្បងប្រសិទ្ធភាពមូលដ្ឋាន

នេះជាវិធីសាមញ្ញដើម្បីប្រៀបធៀបប្រសិទ្ធភាពមូឌែល:

```python
# simple_bench.py - ផ្អែកលើលំនាំ Sample 03
import time
import requests
import json

def test_model_response(model_name, prompt="Explain edge AI in one sentence."):
    """Test a single model with a prompt and measure response time."""
    start_time = time.time()
    
    try:
        response = requests.post(
            "http://localhost:8000/v1/chat/completions",
            headers={"Content-Type": "application/json"},
            json={
                "model": model_name,
                "messages": [{"role": "user", "content": prompt}],
                "max_tokens": 64
            },
            timeout=30
        )
        
        elapsed = time.time() - start_time
        
        if response.status_code == 200:
            result = response.json()
            return {
                "model": model_name,
                "latency_sec": round(elapsed, 3),
                "response": result["choices"][0]["message"]["content"],
                "status": "success"
            }
        else:
            return {
                "model": model_name,
                "status": "error",
                "error": f"HTTP {response.status_code}"
            }
            
    except Exception as e:
        return {
            "model": model_name,
            "status": "error", 
            "error": str(e)
        }

# សាកល្បងម៉ូឌែលដែលកំពុងដំណើរការ
if __name__ == "__main__":
    # សាកល្បងជាមួយម៉ូឌែលផ្សេងៗ (ចាប់ផ្ដើមម៉ូឌែលនិមួយៗជាមុន)
    test_models = ["phi-4-mini", "qwen2.5-7b", "deepseek-r1-7b"]
    
    print("Model Performance Test")
    print("=" * 50)
    
    for model in test_models:
        print(f"\nTesting {model}...")
        print("Note: Make sure this model is running first with 'foundry model run {model}'")
        
        result = test_model_response(model)
        
        if result["status"] == "success":
            print(f"✅ {model}: {result['latency_sec']}s")
            print(f"   Response: {result['response'][:100]}...")
        else:
            print(f"❌ {model}: {result['error']}")
```
