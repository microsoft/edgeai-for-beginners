# Chapter 06 : бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбҹҶбһҺбһ„бҹӢбһҸбһјбһ… SLM: бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ

бһ‘бҹҒбһҹбһ—бһ¶бһ–бһ“бҹғбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҖбҹҶбһ–бһ»бһ„бһҮбһҪбһ”бһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбһҪбһҷбһ–бҹ’бһҡбҹ„бҹҮбһҷбһҫбһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ–бһёбһ”бһ¶бһҸбһҹбҹ’бһ”бҹӮбһҖбһ’бһҳбҹ’бһҳбһҸбһ¶бһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҠбҹ„бһҷбһҹбҹҶбһҺбһ„бҹӢбһ—бһ¶бһҹбһ¶бһҸбһјбһ… (SLMs)бҹ” бһҳбҹҒбһҡбҹҖбһ“бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһ“бҹҒбҹҮбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһ”бһёбһ”бһ¶бһҸбһ»бһҖбһҳбҹ’бһҳбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбҹҶбһҺбһ„бҹӢбһҸбһјбһ…бһҹбһҳбҹҗбһҷбһ‘бҹҶбһ“бһҫбһ”бҹ– бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ function calling бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Model Context Protocol (MCP) бһҠбҹҸбһ”бҹ’бһӣбҹӮбһҖбҹ—бҹ”

## [бһ•бҹ’бһ“бҹӮбһҖбһ‘бһё 1: бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ бһ“бһ·бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбҹҶбһҺбһ„бҹӢбһ—бһ¶бһҹбһ¶бһҸбһјбһ…](./01.IntroduceAgent.md)

бһ•бҹ’бһ“бҹӮбһҖбһҠбҹҶбһ”бһјбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһўбҹҶбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ бһ“бһ·бһ„бһҹбҹҶбһҺбһ„бҹӢбһ—бһ¶бһҹбһ¶бһҸбһјбһ… бһҠбһ¶бһҖбҹӢбһҶбҹ’бһ“бһ¶бҹҶ бҹўбҹ бҹўбҹҘ бһҮбһ¶бһҶбҹ’бһ“бһ¶бҹҶбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҹбһҳбҹҗбһҷ chatbot бһҖбҹ’бһ“бһ»бһ„бһҶбҹ’бһ“бһ¶бҹҶ бҹўбҹ бҹўбҹЈ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбһҫбһ“бһЎбһҫбһ„бһҹбһҳбҹҗбһҷ copilot бһҖбҹ’бһ“бһ»бһ„бһҶбҹ’бһ“бһ¶бҹҶ бҹўбҹ бҹўбҹӨбҹ” бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһҺбҹӮбһ“бһ¶бҹҶбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ **бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҹбҹҠбһё AI** бһҠбҹӮбһӣбһӮбһ·бһҸ бһӮбһјбһҹбһ”бҹ’бһӣбһ„бҹӢ бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹ„бһҷбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҳбһ“бһ»бһҹбҹ’бһҹбһҸбһ·бһ…бһҸбһҪбһ…бһ”бҹҶбһ•бһ»бһҸбҹ”

### бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбҹӮбһӣбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбҹ–
- **бһҹбҹҠбһ»бһҳбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ**бҹ– бһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһҸбҹӢбһ”бҹӮбһ“бһ’бһҳбҹ’бһҳбһҸбһ¶бһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ”бҹ’бһҡбһҫбһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһӮбһҺбһ“бһ¶бһҸбҹҶбһҺбһ¶бһ„бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бҹ”
- **бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ SLM**бҹ– бһ”бһ„бҹ’бһ бһ¶бһүбһҹбҹҶбһҺбһ„бҹӢбһ—бһ¶бһҹбһ¶бһҸбһјбһ…бһҮбһ¶бһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҸбһ·бһ…бһҮбһ¶бһ„ бҹЎбҹ  бһ–бһ¶бһ“бҹӢбһӣбһ¶бһ“бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ…бһҳбҹ’бһҡбһјбһ„бһ…бһҳбҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбҹ”
- **бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ GGUF, бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ”бһҡбһ·бһҳбһ¶бһҺбһӮбһ»бһҺ (Q4_K_M, Q5_K_S, Q8_0) бһ“бһ·бһ„бһҹбҹҠбһ»бһҳбһҖбһҳбҹ’бһҡбһ·бһҸбһӣбһҫбһҖбһҳбһҖ Llama.cpp бһ“бһ·бһ„ Apple MLXбҹ”
- **бһҖбҹҶбһҺбһҸбҹӢбһҸбһ»бһӣбҹ’бһҷ SLM бһҮбһ¶бһҳбһҪбһҷ LLM**бҹ– бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ бҹЎбҹ -бҹЈбҹ  бһҠбһ„бһҮбһ¶бһҳбһҪбһҷ SLM бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ…бҹҶбһ–бҹ„бҹҮ бҹ§бҹ -бҹЁбҹ % бһ“бҹғбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”

бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһҗбҹ’бһ„бҹғбһ“бҹҒбҹҮбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ Ollama, VLLM бһ“бһ·бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ edge бһҡбһ”бһҹбҹӢ Microsoft бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ SLMs бһҮбһ¶бһўбһ“бһ¶бһӮбһҸбһ“бҹғбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҠбҹӮбһӣбһӮбҹ„бһҡбһ–бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҹбһҳбһҡбһҳбҹ’бһҷбҹ”

## [бһ•бҹ’бһ“бҹӮбһҖбһ‘бһё 2: бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ„бҹӢбһ—бһ¶бһҹбһ¶бһҸбһјбһ…](./02.FunctionCalling.md)

бһ•бҹ’бһ“бҹӮбһҖбһ‘бһёбһ–бһёбһҡбһҮбҹ’бһҡбһ¶бһ”бһҮбҹ’бһҡбҹ„бҹҮбһӣбһҫ **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ** бһҠбҹӮбһӣбһҮбһ¶бһӮбһ“бҹ’бһӣбһ„бһ”бҹҶбһӣбҹӮбһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ—бһ¶бһ–бһҗбҹҒбһҡбһҗбҹ’бһҳбһёбһ‘бҹ…бһҮбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ–бһ·бһ—бһ–бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бҹ” бһҖбһ¶бһҡбһңбһ·бһӣбһҸбҹ’бһҡбһЎбһ”бҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһӣбҹҶбһўбһҡбһ–бһёбһҖбһ¶бһҡбһҡбһҖбһҳбһҪбһҷбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„ бһҠбһӣбҹӢбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ…бһҳбҹ’бһӣбһҫбһҷбҹ”

### бһҸбҹҶбһ”бһ“бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бҹ–
- **бһҹбһјбһҳбҹ’бһ”бһёбһң Workflow бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҡбһ”бҹҖбһ”**бҹ– бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһӣбһҳбҹ’бһўбһ·бһҸбһ–бһёбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡ бһҖбһ¶бһҡбһҡбһҖбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„ бһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһ…бһҳбҹ’бһҡбһ¶бһүбҹӢ JSON бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбҹ’бһҡбҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ”
- **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбҹҒбһ‘бһ·бһҖбһ¶бһ–бһ·бһҹбҹҒбһҹбҹ—**бҹ– бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Phi-4-mini бһҮбһ¶бһҳбһҪбһҷ Ollama, бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ Qwen3 бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Microsoft Foundry Localбҹ”
- **бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ бһҖбһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“, бһҮбһҳбҹ’бһҡбһҫбһҹбһ§бһ”бһҖбһҡбһҺбҹҚбһҢбһ¶бһүбҹүбһ¶бһҳбһ·бһ… бһ“бһ·бһ„бһҖбҹ’бһ”бһҪбһ“бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹбһҠбһјбһ…бһҮбһ¶бҹ”
- **бһҖбһ¶бһҡбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһӣбһҫбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ**бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бһҖбһ¶бһҡбһ–бһ¶бһҡбһўбһҸбҹ’бһҡбһ¶ бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» audit, бһңбһ·бһ’бһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһ¶бһҡбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ…бҹҶбһҺбһ»бһ…бһ‘бһҹбҹ’бһҹбһ“бҹҲбһ‘бһ¶бҹҶбһ„бһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһё бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҡбһ№бһ„бһҳбһ¶бҹҶ бһўбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһҖбһ¶бһҡбһ бҹ… API бһҹбһ¶бһҳбһүбҹ’бһү бһ“бһ·бһ„бһ•бҹ’бһ“бҹӮбһҖбһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

## [бһ•бҹ’бһ“бҹӮбһҖбһ‘бһё 3: бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Model Context Protocol (MCP)](./03.IntroduceMCP.md)

бһ•бҹ’бһ“бҹӮбһҖбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҺбҹӮбһ“бһ¶бҹҶ **Model Context Protocol (MCP)** бһҠбҹӮбһӣбһҮбһ¶бһҹбҹҠбһ»бһҳбһҳбһ¶бһ“бһ—бһ¶бһ–бһ”бҹ’бһӣбҹӮбһҖбһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бҹҖбһ”бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹ’бһңбһҫбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһҡбҹ… бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ MCP бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ’бһ–бһ¶бһ“бһҡбһңбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣ AI бһ“бһ·бһ„бһ–бһ·бһ—бһ–бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҸбһ¶бһҳбһҡбһҷбҹҲбһ–бһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ”

### бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбҹ–
- **бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ–бһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ**бҹ– бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһ“бҹӢбһҮбһ¶бһ“бҹӢбһ“бҹ…бһӣбһҫбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһўбһҸбһ·бһҗбһ·бһҮбһ“ LLM бһўбһҸбһ·бһҗбһ·бһҮбһ“ MCP бһ“бһ·бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”
- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡBackend бһ…бҹ’бһҡбһҫбһ“**бҹ– бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһҸбҹӢбһ”бҹӮбһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡ Ollama (бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ) бһ“бһ·бһ„ vLLM (бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ)бҹ”
- **бһ–бһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ**бҹ– бһҡбһ”бҹҖбһ” STDIO бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ“бһ·бһ„бһҡбһ”бҹҖбһ” SSE бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹӮбһҖбһ…бһ¶бһҷбһҸбһ¶бһҳ HTTPбҹ”
- **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бһ·бһ—бһ–бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**бҹ– бһҹбҹ’бһңбҹҗбһҷбһҖбһҳбҹ’бһҳбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ бһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ API бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһ бһ»бһҹбһҹбҹҶбһҡбһ¶бһ”бҹӢбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбҹ”

бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ MCP бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ SLMs бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһҡбҹ… бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ–бҹҒбһүбһҖбһ¶бһҡбһҒбҹ’бһңбҹҮбһҒбһ¶бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһ“бһҪбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҸбһјбһ…бҹ—бһҡбһ”бһҹбҹӢбһңбһ¶ бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһҡбһҖбҹ’бһҹбһ¶бһӣбһҖбҹ’бһҒбһҺбҹҲбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ“бҹғбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ бһ“бһ·бһ„бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ’бһ“бһ’бһ¶бһ“бҹ”

## бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ

бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ бһ”бһёбһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҠбһ»бһҳбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбҹҶбһҺбһ„бҹӢбһҸбһјбһ… SLMбҹ” бһҖбһ¶бһҡбһңбһ·бһңбһҢбҹ’бһҚбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһҸбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ MCP бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһӣбһјбһңбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ‘бҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһҸбҹӢбһ…бҹӮбһ„бһҮбһ¶бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһҠбҹӮбһӣбҹ–

- **бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһ”бһјбһҡбһ”бҹӮбһ”** бһҠбҹ„бһҷбһҳбһ¶бһ“бһҳбҹүбһјбһҠбҹӮбһӣбһҸбһјбһ…бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ
