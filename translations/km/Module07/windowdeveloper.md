# бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ Windows Edge AI

## бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһ•бҹ’бһҸбһҫбһҳ

бһҹбһјбһҳбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҳбһҖбһҖбһ¶бһ“бҹӢ Windows Edge AI Development - бһӮбҹ’бһҡбһҝбһ„бһ…бһҖбҹ’бһҡбһ…бһҳбҹ’бһҡбһ»бҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҹбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһүбҹ’бһүбһ¶бһ”бҹ’бһҡбһҺбһ·бһҸбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣ AI бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶ Windows AI Foundry бһҡбһ”бһҹбҹӢ Microsoftбҹ” бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ Windows бһҠбҹӮбһӣбһ…бһ„бҹӢбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– Edge AI бһҮбһ·бһҸбһҹбҹ’бһ“бһ·бһҸбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒ бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбҹ’бһҖбһ¶бһ”бһҡбһҗбһҷбһ“бҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚ Windows бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбҹ”

###бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ Windows AI

Windows AI Foundry бһӮбһәбһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбҹӮбһӣбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ, бһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһёбһңбһ·бһҸбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ AI бһ–бҹҒбһүбһӣбҹҒбһү - бһ…бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹ…бһҠбһӣбҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҹбһӮбҹ’бһ“бһ¶бһ“бҹ…бһӣбһҫ CPU, GPU, NPU бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ–бһ–бһҖбһҖбҹ„бһҹбһ·бһҖбһ¶бһҖбһ¶бһҸбҹӢбһ•бҹ’бһҹбҹҶбҹ” бһңбҹҒбһ‘бһ·бһҖбһ¶бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ AI бһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбһ¶бһҖбһҹбһҳбһ“бһ№бһ„бһҳбһ“бһ»бһҹбҹ’бһҹбһ‘бһјбһ‘бҹ…бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бҹ–

- **бһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ№бһ„**бҹ– бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһӣбһҫбһҹбҹҠбһёбһӣбһёбһҖбһ»бһ“ AMD, Intel, NVIDIA бһ“бһ·бһ„ Qualcomm
- **бһ”бһүбҹ’бһүбһ¶бһ”бҹ’бһҡбһҺбһ·бһҸбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ**бҹ– AI бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡвҖӢбһҜбһҖбһҮбһ“бһ—бһ¶бһ–бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһҡбһ¶бһҖбҹӢ
- **бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң**бҹ– бһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҗбһ¶бһҳбһ–бһӣ Windows
- **бһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ**бҹ– бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһӣбҹ„бһҳбһҮбһ¶бһӮбһ»бһҺбһңбһ·бһ”бһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһҖбһҫбһ„

### Windows ML
Windows Machine Learning (ML) бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ C#, C++, бһ“бһ·бһ„ Python бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ ONNX AI бһҖбҹ’бһ“бһ»бһ„бһҠбҹғбһӮбһјбһҡбһҠбҹ’бһӢбһ¶бһҖбһҡ Windows бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ ONNX Runtime бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ•бҹ’бһ“бҹӮбһҖбһ”бһүбҹ’бһ…бҹҒбһүбһҖбһ¶бһҡбһҡбһҸбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ— (CPU, GPU, NPU)бҹ” [ONNX Runtime](https://onnxruntime.ai/docs/) бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһ–бһё PyTorch, Tensorflow/Keras, TFLite, scikit-learn бһ“бһ·бһ„бһҹбҹҠбһ»бһҳбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

![WindowsML A diagram illustrating an ONNX model going through Windows ML to then reach NPUs, GPUs, and CPUs.l](https://learn.microsoft.com/en-us/windows/ai/images/winml-diagram.png)

Windows ML бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһ…бһҳбҹ’бһӣбһ„бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ Windows бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһ“бҹғ ONNX Runtime бһӣбһҫбһҹбһ–бһёбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ‘бһ¶бһүбһҷбһҖбһ—бһ¶бһҹбһ¶бһўбһ“бһ»бһңбһҸбҹ’бһҸ (EPs) бһҠбҹ„бһҷ ОҙП…ОҪОұОјО№ОәО¬бҹ”

### бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ Windows бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Edge AI?

**бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҖбһӣ**
Windows ML бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһҸбһ¶бһҳбһҡбһҷбҹҲбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Windows бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһӮбҹ’бһҡбһ”бҹӢбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҹбҹҠбһёбһӣбһёбһҖбһ»бһ“бҹ”

**бһҳбҹүбһјбһ‘бҹҗбһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ AI бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ**
бһҳбҹүбһјбһ‘бҹҗбһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Windows ML бһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӣбһ»бһ”бһ”бҹҶбһ”бһ¶бһҸбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ–бҹ’бһҡбһҪбһҷбһ”бһ¶бһҡбһҳбҹ’бһ—бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбҹ”

**бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ Copilot+ PC**
API бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ Windows бһҮбҹҶбһ“бһ¶бһ“бҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“ NPU бһҠбһ¶бһҖбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһҗбһ¶бһҳбһ–бһӣбҹ”

**бһ”бһҡбһ·бһ”бһјбһҺбҹҢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ**
бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бҹҒбһүбһӣбҹҒбһү бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„ Visual Studio, бһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһү бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹҶбһҡбһјбһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһңбһҠбҹ’бһҸбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбҹ”

- бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ

- бһ•бҹӮбһ“бһҖбһ¶бһҡбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ

**бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI**
- бһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ•бҹ’бһҹбҹҶ бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҡбһңбһ¶бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢ бһ“бһ·бһ„бһ–бһ–бһҖ
- бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһўбҹҶбһ–бһёбһҸбһҳбҹ’бһӣбҹғбһ”бҹ’бһҠбһјбһҡбһҡбһңбһ¶бһ„бһ‘бҹҶбһ бҹҶбһҳбҹүбһјбһҠбҹӮбһӣ, бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһӣбҹ’бһ”бһҝбһ“бһ“бҹғбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳ
- бһ•бҹӮбһ“бһ…бһҡбһ“бҹ’бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҗбҹӮбһҡбһ‘бһ»бһҖбһ—бһ¶бһ–бһҜбһҖбһҮбһ“бһҒбһҺбҹҲбһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһүбҹ’бһүбһ¶
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҗбҹ„бһҖ бһ“бһ·бһ„бһўбһ¶бһ…бһҖбһҫбһ“бһЎбһҫбһ„бһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ

**бһ‘бһёбһҸбһ¶бҹҶбһ„бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡ**
- бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ•бҹ’бһ“бҹӮбһҖбһ”бҹ’бһҡбһҖбһҪбһҸбһ”бҹ’бһҡбһҮбҹӮбһ„бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҠбһҫбһҳ Windows
- бһҖбҹҶбһҺбһҸбҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһҠбҹӮбһӣ AI бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҸбһӣбҹӢбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһӣбҹ’бһўбһҮбһ¶бһ„бһҳбһ»бһ“
- бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бҹ…бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Windows бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ AI

**Windows бһҖбһ¶бһҡбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҡбһјбһ”бһ—бһ¶бһ– APIs**
- **бһҖбһ¶бһҡбһҹбҹ’бһӮбһ¶бһӣбҹӢбһўбһҸбҹ’бһҗбһ”бһ‘ (OCR)**бҹ– бһҠбһҖбһўбһҸбҹ’бһҗбһ”бһ‘бһ–бһёбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹ„бһҷбһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҒбҹ’бһ–бһҹбҹӢ
- **бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһ№бһ„бһҡбһјбһ”бһ—бһ¶бһ–**бҹ– бһҖбҹӮбһӣбһҳбҹ’бһўбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ
- **бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҡбһјбһ”бһ—бһ¶бһ–**бҹ– бһҖбҹҶбһҺбһҸбҹӢ бһ бһҫбһҷбһҠбһҖбһңбһҸбҹ’бһҗбһ»бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһ—бһ¶бһ–
- **бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҡбһјбһ”бһ—бһ¶бһ–**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ–бһ бһ»бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“

**бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ–бһ бһ»бһҡбһ”бҹҖбһ”**

- **бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҸбһ¶бһҳбһ”бҹ’бһҡбҹӮбһҹбһҳбҹҗбһҷ**бҹ– бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҳбһ¶бһ“бһҹбҹҶбһҺбһҪбһҡбһҮбһ¶бһ—бһ¶бһҹбһ¶бһӮбҹ’бһҡбһ”бҹӢбһ”бҹӮбһ”бһ”бһ‘бһҖбҹ’бһ“бһ»бһ„бһҒбһҪбһ”бһ•бҹ’бһҹбҹҒбһ„бҹ—
- **бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ

### 2. Foundry Local

Foundry Local бһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ“бһјбһңбһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһӣбһҝбһ“бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһ”бҹ’бһҡбһ—бһ–бһ”бһҫбһҖбһ“бҹ…бһӣбһҫбһҹбҹҠбһёбһӣбһёбһҖбһ»бһ“ Windows бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҮбһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ»бһҖбһҡбһҖ, бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„, бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбҹ”

### бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбһ¶бһ“бһҖбһ¶бһҡбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһӣбһҫ Hardware

**бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ…бһ¶бһҖбҹӢбһ бҹ’бһҹбһ¶бһҹ (Caching)**
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ…бһ¶бһҖбҹӢбһ бҹ’бһҹбһ¶бһҹбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһ¶бһҮбҹ’бһүбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһўбһ¶бһҷбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢ
- бһҡбһ…бһ“бһ¶бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбһ¶бһҡбһ”бһҠбһ·бһҹбҹҒбһ’бһ…бһ¶бһҖбҹӢбһ бҹ’бһҹбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ
- бһ”бҹ’бһҡбһҫбһ…бһ¶бһҖбҹӢбһ бҹ’бһҹбһ¶бһҹбһ”бһ“бҹ’бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҡбһ”бһҹбҹӢ preprocessing бһҠбҹӮбһӣбһҗбҹ’бһӣбҹғбһҗбҹ’бһ“бһјбһҡ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бһ¶бһҖбҹӢбһ бҹ’бһҹбһ¶бһҹбһ…бҹӮбһҖбһ…бһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ бҹҒбһҸбһ»бһҖбһ¶бһҡбһҺбҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“

## бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ¶бһҡвҖӢбһҜбһҖбһҮбһ“бһ—бһ¶бһ–бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ

### бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

**бһҖбһ¶бһҡбһӣбҹҶбһ бҹӮбһҡбһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ**
- бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹҶбһ„бҹҶбһҳбһ·бһ“бһ…бҹҒбһүбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“
- бһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– Windows бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ sandboxing бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ¶бһ„бһҖбһҳбҹ’бһҡбһ·бһҸ

**бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣ**
- бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҳбһ»бһ“бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
- бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ»бҹҮбһ бһҸбҹ’бһҗбһӣбҹҒбһҒбһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһ¶бһҡбһҖбһ¶бһҡбһ”бҹҶбһ–бһ¶бһ“
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҒбҹ’бһ“бһ¶бһҸбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

### бһҖбһ¶бһҡбһ”бҹҶбһ–бҹҒбһүбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’

**бһҖбһ¶бһҡбһўбһ“бһ»бһӣбҹ„бһҳбһ”бһ‘бһ”бһүбҹ’бһҮбһ¶**
- бһҡбһ…бһ“бһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһұбҹ’бһҷбһҶбҹ’бһӣбһҫбһҷбһҸбһ” GDPR, HIPAA бһ“бһ·бһ„бһ”бһ‘бһ”бһүбҹ’бһҮбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһё audit бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһҳбҹ’бһҡбҹҒбһ… AI
- бһ•бҹ’бһҸбһӣбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҹбһ¶бһҡбһ”бһ¶бһ“бһ…бҹҒбһүбһҖбҹ’бһ“бһ»бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣ AI
- бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ AI

**бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ**
- бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһӮбҹ„бһӣбһ“бһҷбҹ„бһ”бһ¶бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– Windows бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһ¶бһҳбһҸбһҪбһ“бһ¶бһ‘бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ AI
- бһ•бҹ’бһҠбһӣбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһҠбҹ’бһӢбһ”бһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ AI

## бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ бһ“бһ·бһ„ бһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҖбҹҶбһ бһ»бһҹ

### бһ”бһүбҹ’бһ бһ¶бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбһ»бһҒ

**бһ”бһүбҹ’бһ бһ¶бһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһҹбҹҶ Build Configuration**
- бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“ ARM64 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Windows AI API
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ Windows App SDK бһҖбҹҶбһҺбҹӮбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң (1.8.1+ бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ)
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҖбһүбҹ’бһ…бһ”бҹӢбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң (бһҸбһҳбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Windows AI APIs)
- бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ§бһ”бһҖбһҡбһҺбҹҚ Build бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбҹҶбһҺбҹӮ Framework бһӮбҹ„бһӣбһҠбҹ…

**бһ”бһүбҹ’бһ бһ¶бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣ**
- бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҳбҹүбһјбһҠбҹӮбһӣ ONNX бһҮбһ¶бһҳбһҪбһҷ Windows ML
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹүбһјбһҠбҹӮбһӣ
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– hardware бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
- бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣ
- бһ”бҹ’бһҡбһ¶бһҖбһҠбһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“ execution provider бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӣбһҝбһ“ hardware

**бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҳбҹүбһјбһҠбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ**
- **бһҳбҹүбһјбһҠ Self-Contained**: бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ–бҹҒбһүбһӣбҹҒбһүбһҮбһ¶бһҳбһҪбһҷбһ‘бҹҶбһ бҹҶбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ’бҹҶ
- **бһҳбҹүбһјбһҠ Framework-Dependent**: бһ‘бҹҶбһ бҹҶбһҸбһјбһ… бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бһ бһ» runtime
- **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһ·бһ“бһ”бһ¶бһ“бһ…бһ„ Package**: бһҳбһ·бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Windows AI APIs
- бһ”бҹ’бһҡбһҫ `dotnet run -p:Platform=ARM64 -p:SelfContained=true` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһ…бҹҒбһү ARM64 бһ”бҹ’бһҡбһўбһ”бҹӢбһҒбҹ’бһ‘бһ¶бһҸбҹӢбһҒбҹ’бһҳбҹ…

**бһ”бһүбҹ’бһ бһ¶бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ**
- бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ API бһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһҺбһ»бһ…бһҖбҹҶбһ бһ»бһҹбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·

бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ Windows Edge AI бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– AI бһўбҹҶбһҺбһ¶бһ…бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷбһңбҹҒбһ‘бһ·бһҖбһ¶ Windows бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„бһўбһ¶бһ…бһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһ”бһ¶бһ“бҹ” бһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһўбҹҒбһҖбһјбһҹбҹҠбһёбһҹбҹ’бһ‘бҹӮбһ“ Windows AI Foundry бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһҒбһҺбҹҲбһ–бҹҒбһӣбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢбһ“бҹғбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡвҖӢбһҜбһҖбһҮбһ“бһ—бһ¶бһ– бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бҹ”

бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҡбһңбһ¶бһ„ Windows AI APIs, Foundry Local бһ“бһ·бһ„ Windows ML бһ•бҹ’бһҸбһӣбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Windows бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҮбҹҶбһ“бһ¶бһ“бҹӢбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ” бһ“бҹ…бһ–бҹҒбһӣ AI бһ”бһ“бҹ’бһҸбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһңбҹҒбһ‘бһ·бһҖбһ¶ Windows бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҸбһҳбҹ’бһҖбһҫбһ„бһҮбһ¶бһҳбһҪбһҷбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҗбҹ’бһҳбһёбҹ—бһҠбҹ„бһҷбһҡбһҖбҹ’бһҹбһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶ hardware Windows бһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶бҹ”
