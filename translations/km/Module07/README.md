# бһҹбҹҖбһңбһ—бҹ…бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 07 : бһҸбҹҶбһҺбһ¶бһ„ EdgeAI

Edge AI бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһ“бҹғбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҗбҹ’бһҳбһёбһҡбһ”бһҹбҹӢбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҮбһ¶бһҳбһҪбһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһӮбһҺбһ“бһ¶бһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бһ”бҹӢ (edge computing) бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ—бһ¶бһ–бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ бһҠбҹ„бһҷбһҳбһ·бһ“бһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһ–бһ–бһҖбҹ” бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһ”бһ¶бһ“бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ EdgeAI бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ”бҹ’бһҡбһ¶бҹҶбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ“бҹ…бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһ“бһ·бһ„бһҹбҹҶбһҺбһ»бҹҶбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ—бһ¶бһ–бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ“бһ·бһ„бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶбһ“бҹғбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҳбҹүбһјбһҠбҹӮбһӣ AI бһ“бҹ…бһҖбһҳбҹ’бһҡбһ·бһҸ edgeбҹ”

## 1. EdgeAI бһ“бҹ…бһӣбһҫ NVIDIA Jetson Orin Nano

NVIDIA Jetson Orin Nano бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ edge AI бһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһҮбһ¶ бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҖбҹ’бһӣбһ¶бһҷбһҠбһӣбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢ AI бһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҠбһӣбҹӢ 67 TOPS бһҖбҹ’бһ“бһ»бһ„бһ‘бҹҶбһ бҹҶбһҸбһјбһ…бһҠбһјбһ…бһҖбһ¶бһҸбһҘбһҺбһ‘бһ¶бһ“бҹ” бһңбҹҒбһ‘бһ·бһҖбһ¶бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ—бһ¶бһ–бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҠбһӣбҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ Generative AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҹбһ·бһҹбҹ’бһҹбһ“бһ·бһҹбҹ’бһҹбһ·бһҸ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбҹ” 

### бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—
- бһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– AI бһҠбһӣбҹӢ 67 TOPS бһҠбҹӮбһӣбһӣбһҫбһҹбһ–бһёбһҳбһ»бһ“ 1.7 бһҠбһ„
- бһҳбһ¶бһ“ 1024 CUDA cores бһ“бһ·бһ„ 32 Tensor Cores бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ AI
- бһ”бҹ’бһҡбһҫ CPU Arm Cortex-A78AE v8.2 64-bit 6 бһ•бҹ’бһ“бҹӮбһҖ бһ”бҹ’бһҡбһңбҹӮбһ„бһ”бһҡбһ·бһҳбһ¶бһҺбһҖбҹҶбһҺбһҸбҹӢбһӣбҹ’бһ”бһҝбһ“бһўбһҸбһ·бһ”бһҡбһҳбһ¶ 1.5 GHz
- бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһ№бһҳ $249 бһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҹбһҳбһҡбһҳбҹ’бһҷбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһҹбһ·бһҹбҹ’бһҹ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ•бһӣбһ·бһҸбҹ”

### бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ
Jetson Orin Nano бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ AI бһ”бҹ’бһҡбһ—бҹҒбһ‘ Generative бһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбһјбһ…бһҮбһ¶ Vision Transformers, бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ, бһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҮбһ¶бһҳбһҪбһҷбһңбһ·бһҹбҹҗбһҷбһҳбһҫбһӣбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбҹ„бһҷбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ GenAI бһ бһҫбһҷбһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ LLMs бһ–бһё Devices бһҠбҹғбһ”бһ¶бһ“бҹ” бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҡбһҪбһҳбһҳбһ¶бһ“бһҡбҹүбһјбһ”бһјбһҸбһҶбҹ’бһӣбһ¶бһҸбһңбҹғ, бһҢбҹ’бһҡбһјбһ“бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ, бһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ, бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚ Autonomous edgeбҹ”

## 2. EdgeAI бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҮбһ¶бһҳбһҪбһҷ .NET MAUI бһ“бһ·бһ„ ONNX Runtime GenAI

бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһңбһ·бһ’бһёбһ”бһүбҹ’бһ…бһјбһӣ Generative AI бһ“бһ·бһ„ бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLMs) бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһ…бһҳбҹ’бһҡбһ»бҹҮбһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ .NET MAUI (Multi-platform App UI) бһ“бһ·бһ„ ONNX Runtime GenAIбҹ” бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ .NET бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҠбҹ„бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ Android бһ“бһ·бһ„ iOSбҹ”

### бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—
- бһҹбһ„бҹӢбһӣбһҫбһ•бҹ’бһ“бҹӮбһҖбһ”бһҺбҹ’бһҠбһ¶бһү .NET MAUI бһ•бҹ’бһҸбһӣбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбһјбһҠбһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„ Android бһ“бһ·бһ„ iOS
- бһ”бһүбҹ’бһ…бһјбһӣ ONNX Runtime GenAI бһўбһ¶бһ…бһ”бһүбҹ’бһҮбһ¶ AI Generative бһ“бһ¶вҖӢбһ‘бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһ”бһ¶бһ“
- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбҹҶбһ“бһҪбһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ бһ¶бһҸбһңбҹӮбһҡбһ…бһҳбҹ’бһҡбһ»бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҒбҹ’бһӣбһҪбһ“бһҡбһҪбһҳбһҳбһ¶бһ“ CPU, GPU, бһ“бһ·бһ„ Mobile AI Processors
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһҖбҹ’бһҒбһҺбҹҮ platform бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҠбһјбһ…бһҮбһ¶ CoreML бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ iOS бһ“бһ·бһ„ NNAPI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Android бһҸбһ¶бһҳбһҡбһҷбҹҲ ONNX Runtime
- бһўбһ“бһ»бһңбһҸбҹ’бһҸ бһңбһҠбҹ’бһҠ Generative AI бһ–бҹҒбһүбһӣбҹҒбһү бһҡбһҪбһҳбһҳбһ¶бһ“ бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҳбһ»бһ“бһ“бһ·бһ„бһҖбҹ’бһҡбҹ„бһҷ, бһҖбһ¶бһҡбһҮбҹ’бһҡбҹҖбһҸбһ…бһјбһӣ, бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһӣбһҡ, бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ“бһ·бһ„ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣ KV Cache

бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ .NET MAUI бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһңбһ¶бһҷбһ“бһ—бһҺбҹ’бһҢбһҮбҹҶбһ“бһ¶бһү C# бһ“бһ·бһ„ .NET бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҒбҹ’бһ–бһҹбҹӢбһҮбҹҶбһ“бһ¶бһүбҹ” ONNX Runtime GenAI бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶ Llama, Mistral, Phi, Gemma бһҮбһ¶бһҠбһҫбһҳбҹ” Kernel ARM64 бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһү INT4 Quantized matrix multiplication бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҡбһ бҹҗбһҹбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҳбһ·бһ“бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ .NET бҹ”

### бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ“бҹҒбҹҮбһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһҠбҹӮбһӣбһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘ бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ .NET бһҡбһҪбһҳбһҳбһ¶бһ“ Chatbots бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҡбһјбһ”бһ—бһ¶бһ– бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһҖбһ”бҹ’бһҡбҹӮбһ—бһ¶бһҹбһ¶ бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҺбҹӮбһ“бһ¶бҹҶбһ”бһ»бһӮбҹ’бһӮбһӣ бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ—бһ¶бһ–бһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ–бһёбһҖбҹ’бһҡбҹ…бһ”бһҺбҹ’бһҸбһ¶бһүбҹ”

## 3. EdgeAI бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҒбҹ’бһ“бһ¶бһҸбһҸбһјбһ…

бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ EdgeAI бһ“бҹ…бһӣбһҫ Azure бһҡбһ”бһҹбҹӢ Microsoft бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ‘бһ»бһҖ Small Language Models (SLMs) бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҡбһңбһ¶бһ„бһ–бһ–бһҖ бһ“бһ·бһ„ edge hybridбҹ” бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ“бҹҒбҹҮбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҸбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ AI бһӣбһҫбһҖбһҳбҹ’бһҡбһ·бһҸбһ–бһ–бһҖбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣ edgeбҹ”

## [5. EdgeAI бһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Foundry Local](./foundrylocal.md)

Foundry Local бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ Windows бһ“бһ·бһ„ Mac бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Retrieval Augmented Generation (RAG) бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ .NET бһ”бҹ’бһҡбһҫбһ—бһ¶бһҹбһ¶бһҳбҹүбһјбһҠбҹӮбһӣбһӣбһҫ local бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ semanticsбҹ” бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢ бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ—бһ¶бһ–бһҜбһҖбһҮбһ“бһ–бҹҒбһүбһӣбҹҒбһүбһӣбһҫ local infrastructure бҹ”

### бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ
- бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶ Phi, Local Embeddings бһ“бһ·бһ„ Semantic Kernel бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– RAG
- бһ”бҹ’бһҡбһҫ embeddings бһҮбһ¶ бһңбһ·бһ…бһ‘бҹҗбһҡ (arrays) бһ“бҹғбһҸбһҳбҹ’бһӣбҹғ floating-point бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһҺбһ¶бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ“бҹҗбһҷ
- Semantic Kernel бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҖбҹҶбһ–бһјбһӣбһҖбһ¶бһҡбһҮбһҪбһ”бһҡбһҪбһҳ Phi бһ“бһ·бһ„ Smart Components бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸ pipeline RAG бһҠбҹӮбһӣбһҡбһӣбһјбһ“
- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ vector бһҹбҹ’бһҸбһ»бһҖбһҖбҹ’бһ“бһ»бһ„ local бһҡбһҪбһҳбһҳбһ¶бһ“ SQLite бһ“бһ·бһ„ Qdrant

### бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹғбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ
RAG (Retrieval Augmented Generation) бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ„бҹ’бһҖбҹҒбһҸбҹ” бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҫ local бһ“бҹҒбҹҮбһ’бһ¶бһ“бһ¶бһ–бһ»бһҖбҹ’бһҡбһ№бһҸбҹ’бһҠбһ—бһ¶бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһҶбҹ’бһӣбһ¶бһҸбһңбҹғвҖӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҠбһ¶бһүбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ” бһңбһ¶бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһ¶бһҖбҹӢбһ…бҹҒбһүбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ”бһҺбҹ’бһҸбһ¶бһүбҹ”

### бһңбҹҒбһ‘бһ·бһҖбһ¶ Windows AI Foundry
бһңбҹҒбһ‘бһ·бһҖбһ¶ Windows AI Foundry бһ•бҹ’бһҸбһӣбҹӢбһҹбҹҶбһҺбһ»бҹҶбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„ API бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ Edge AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ Windowsбҹ” бһҡбһҪбһҳбһҳбһ¶бһ“бһҮбҹҶбһ“бһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ NPU-accelerated hardware, бһ…бһјбһӣбһҡбһҪбһҳ Windows ML бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–бһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбҹҒбһ‘бһ·бһҖбһ¶бҹ”

**бһҹбҹҖбһңбһ—бҹ…бһҺбҹӮбһ“бһ¶бҹҶбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ**бҹ– [Windows EdgeAI Development Guide](../windowdeveloper.md)

бһҹбҹҖбһңбһ—бҹ…бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ:
- бһҹбһ„бҹ’бһҒбҹҒбһ”бһ“бҹғбһңбҹҒбһ‘бһ·бһҖбһ¶ Windows AI Foundry бһ“бһ·бһ„бһҹбһҳбһ¶бһҹбһ—бһ¶бһ–
- API Phi Silica бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҮбҹ’бһҡбҹҖбһҸбһ…бһјбһӣбһӣбһҝбһ“бһӣбһҫ hardware NPU
- API Computer Vision бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„ OCR
- бһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– Windows ML runtime
- CLI Foundry Local бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ local
- бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– hardware бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ Windows
- бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһ“бһ·бһ„бһңбһёбһ’бһёбһӣбҹ’бһўбҹ—

### [AI Toolkit бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ—бһ·бһңбһҢбҹ’бһҚ Edge AI](./aitoolkit.md)
бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ Visual Studio Code, бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ AI Toolkit бһ•бҹ’бһҸбһӣбҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ‘бҹҶбһ–бҹҗбһҡбһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ„бҹ’бһҡбҹ’бһӮбҹ„бҹҮ, бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Edge AIбҹ” бһ’бһјбһӣбһҫ AI Toolkit бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ Edge AI бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Codeбҹ”

**бһҹбҹҖбһңбһ—бҹ…бһҺбҹӮбһ“бһ¶бҹҶбһўбһ—бһ·бһңбһҢбҹ’бһҚ**бҹ– [AI Toolkit бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ—бһ·бһңбһҢбҹ’бһҚ Edge AI](./aitoolkit.md)

бһҹбҹҖбһңбһ—бҹ… AI Toolkit бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбҹ–
- бһҡбһҖбҹ’бһҹбһ¶бһҹбһ·бһ‘бҹ’бһ’бһ· бһ“бһ·бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢ edge
- бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһҡбһјбһ”бһ”бҹӮбһ” бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„ local
- бһ”бһүбҹ’бһ…бһјбһӣ ONNX бһ“бһ·бһ„ Ollama бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ edge
- бһ”бһҳбҹ’бһӣбҹӮбһ„бһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ quantization
- бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ—бҹҒбһҮбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– edge
- бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһҸбһ¶бһҳбһҠбһ¶бһ“
- бһҡбҹҖбһ”бһ…бҹҶбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢбҹӨ бһ“бһ·бһ„бһңбһ·бһ’бһёбһӣбҹ’бһўбҹ—

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“

бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ EdgeAI бһ–бһёбһҡбҹҖбһ”бһ…бҹҶбһ”бҹ’бһҡбһ¶бҹҶбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ—бһ¶бһ–бһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһ“бҹғбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ edge AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ” бһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚ edge бһҳбһ¶бһ“ Hardware бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҠбһјбһ… Jetson Orin Nano бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҹбҹҶбһҺбһ»бҹҶбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ software бһҠбһјбһ… ONNX Runtime GenAI бһ“бһ·бһ„ Windows ML бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹбһҳбһ·бһ“бһ’бһҳбҹ’бһҳбһҸбһ¶бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҹбҹҠбһҫбһ”бһўбһ„бҹ’бһҖбҹҒбһҸбһ“бҹ…бһҖбһҳбҹ’бһҡбһ·бһҸ edgeбҹ”

бһҒбҹ’бһҹбҹӮбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҡбһҪбһҳбһҡбһ”бһҹбҹӢбһңбҹҒбһ‘бһ·бһҖбһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһәбһҖбһ¶бһҡбһ•бҹ’бһӮбһ»бҹҶбһҮбһ¶бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһ“бһјбһңбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– AI бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ machine learning бһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбҹ— бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҸбһ» бһ¬бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҢбһ»бһӣбҹ” бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбҹ’бһҠбһёбһӣбҹ’бһў Jetson Orin Nano бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ edge hardware, ONNX Runtime GenAI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһ…бһҳбҹ’бһҡбһ»бҹҮбһңбҹҒбһ‘бһ·бһҖбһ¶, Azure EdgeAI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ cloud-edge, Windows ML бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Windows, бһ бһҫбһҷ Foundry Local бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ RAG бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ—бһ¶бһ–бһҜбһҖбһҮбһ“бҹ” бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбҹ’бһ“бһ¶бһҸбҹҶбһҺбһ¶бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбһ—бһ·бһңбһҢбҹ’бһҚ EdgeAI бһҠбҹ„бһҷбһҹбһҡбһ»бһ”бҹ”
