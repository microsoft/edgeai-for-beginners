# Chapter 04 : Model Format Conversion and Quantization - бһҮбҹҶбһ–бһјбһҖбһҹбһ„бҹ’бһҒбҹҒбһ”

бһҖбһ¶бһҡбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһ“бҹғ EdgeAI бһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„ бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбһҷбһҸбһҳбҹ’бһӣбҹғ (quantization) бһҮбһ¶бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– machine learning бһ…бҹҶбһҡбһјбһ„бһӣбһҫбһҡбһ”бҹҖбһ”бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһ’бһ“бһ’бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбҹ” бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҺбҹӮбһ“бһ¶бҹҶбһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„ бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһҶбҹ’бһңбҹҒбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

## рҹ“ҡ бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбҹҶбһ–бһјбһҖ бһ“бһ·бһ„ бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҮбһ¶бҹ§бһ•бҹ’бһ“бҹӮбһҖбһҠбҹ„бһҷбһӣбҹҶбһҠбһ¶бһ”бҹӢ бһҹбҹӮбһӣбһҳбһҪбһҷбҹ—бһҹбһ¶бһ„бһҹбһ„бҹӢбһӣбһҫбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҳбһ»бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бҹҒбһүбһӣбҹҒбһүбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ edge computing:

## [Section 1: бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„ Quantization](./01.Introduce.md)

### рҹҺҜ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”
бһ•бҹ’бһ“бҹӮбһҖбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹҒбҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҠбһ»бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘ edge computing бһҠбҹ„бһҷбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ–бһёбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“ quantization бһ…бһ¶бһ”бҹӢбһ–бһё 1-bit бһҠбһӣбҹӢ 8-bit бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бҹҶбһӣбҹӮбһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бҹ”

**бһ”бҹ’бһҡбһ’бһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—:**
- бһҹбҹҠбһ»бһҳбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһ–бҹ’бһҡбһёбһҹбҹҠбһёбһҹбҹ’бһҷбһ»бһ„ (ultra-low, low, medium precision)
- бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ бһ“бһ·бһ„бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ GGUF бһ“бһ·бһ„ ONNX
- бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹғ quantization бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһҖбһ¶бһҡ бһ“бһ·бһ„бһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ
- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһ¶бһҺбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ‘бҹҶбһ бҹҶбһ…бһ„бһ…бһ¶бҹҶ

**бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶:**
- бһҷбһӣбҹӢбһ–бһёбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһ“бһ·бһ„бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһ”бһҹбҹӢ quantization
- бһҖбҹҶбһҺбһҸбҹӢбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҠбҹӮбһӣбһҹбһ¶бһҖбһҹбһҳ
- бһҡбҹҖбһ“бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһҶбҹ’бһңбҹҒбһ„бһ§бһ”бһҖбһҡбһҺбҹҚ

## [бһ•бҹ’бһ“бҹӮбһҖ 2: бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ Llama.cpp](./02.Llamacpp.md)

### рҹҺҜ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”
бһҳбҹҒбһҡбҹҖбһ“бһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Llama.cpp бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ C++ бһҒбҹ’бһӣбҹҮ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷ inference бһ“бҹғ Large Language Model бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–бһҠбҹ„бһҷбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҸбһ·бһ… бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһҡбһҫбһ“бһӣбһҫбһҡбһ”бҹҖбһ”бһ бһ¶бһҠбҹ’бһңбҹӮбһҡбҹ”

**бһ”бҹ’бһҡбһ’бһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—:**
- бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶ Windows, macOS, бһ“бһ·бһ„ Linux
- бһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ GGUF бһ“бһ·бһ„ бһҖбһҳбҹ’бһҡбһ·бһҸ quantization бһ•бҹ’бһҹбҹҒбһ„бҹ— (Q2_K бһҠбһӣбҹӢ Q8_0)
- бһҖбһ¶бһҡбһҸбҹ’бһҡбһёбһҖбҹ’бһҹбһ¶бһ бҹҒбһҠбҹ’бһңбҹӮбһҡвҖӢбһҠбҹ„бһҷ CUDA, Metal, OpenCL, бһ“бһ·бһ„ Vulkan
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Python бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһҮбһ¶бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

**бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶:**
- бһҮбҹҶбһ“бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҶбҹ’бһӣбһҫбһҷбһ”бһ¶бһ“бһӣбһҫбһӮбҹ„бһӣбһҠбҹ… cross-platform бһ“бһ·бһ„ бһҖбһҹбһ¶бһ„бһ–бһё source
- бһўбһ“бһ»бһңбһҸбҹ’бһҸ quantization бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“
- бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹ…бһҳбҹүбһјбһҠбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ REST API

## [бһ•бҹ’бһ“бҹӮбһҖ 3: бһҖбһүбҹ’бһ…бһ”бҹӢбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“ Microsoft Olive](./03.MicrosoftOlive.md)

### рҹҺҜ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”
бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһё Microsoft Olive бһҠбҹӮбһӣбһҮбһ¶бһҖбһүбҹ’бһ…бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһ бҹҒбһҠбҹ’бһңбҹӮбһҡ бһҳбһ¶бһ“бһҮбҹҶбһ–бһјбһҖбһ”бһ„бҹ’бһҖбһ”бҹӢбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҮбһ¶бһ„ 40+ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбҹҸбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһӣбһҫбһ бҹҒбһҠбҹ’бһңбҹӮбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

**бһ”бҹ’бһҡбһ’бһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—:**
- бһҳбһ»бһҒбһ„бһ¶бһҡ auto-optimization бһҮбһ¶бһҳбһҪбһҷ quantization бһ‘бһёбһүбһ·бһ„бһҹбҹ’бһҗбһ·бһҸбһ· (dynamic бһ“бһ·бһ„ static)
- бһ”бһүбҹ’бһүбһ¶бһўбһ¶бһ“бһЎбһәбһ…бһ…бҹҶбһ–бҹ„бҹҮбһ бҹҒбһҠбҹ’бһңбҹӮбһҡ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ CPU, GPU бһ“бһ·бһ„ NPU
- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳ (Llama, Phi, Qwen, Gemma) бһ–бһёбһ”бҹ’бһҡбһўбһ”бҹӢ
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷ Azure ML бһ“бһ·бһ„ workflow бһ•бһӣбһ·бһҸ

**бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶:**
- бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһ¶бһүбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ•бҹ’бһҹбҹҒбһ„бҹ—
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҶбҹ’бһӣбһ„бһңбҹҒбһ‘бһ·бһҖбһ¶
- бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹҶбһ–бһ„бҹӢбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҠбҹӮбһӣбһҹбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

## [бһ•бҹ’бһ“бҹӮбһҖ 4: бһҖбһүбҹ’бһ…бһ”бҹӢ OpenVINO Toolkit Optimization](./04.openvino.md)

### рҹҺҜ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”
бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһ–бһё OpenVINO toolkit бһҡбһ”бһҹбҹӢ Intel бһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶бһҜбһҖбһҹбһ¶бһҡбһӣбһҫбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ AI бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӣбһҝбһ“бһӣбһҫ cloud, on-premises бһ“бһ·бһ„ edge бһҮбһ¶бһҳбһҪбһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– Neural Network Compression Framework (NNCF) бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢбҹ”

**бһ”бҹ’бһҡбһ’бһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—:**
- бһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһ…бһҳбҹ’бһҡбһ»бҹҮбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһӣбһ¶бһҸбһҸбҹ’бһҡбһҠбһ¶бһ”бҹӢбһ бҹҒбһҠбҹ’бһңбҹӮбһҡ (CPU, GPU, VPU, AI accelerators)
- Neural Network Compression Framework (NNCF) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ quantization бһ“бһ·бһ„ pruning бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ
- OpenVINO GenAI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ LLMs
- бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹӮбһӣбһўбһ¶бһ…бһңбһ·бһҹбһ¶бһӣ

**бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶:**
- бһҮбҹҶбһ“бһ¶бһүбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„ бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҖбҹ’бһ“бһ»бһ„ OpenVINO
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ quantization бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷ NNCF
- бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһҮбһ¶бһӣбһҫбһ бҹҒбһҠбҹ’бһңбҹӮбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҮбһ¶бһҳбһҪбһҷ Model Server

## [бһ•бҹ’бһ“бҹӮбһҖ 5: бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһӣбһҳбҹ’бһўбһ·бһҸ Apple MLX Framework](./05.AppleMLX.md)

### рҹҺҜ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”
бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ–бҹҒбһүбһӣбҹҒбһүбһўбҹҶбһ–бһё Apple MLX бһҠбҹӮбһӣбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһӣбҹӮбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ machine learning бһӣбһҫ Apple Silicon бһҠбҹ„бһҷбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– LLM бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ‘бһ№бһҖбһҠбһёбҹ”

**бһ”бҹ’бһҡбһ’бһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—:**
- бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹҒбһҳбҹүбһјбһҡбһёбһҡбһҪбһҳ бһ“бһ·бһ„ Metal Performance Shaders
- бһӮбһ¶бҹҶбһ‘бҹ’бһҡ LLaMA, Mistral, Phi-3, Qwen, бһ“бһ·бһ„ Code Llama
- LoRA fine-tuning бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҠбһјбһҡбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ Hugging Face бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡ quantization (4-bit бһ“бһ·бһ„ 8-bit)

**бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶:**
- бһҮбҹҶбһ“бһ¶бһүбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“ Apple Silicon бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ LLM
- бһўбһ“бһ»бһңбһҸбҹ’бһҸ fine-tuning бһ“бһ·бһ„бһ”Customizer бһҳбҹүбһјбһҠбҹӮбһӣ
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһ—бһ¶бһ–бһҜбһҖбһҮбһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһӣбҹ’бһў

## [бһ•бҹ’бһ“бҹӮбһҖ 6: бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһҪбһӣбһӣбҹҶбһ бһјбһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ Edge AI](./06.workflow-synthesis.md)

**бһ”бҹ’бһҡбһ’бһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—:**
- бһўбһ¶бһӮбһ¶бһҡбһӣбҹҶбһ бһјбһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ framework бһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶
- бһҠбһҫбһҳбһҲбһҫбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹ framework бһ“бһ·бһ„бһңбһ·бһ—бһ¶бһӮбһҖбһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–
- бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһӣбһјбһңбһ•бһӣбһ·бһҸ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ

- бһҮбҹҶбһ“бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹ framework бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһҫбһӣбһ–бһёбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹҶбһ–бһ„бҹӢ Edge AI бһҠбһӣбҹӢбһ•бһӣбһ·бһҸбһ•бһӣбһҮбһ¶бһҳбһҪбһҷбһҳбҹҒбһ’бһ¶бһ“бһёбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„
- бһ•бҹ’бһӮбһ»бҹҶбһ“бһјбһңбһӣбҹҶбһ бһјбһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһҠбһҫбһҡбһҸбһ¶бһҳбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бҹғбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ“бһ¶бһ–бҹҒбһӣбһўбһ“бһ¶бһӮбһҸ

## [бһ•бҹ’бһ“бҹӮбһҖ 7: бһҖбһүбҹ’бһ…бһ”бҹӢбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“ Qualcomm QNN](./07.QualcommQNN.md)

### рҹҺҜ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”
бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бҹҒбһүбһӣбҹҒбһүбһўбҹҶбһ–бһё Qualcomm QNN (Qualcomm Neural Network) бһҠбҹӮбһӣбһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶ inference AI бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһҺбһ“бһ¶ heterogeneous бһҡбһ”бһҹбҹӢ Qualcomm бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ Hexagon NPU, Adreno GPU бһ“бһ·бһ„ Kryo CPU бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһҗбһ¶бһҳбһ–бһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһӣбҹҗбһҸбһ“бһ·бһ„ Edgeбҹ”

**бһ”бҹ’бһҡбһ’бһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—:**
- бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһ”бһҪбһ“бһҒбҹ’бһ“бһ¶бһҸ heterogeneous бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһҪбһҳбһ‘бҹ… NPU, GPU бһ“бһ·бһ„ CPU
- бһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбҹҒбһ‘бһ·бһҖбһ¶ Snapdragon бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбһ¶бһҸбһ–бҹ’бһңбһҖбһ·бһ…бҹ’бһ…бһҷбҹүбһ¶бһ„бһҶбҹ’бһӣбһ¶бһҸ
- бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ quantization бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ (INT8, INT16, mixed-precision) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһӣбһҫбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘
- inference бһ”бҹ’бһҡбһҫбһҗбһ¶бһҳбһ–бһӣбһҸбһ·бһ…бһҠбҹӮбһӣбһўбһјбһ”бһ‘бһёбһҳбҹүбһ¶бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбһ№бһҖбһҗбҹ’бһҳ бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бҹҒбһӣбһ–бһ·бһҸ

**бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶:**
- бһҮбҹҶбһ“бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫбһ бҹҒбһҠбҹ’бһңбҹӮбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҡбһ”бһҹбҹӢ Qualcomm бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI бһӣбһҫбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ”бҹ’бһҡбһҫбһҗбһ¶бһҳбһ–бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ edge computing
- бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ…бҹҒбһүбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһӣбһҫбһўз”ҹжҖҒ Qualcomm бһҮбһ¶бһҳбһҪбһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ

## рҹҺҜ бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹғбһҮбҹҶбһ–бһјбһҖ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһҮбҹҶбһ–бһјбһҖбһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ“бһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бҹ–

### **бһҮбҹҶбһ“бһ¶бһүбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ**
- бһҷбһӣбҹӢбһҠбһ№бһ„бһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бһўбҹҶбһ–бһёбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһ“бҹғ quantization бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣ
- бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҮбһ¶бһҳбһҪбһҷ framework бһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһ…бҹ’бһҡбһҫбһ“
- бһҮбҹҶбһ“бһ¶бһүбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘ edge computing

### **бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ**
- бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҮбҹ’бһҡбһҫбһҹбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҹбһ·бһ“бһҠбҹ„бһҷбһҷбһҖбһ бҹҒбһҠбҹ’бһңбҹӮбһҡunner бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ…бһ·бһҸбҹ’бһҸ
- бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҠбҹ„бһҷбһҳбһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһўбҹҶбһ–бһёбһҖбһ„бҹ’бһңбҹҮ-бһӣбһ‘бҹ’бһ’бһ•бһӣ
- бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
