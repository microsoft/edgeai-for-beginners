- **គោលបំណង**: គ្រឿងបន្លាស់បង្កើតប្រសិទ្ធភាពគំរូសម្រាប់ដាក់នៅចុង
- **លក្ខណៈសំខាន់ៗ**:
  - លំហូរអូតូម៉ាទិចសម្រាប់បង្កើតប្រសិទ្ធភាពគំរូ
  - ការបង្កើតប្រសិទ្ធភាពខាងឧបករណ៍
  - ផ្សំរួមជាមួយ ONNX Runtime
  - ឧបករណ៍វាស់ប្រសិទ្ធភាព
- **ការដំឡើងនិងការប្រើប្រាស់មូលដ្ឋាន**:
  ```bash
  # ដំឡើង Olive
  pip install olive-ai
  ```
  
  # ខ្នាតឯកសារលេខ Python សម្រាប់បង្កើតប្រសិទ្ធភាពគំរូ
  ```python
  from olive.model import ONNXModel
  from olive.workflows import run_workflow
  
  # កំណត់ម៉ូដែល និងការកំណត់បរិយាកាសបង្កើតប្រសិទ្ធភាព
  model = ONNXModel("original_model.onnx")
  config = {
      "input_model": model,
      "systems": {
          "local_system": {
              "type": "LocalSystem"
          }
      },
      "engine": {
          "log_severity_level": 0,
          "cache_dir": "cache"
      },
      "passes": {
          "quantization": {
              "type": "OrtQuantization",
              "config": {
                  "quant_mode": "static",
                  "activation_type": "int8",
                  "weight_type": "int8"
              }
          }
      }
  }
  
  # ដាំដើមដំណើរការបង្កើតប្រសិទ្ធភាព
  result = run_workflow(config)
  optimized_model = result.optimized_model
  
  # រក្សាទុកម៉ូដែលដែលបានបង្កើតប្រសិទ្ធភាព
  optimized_model.save("optimized_model.onnx")
  ```

  ```python
  # ផ្ទុកទំងន់ដែលបានបណ្ដុះបណ្ដាលរួច (ឧទាហរណ៍ជាមួយ MLP ងាយស្រួល)
  class MLP(nn.Module):
      def __init__(self, dim=768, hidden_dim=3072):
          super().__init__()
          self.fc1 = nn.Linear(dim, hidden_dim)
          self.fc2 = nn.Linear(hidden_dim, dim)
          
      def __call__(self, x):
          return self.fc2(mx.maximum(0, self.fc1(x)))
  
  # បង្កើតម៉ូដែល និងផ្ទុកទំងន់
  model = MLP()
  weights = mx.load("original_weights.npz")
  model.update(weights)
  
  # បំលែងទំងន់ម៉ូដែលទៅជា FP16
  def quantize_weights(model):
      params = {}
      for k, v in tree_flatten(model.parameters()):
          params[k] = v.astype(mx.float16)
      model.update(params)
      return model
  
  quantized_model = quantize_weights(model)
  
  # រក្សាទុកម៉ូដែលដែលបានបំលែង
  mx.save("quantized_model.npz", quantized_model.parameters())
  
  # រត់ការព្យាករណ៍
  input_data = mx.random.normal((1, 768))
  output = quantized_model(input_data)
  ```

- **លក្ខណៈសំខាន់ៗ**:
  - បង្កើតប្រសិទ្ធភាពផ្អែកលើឧបករណ៍ជាក់លាក់ទាំង CPU, GPU, NPU
  - បង្កើតប្រសិទ្ធភាពក្រាហ្វសម្រាប់ការចេញលទ្ធផល
  - គាំទ្រការតំលៃកំណត់(Qantization)
  - គាំទ្រភាសាច្រើន (Python, C++, C#, JavaScript)
- **ការដំឡើងនិងការប្រើប្រាស់មូលដ្ឋាន**:
  ```bash
  # តំឡើង ONNX Runtime
  pip install onnxruntime
  
  # សម្រាប់គាំទ្រ GPU
  pip install onnxruntime-gpu
  ```
  
  ```python
  import onnxruntime as ort
  import numpy as np
  
  # បង្កើត​សម័យ​សន្និដ្ឋាន​ជាមួយ​ការ​ប្រសើរឡើង
  sess_options = ort.SessionOptions()
  sess_options.graph_optimization_level = ort.GraphOptimizationLevel.ORT_ENABLE_ALL
  sess_options.enable_profiling = True  # បើកការតាមដានការល្បឿន
  
  # បង្កើតសម័យជាមួយការជ្រើសរើសអ្នកផ្តល់សម្រាប់ ការបង្វែរ​ទន់​ផ្ទាល់ខ្លួន
  providers = ['CUDAExecutionProvider', 'CPUExecutionProvider']  # ប្រើ GPU ប្រសិនបើមាន
  session = ort.InferenceSession("model.onnx", sess_options, providers=providers)
  
  # រៀបចំទិន្នន័យបញ្ចូល
  input_name = session.get_inputs()[0].name
  input_shape = session.get_inputs()[0].shape
  input_data = np.random.rand(*input_shape).astype(np.float32)
  
  # ប្រតិបត្តិការ​សន្និដ្ឋាន
  outputs = session.run(None, {input_name: input_data})
  
  # ទទួលបានទិន្នន័យ​តាមដាន
  prof_file = session.end_profiling()
  print(f"Profiling data saved to: {prof_file}")
  ```
