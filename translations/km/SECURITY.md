## бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ

бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“ Microsoft бһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ…бҹҶбһ–бҹ„бҹҮбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһ“бҹғбһ•бһӣбһ·бһҸбһ•бһӣбһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҠбҹӮбһӣбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ¶бҹҶбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҖбһјбһҠбһ”бҹ’бһҡбһ—бһ–бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһӣбһҫ GitHubбҹ”

**бһҹбһјбһҳбһҖбһ»бҹҶбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ…бҹҶбһ–бҹ„бҹҮбһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһҹбҹ„бһҷбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бһүбҹ’бһ бһ¶бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһ“бҹ…бһӣбһҫ GitHubбҹ”**

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ бһ‘бһёбһҸбһ¶бҹҶбһ„ бһӣбҹҒбһҒбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ бһ“бһ·бһ„бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҗбҹ’бһҳбһёбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Microsoft бһ“бҹ… [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)бҹ”
