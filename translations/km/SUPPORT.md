# គាំទ្រ

## របៀបដាក់បញ្ហារួចបានជំនួយ  

គម្រោងនេះប្រើ GitHub Issues ដើម្បីតាមដានកំហុស និងសំណើមុខងារ។ សូមស្វែងរកបញ្ហាដែលមានរួចមុនពេលដាក់បញ្ហាថ្មី ដើម្បីជៀសវាងការយកបញ្ហាដដែល។ សម្រាប់បញ្ហាថ្មី សូមដាក់កំហុស ឬសំណើមុខងារជាបញ្ហាថ្មី។

សម្រាប់ជំនួយ និងសំណួរអំពីការប្រើប្រាស់គម្រោងនេះ សូម៖
- ពិនិត្យ [Azure AI Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ដែលជាកន្លែងអ្នកជំនាញ និងអ្នកអភិវឌ្ឍន៍ផ្សេងៗប្រមូលផ្តុំគ្នា

## នយោបាយគាំទ្ររបស់ Microsoft  

ការគាំទ្រសម្រាប់គម្រោង Model Context Protocol (MCP) សម្រាប់អ្នកចាប់ផ្តើម ត្រូវបានកំណត់ត្រឹមតែធនធានដែលបានរាយនាមខាងលើ។
