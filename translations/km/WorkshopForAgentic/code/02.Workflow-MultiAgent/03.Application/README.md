# កម្មវិធី Podcast

កម្មវិធីរត់តាមផ្ទាំងពាក្យបញ្ជា សម្រាប់បង្កើតស្គ្រីប Podcast ដោយប្រើភ្នាក់ងារ AI។

## Usage

```bash
python podcast_app.py
```

## ដំណើរការ

1. **សូមស្វាគមន៍** - កម្មវិធីស្វាគមន៍អ្នកប្រើ
2. **បញ្ចូលប្រធានបទ** - អ្នកប្រើផ្តល់ប្រធានបទសម្រាប់ Podcast
3. **ភ្នាក់ងារស្វែងរក** - ស្វែងរកព័ត៌មានដែលពាក់ព័ន្ធ
4. **ភ្នាក់ងារបង្កើតស្គ្រីប** - បង្កើតស្គ្រីប Podcast
5. **ការពិនិត្យ** - អ្នកប្រើពិនិត្យ និងអនុម័ត/បដិសេធស្គ្រីប
6. **រក្សាទុក** - ស្គ្រីបដែលបានអនុម័តត្រូវបានរក្សាទុកទៅកាន់ `podcast.md`

## តម្រូវការ

- Python 3.12+
- agent_framework
- អាស្រ័យភាពទាំងអស់ពី 02.WorkflowDevUI
