## бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһҪбһӣбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ

бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бҹ” бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҹбҹ’бһ‘бһ¶бһҸбҹӢ_SCRIPTбҹ” бһҘбһЎбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһ–бһјбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бҹ’бһӣбҹӮбҹ– бһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҮбһ¶бһҹбһҳбҹ’бһӣбҹҒбһ„бһ–бһ·бһғбһ¶бһҸбһҖбһ¶бһҹбһ–бһ·бһҸбһҮбһ¶бһҳбһҪбһҷбһҹбҹҶбһӣбҹҒбһ„бһҠбҹӮбһӣбһҹбҹ’бһҡбһҠбҹҖбһ„бһ“бһ№бһ„бһҳбһ“бһ»бһҹбҹ’бһҹ!

бһҺбҹӮбһ“бһ¶бҹҶбһ‘бҹ…бһҖбһ¶бһ“бҹӢ **VibeVoice** вҖ” бһҮбһ¶ TTS (text-to-speech) бһҳбһҖбһ–бһё Microsoft Research бһҠбҹӮбһӣбһ”бһҫбһҖбһ…бҹҶбһ бһҡбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ:
- рҹҺӯ бһҹбҹҶбһӣбҹҒбһ„бһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбһ’бһҳбҹ’бһҳбһҸбһ¶
- рҹ‘Ҙ бһўбҹ’бһ“бһҖбһ“бһ·бһҷбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“бһ“бһ¶бһҖбҹӢ (бһҡбһ бһјбһҸбһ‘бҹ… 4!)
- вҸұпёҸ бһҹбһҳбҹ’бһӣбҹҒбһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҷбһјбһҡ (бһҡбһ бһјбһҸбһҠбһӣбҹӢ 90бһ“бһ¶бһ‘бһё!)
- рҹҺө бһҖбһ¶бһҡбһ…бҹ’бһҡбҹҒбһёбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ (бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҹбһҳбҹ’бһӣбҹҒбһ„бһҷбһ“бҹ’бһҸбһ”бҹ’бһҡбһјбһ”!)

бһ“бҹҒбҹҮбһҮбһ¶бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһӮбҹ’бһҡбһ”бҹӢбһҖбҹ’бһҡбҹ„бһҳбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹғбһ–бһ·бһғбһ¶бһҸбһҖбһ¶бһҹбһҹЧҷЧ ЧӘбҹҒбһҸбһ·бһҖбҹ” бһҷбһҫбһ„бһ¶бһ’бҹ’бһңбһҫбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһІбҹ’бһҷбһ”бһ¶бһ“!

## VibeVoice бһӮбһәбһўбҹ’бһңбһё? (бһўбҹ’бһңбһёбһҠбҹӮбһӣбһӮбһҪбһҡбһұбҹ’бһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһү)

VibeVoice бһӮбһәбһҮбһ¶бһўбҹҶбһҺбҹ„бһҷбһ–бһё Microsoft Research бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһӣбҹҒбһ„бһҹбһ“бҹ’бһ‘бһ“бһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ–бһ·бһғбһ¶бһҸбһҖбһ¶бһҹбҹ”

### бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһңбһ¶бһ–бһ·бһҹбҹҒбһҹ рҹ”Ҙ

* **вҸұпёҸ бһҹбһҳбҹҗбһҷбһҳбҹүбһ¶бһҡбҹүбһ¶бһҸбһ»бһ„**: бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһӣбҹҒбһ„бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҡбһ бһјбһҸбһҠбһӣбҹӢ 90 бһ“бһ¶бһ‘бһёбһҠбҹ„бһҷбһҲбһҡбһҸбҹӮбһҖбһ“бҹ’бһӣбһ„бһҸбҹӮбһҳбһҪбһҷ (бһ“бҹ„бҹҮбһӮбһәбһҮбһ¶бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ–бһ·бһғбһ¶бһҸбһҖбһ¶бһҹбһ–бҹҒбһүбһҳбһҪбһҷбһ—бһ¶бһӮ!)
* **рҹ‘Ҙ бһҸбһ“бҹ’бһҸбҹ’бһҡбһёбһўбҹ’бһ“бһҖбһ“бһ·бһҷбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“**: бһҡбһ бһјбһҸбһҠбһӣбҹӢ 4 бһҹбҹҶбһӣбҹҒбһ„бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһ—бһ¶бһ–бһңбһ·бһ“бһ·бһ…бҹ’бһҶбҹҗбһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„
* **вҡЎ бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһӣбһ¶бҹҶбһ„**: бһ”бҹ’бһҡбһҫбһўбһҸбҹ’бһҡбһ¶бһҒбҹ’бһ“бһ¶бһҸбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ‘бһ¶бһ” 7.5 Hz бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ“бҹ’бһҹбҹҶбһҗбһ¶бһҳбһ–бһӣбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ
* **рҹ§  бһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·**: бһ•бҹ’бһӮбһ»бҹҶбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ LLM (бһҷбһӣбҹӢбһ”бһҡбһ·бһ”бһ‘) бһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣ diffusion (бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһӣбҹҒбһ„бһҠбҹӮбһӣбһҹбҹ’бһҡбһҠбҹҖбһ„)
* **рҹҺӯ бһ‘бҹҶбһ“бһ·бһүбһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·**: бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ…бҹ’бһҡбҹҒбһёбһ“, бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҲбһ”бҹӢ, бһ“бһ·бһ„бһ”бһ¶бһүбҹӢбһ…бҹҒбһүбһҹбһ“бҹ’бһ‘бһ“бһ¶ бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·

бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮ: VibeVoice бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһўбһ¶бһ“бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһ‘бҹҒ вҖ” бһңбһ¶ *бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҸбҹӮбһ„* бһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбһ–бһ·бһҸбҹ—бһҖбҹҶбһ–бһ»бһ„бһ“бһ·бһҷбһ¶бһҷбҹ”

## бһҳбһ»бһ“бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ рҹҡҖ

**бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ**:

* рҹҗҚ **Python 3.10+** (бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҳбһ¶бһ“бһ“бҹҒбҹҮбһ–бһё бһ’бһ¶бһҸбһ»бһ‘бһё 1 & 2)
* рҹҡҖ **uv** (бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„Package Python бһӣбһҝбһ“ вҖ” бһҷбһҫбһ„бһ“бһ№бһ„бһҸбҹҶбһЎбһҫбһ„бһңбһ¶)
* рҹ“қ **бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ**: бһҜбһҖбһҹбһ¶бһҡ `podcast.txt` бһ–бһё Act 2 (бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `../03.Application/`)

**бһӮбһ“бҹ’бһӣбһ№бҹҮбһҮбҹҶбһ“бһ¶бһү**: бһҮбҹҶбһ бһ¶бһ“бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһүбһҷбһҖбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ бҹ’бһңбһәбһ„бһҮбһ¶бһҳбһ»бһ“бҹ” бһ…бһ¶бһ”бҹӢбһҖбһ¶бһ бҹ’бһңбҹҒбһҳбһҪбһҷбһҖбҹӮбһң! вҳ•

## бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ! бһңбһ·бһ’бһёбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ рҹҺ¬

бһҷбһҫбһ„бһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһңбһ¶бһІбҹ’бһҷбһҹбһ¶бһҳбһүбҹ’бһүбһҺбһ¶бһҹбҹӢбҹ” бһҹбҹ’бһӮбҹ’бһҡбһёбһ” shell бһҳбһҪбһҷбһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бҹ”

### бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ

1. **бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ”бһ¶бһ“**:
```bash
chmod +x run_vibe_voice.sh
```

2. **бһҡбһҸбҹӢбһңбһ¶**:
```bash
./run_vibe_voice.sh
```

3. **бһҡбһ„бһ…бһ¶бҹҶбһҖбһ¶бһҡбһ–бҹ’бһҷбһјбһҡбһ–бҹҒбһӣ** (бһ“бҹҒбҹҮбһўбһ¶бһ…бһ…бҹҶбһҺбһ¶бһҷбһ–бһёбһҡбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһ“бһ¶бһ‘бһёбһ“бҹ…бһӣбһҫбһҖбһ¶бһҡбһҡбһҸбҹӢбһҠбҹҶбһ”бһјбһ„)

### бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһҶбһ¶бһҖ рҹҺӯ

бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһӮбһәбһҠбһјбһ…бһҮбһ¶бһңбһ·бһҹбҹ’бһңбһҖбһҡбһҹбһҳбҹ’бһӣбҹҒбһ„бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ:

1. **рҹ“Ҙ бһ‘бһ¶бһүбһҷбһҖ VibeVoice**: бһӮбһјбһҹбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ repo бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ–бһё GitHub
2. **рҹ“Ұ бһҸбҹҶбһЎбһҫбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ**: бһ”бҹ’бһҡбһҫ `uv pip` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҖбһүбҹ’бһ…бһ”бҹӢбһҷбҹүбһ¶бһ„бһӣбһҝбһ“
3. **рҹҺ¬ бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһӣбҹҒбһ„**: бһҡбһҸбҹӢбһҹбҹ’бһӮбҹ’бһҡбһёбһ” inference бһҮбһ¶бһҳбһҪбһҷ:
   * `--model_path`: бһҳбҹүбһјбһҠбҹӮбһӣ VibeVoice-7B бһҠбҹӮбһӣбһ”бһ¶бһ“бһ бҹ’бһңбһәбһ„бһҮбһ¶бһҳбһ»бһ“
