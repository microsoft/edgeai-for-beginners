# ផ្នែកទី៣៖ ធ្វើឲ្យប៉ូដខាសរបស់អ្នករស់រវើក 🎤

![មន្ទីរពិសោធន៍ 3](../../../../../translated_images/km/lab3.b009c55c4442250b.webp)

## ចុងក្រោយ

អ្នកបានស្រាវជ្រាវប្រធានបទរួច។ អ្នកបានសរសេរស្គ្រីបរួច។ ឥឡូវនេះ​គឺជាការបន្ថែមទៀត៖ បម្លែងអត្ថបទរបស់អ្នកទៅជាអូឌីយោប៉ូដខាសពិតជាមួយសម្លេងដែលស្រដៀងមនុស្ស!

ចូលស្គាល់ **VibeVoice** —— មន្ត TTS (អត្ថបទទៅសម្លេង) បើកប្រភពពី Microsoft Research ដែលបង្កើត៖
- 🎭 សន្ទនាធម្មជាតិ និងរលូន
- 👥 អ្នកនិយាយ​ច្រើននាក់ (ដល់ 4 នាក់!)
- ⏱️ អូឌីយោរយៈវែង (ដល់ 90 នាទី!)
- 🎵 ការបង្ហាញមានអារម្មណ៍ (មិនមែនសម្លេងម៉ាស៊ីនទេ!)

នេះជាបច្ចេកវិទ្យាខ្លាំង​នៅពីក្រោយការសម្រួលប៉ូដខាស។ មកបង្កើតរបស់អ្នកគ្នា!

## VibeVoice ជាអ្វី?（រឿងល្អ）

VibeVoice គឺជាអំណោយពី Microsoft Research សម្រាប់ពិភពលោក។ វាត្រូវបានរចនាឡើងសម្រាប់ទ្រង់ទ្រាយសន្ទនាបែបប៉ូដខាស។

### ហេតុអ្វីបានជា​វាអស្ចារ្យ 🔥

* **⏱️ សន្ទនារយៈវែង**：បង្កើតសម្លេងបន្តរបានរហូតដល់ 90 នាទី (នោះគឺជាឧបភោគប៉ូដខាសពេញលេញមួយ!)
* **👥 មន្តច្រើនអ្នកនិយាយ**：រហូតដល់ 4 សម្លេងផ្សេងគ្នា ជាមួយបុគ្គលិកលក្ខណៈសរុបស្រដៀង
* **⚡ មានប្រសិទ្ធភាពខ្ពស់**：ប្រើអត្រាខ្នាត 7.5 Hz ទាបដើម្បីសន្សំតម្លៃគណនា
* **🧠 សម្លេងឆ្លាតវៃ**：បញ្ចូល LLM (យល់ពីបរិបទ) និងម៉ូឌែល diffusion (បង្កើតអូឌីយោស្រដៀងពិត)
* **🎭 ធម្មជាតិ និងរលូន**：ដោះស្រាយការប្តូរនិយាយ, ពេលឈប់ និងទំនុកចិត្តនៃសន្ទនាដោយស្វ័យប្រវត្តិ

**សេចក្ដីប្រែ**：VibeVoice មិនត្រឹមតែអានស្គ្រីបរបស់អ្នកទេ —— វា *បង្ហាញ* វាដូចមនុស្សពិត។

## មុនចាប់ផ្តើម 🚀

**អ្នកត្រូវការអ្វីខ្លះ**：

* 🐍 **Python 3.10+**（អ្នកបានមាននេះពីផ្នែកទី១ និងទី២）
* 🚀 **uv**（កម្មវិធីគ្រប់គ្រងកញ្ចប់ Python ល្បឿនមួយ —— យើងនឹងតំឡើងវា）
* 📝 **ស្គ្រីបរបស់អ្នក**：ឯកសារ `podcast.txt` ពីផ្នែកទី២（នៅក្នុង `../03.Application/`）

**គន្លឹះវិជ្ជាជីវៈ**：ជំហាននេះត្រូវការតំណភ្ជាប់អ៊ីនធឺណេទល្អសម្រាប់ទាញយកម៉ូឌែលដែលបានបណ្ដុះរួច។ ហៅកាហ្វេមួយកែវ! ☕

## ចាប់ផ្តើម吧！វិធីសាមញ្ញ 🎬

យើងធ្វើឲ្យវាមានភាពងាយស្រួល។ ស្គ្រីប shell មួយធ្វើអ្វីៗទាំងអស់។

### ដំណើរការ

1. **ធ្វើឲ្យវាអាចអនុវត្តបាន**：
```bash
chmod +x run_vibe_voice.sh
```

2. **រត់វា**：
```bash
./run_vibe_voice.sh
```

3. **រង់ចាំមន្ត**（ការ​រត់លើកដំបូងប្រហែលត្រូវការប៉ុន្មាននាទី）

### អ្វីកើតឡើងខាងក្រោយ 🎭

ស្គ្រីបគឺជาวิចិត្រសិល្បៈវិស្វករអូឌីយោស្វ័យប្រវត្តិរបស់អ្នក៖

1. **📥 ទាញយក VibeVoice**：ចម្លង repository ផ្លូវការពី GitHub
2. **📦 តំឡើងអាស្រ័យការ**：ប្រើ `uv pip` ដើម្បីតំឡើងកញ្ចប់យ៉ាងលឿន
3. **🎬 បង្កើតអូឌីយោ**：រត់ស្គ្រីប inference ដោយប្រើ：
   * `--model_path`：ម៉ូឌែល VibeVoice-7B ដែលបានបណ្ដុះរួច
   * `--txt_path`：ស្គ្រីប `podcast.txt` របស់អ្នក
   * `--speaker_names`：ផ្តល់ឈ្មោះអ្នកនិយាយ (លំនាំដើមជា Xinran និង Anchen)

**លទ្ធផល**：ស្គ្រីបរបស់អ្នកបានបម្លែងទៅជាផ្នែកប៉ូដខាសពិត!🎉

## ភារកិច្ចរបស់អ្នក 🎯

ចាំមើលសប្បាយទៅទាំងនេះ៖

### ភារៈកិច្ច 1：បង្កើតមាតិច
កែសម្រួល `../03.Application/podcast.txt` ជាសន្ទនារវាងមនុស្សពីរនាក់។ ឲ្យវាពាក់ព័ន្ធនឹងបច្ចេកវិទ្យា ឬចំណង់ចំណូលចិត្ត ឬអ្វីនៅណា — តែអោយវាជាសន្ទនា។

**ទម្រង់ឧទាហរណ៍**：
```
说话人 1：嘿！你听说新的 AI 模型了吗？
说话人 2：不会吧！告诉我更多！
说话人 1：它叫...
```

### ភារៈកិច្ច 2：生成音频
រត់ស្គ្រីប ហើយមើលមន្តកើតឡើង។ លើកដំបូងនឹងចំណាយពេលយូរ (ទាញយកម៉ូឌែល)។

### ភារៈកិច្ច 3：聆听和分析
- តើសម្លេងស្តាប់មានធម្មជាតេទេ?
- តើអ្នកនិយាយមានសម្លេងខុសគ្នាទេ?
- តើការប្តូរនិយាយតាមលំដាប់រលូនទេ?
- តើមានសម័យដែលដូចម៉ាស៊ីនឬសំឡេងប្រពៃណីទេ?

### ភារៈកិច្ច 4：实验（给勇敢的人）
កែ `run_vibe_voice.sh` ហើយផ្លាស់ `--speaker_names` ដើម្បី​ព្យាយាមបង្វិលការចងក្រុមសម្លេងផ្សេងៗ។ VibeVoice មានសម្លេងដែលបានបណ្ដុះរួចជាច្រើន!

**បញ្ហាស្វាគមន៍**：ព្យាយាមសន្ទនារវាងអ្នកនិយាយ 3 នាក់!🎆
