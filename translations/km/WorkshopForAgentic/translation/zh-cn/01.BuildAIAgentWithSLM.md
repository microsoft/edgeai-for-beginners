## ការប្រឈម

អ្នកកំពុងចាប់ផ្តើម «未来字节» ប៉ូដខាសបច្ចេកវិទ្យាថ្មីរបស់អ្នក។ ភាគទី១ ស្តីអំពីការបំផ្លាញថ្មីៗក្នុងវិស័យ AI ប៉ុន្តែអ្នកមានពេលត្រឹម 24 ម៉ោងដើម្បី：
1. ស្រាវជ្រាវប្រធានបទ
2. ស្វែងរកប្រភពដែលអាចទុកចិត្តបាន
3. សរសេរស្គ្រីបដែលទាក់ទាញ
4. ធ្វើឲ្យវាហាក់ដូចជាធម្មជាតិ

**ការបង្វិលរឿង**：អ្នកមិនចាំបាច់ធ្វើវាដោយខ្លួនឯងទេ។ អ្នកនឹងបង្កើតជំនួយការលើកដំបូងរបស់អ្នក — វាជាដៃគូស្រាវជ្រាវដែលមិនដែលឈប់នឿយ និងមិនចាំបាច់ដេក។

## ហេតុអ្វីជ្រើសម៉ូដែលភាសាតូច？（剧透：它们很棒）

គិតពី小型语言模型（SLM）ជាឧបករណ៍ AI ផ្ទាល់ខ្លួនដែលដំណើរការលើកុំព្យូទ័ររបស់អ្នក។ គ្មានពពក、គ្មានថ្លៃប្រចាំខែ、គ្មានការចែករំលែកទិន្នន័យដែលគួរឲ្យសង្ស័យ។

**ហេតុអ្វី SLM ល្អៗ🔥：**
- **🏠 在你的机器上运行**：កុំព្យូទ័រយួរដៃ、កុំព្យូទ័រផ្ទាល់តុ，甚至是高性能的 Raspberry Pi
- **💸 零持续成本**：គ្មានថ្លៃ API ប្រចាំដែលបរិភោគប្រាក់របស់អ្នក
- **🔒 隐私优先**：ទិន្នន័យរបស់អ្នកនឹងមិនចាកចេញពីឧបករណ៍របស់អ្នកទេ
- **⚡ 闪电般快速**：គ្មានពន្យាពេលបណ្ដាញ，ឆ្លើយតបទាន់ភ្លាម
- **🪦 轻量级**：10 亿-100 亿参数 vs 大模型的 1000 亿+

**流行的 SLM**：Qwen 3、Phi-4、Gemma 3（本工作坊使用 Qwen）

## 你的工具包

### Ollama：你的 AI 模型管理器

[Ollama](https://ollama.com/) 就像 AI 模型的 Steam。用简单的命令下载、运行和管理模型。

**它的酷之处：**
- 一个命令就能下载和运行任何模型
- 适用于 Mac、Windows、Linux
- 如果你有 GPU，会自动使用
- 超级节省内存

### Microsoft Agent Framework：魔法发生的地方

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework) 是你构建 AI 智能体的游乐场,它们可以:

- 💬 聊天并记住你们谈论的内容
- 🛠️ 使用自定义工具(如搜索网络或查看天气)
- 🧠 逐步思考复杂问题
- 🤝 作为团队与其他智能体协作
- 🔌 连接到不同的 AI 提供商（OpenAI、Ollama、Azure）

**构建模块：**
- **代理（Agents）**：有特定工作的 AI 助手
- **工具（Tools）**：你赋予它们的特殊能力
- **记忆（Memory）**：这样它们不会忘记你的对话
- **推理（Reasoning）**：教它们思考，而不仅仅是回应

## 当事情出错时（以及如何修复）🔧

### "Alex 加载不了！内存不足！"
**解决方案**：你的电脑在挣扎。尝试关闭其他应用，或切换到更小的模型。8GB 内存是最低要求。

### "Alex 太慢了"
**解决方案**：在 Ollama 设置中启用 GPU 加速。或者减少上下文窗口大小。速度恶魔模式激活！🏎️

### "工具不工作！"
**解决方案**：仔细检查你的函数签名。Alex 需要正确的类型提示来理解工具的作用。把它想象成给出清晰的指令。
