## бһҹбһҳбҹҗбһҷ 04: бһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”вҖӢбһҳбҹүбһјбһҢбҹӮбһӣ

### аӨүаӨҰаҘҚаӨҰаҘҮаӨ¶аҘҚаӨҜ
бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ‘бһёбһҸбһ¶бҹҶбһ„бһҖбһ¶бһҡбһҹбһҳбҹ’бһҸбҹӮбһ„бһҡбһңбһ¶бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҒбҹ’бһ“бһ¶бһҸбһҸбһјбһ… (SLM) бһ“бһ·бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҒбҹ’бһ“бһ¶бһҸбһ’бҹҶ (LLM)бҹ”

### бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҡбһ бҹҗбһҹ

```bash
# бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ (бһ”бһҫбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ)
foundry service start

# бһ•бҹ’бһ‘бһ»бһҖбһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ
foundry model run phi-4-mini
foundry model run qwen2.5-3b
```

### бһҡбһ”бҹҖбһ”бһҡбһҸбҹӢвҖӢбһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶

1. **Open** `session04_model_compare.ipynb` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code бһ¬ Jupyter
2. **Restart kernel** (Kernel вҶ’ Restart Kernel)
3. **Run all cells** бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢ

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбҹҶбһҒбһ¶бһ“бҹӢ

**бһҳбҹүбһјбһҢбҹӮбһӣбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ:**
- **SLM:** `phi-4-mini` (~4GB RAM, бһӣбһҝбһ“бһҮбһ¶бһ„)
- **LLM:** `qwen2.5-3b` (~3GB RAM, бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбҹҒбһҳбҹүбһјбһҡбһё)

**бһўбһҗбҹҒбһҡбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ):**
```python
import os
os.environ['SLM_ALIAS'] = 'phi-4-mini'
os.environ['LLM_ALIAS'] = 'qwen2.5-3b'
os.environ['FOUNDRY_LOCAL_ENDPOINT'] = 'http://localhost:59959/v1'
```

**бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣбһ•бҹ’бһҹбҹҒбһ„:**
