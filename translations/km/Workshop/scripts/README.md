### Running Locally
From the repository root (recommended):

Windows (PowerShell):
```powershell
python Workshop\scripts\lint_markdown_cli.py --verbose
```

macOS / Linux:
```bash
python Workshop/scripts/lint_markdown_cli.py --verbose
```

### Pre-Commit Hook (Optional)
```bash
echo "python Workshop/scripts/lint_markdown_cli.py" > .git/hooks/pre-commit
chmod +x .git/hooks/pre-commit
```
This blocks commits that introduce deprecated patterns.

### CI Integration
A GitHub Action workflow (`.github/workflows/markdown-cli-lint.yml`) runs the linter on every push and pull request to `main` and `Reactor` branches. Failing jobs must be fixed before merging.

### Adding New Deprecated Patterns
1. бһ”бһҫбһҖ `lint_markdown_cli.py`.
2. бһ”бһүбҹ’бһ…бһјбһӣ tuple `(regex, suggestion)` бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһё `DEPRECATED`бҹ” бһ”бҹ’бһҡбһҫ raw string бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣ `\b` бһҮбһ¶бһҖбҹҶбһҺбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбҹ”
3. бһҡбһҸбҹӢбһӣбһ·бһ“бһ’бҹҗбһҡбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбҹ”
4. Commit and push; CI бһ“бһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҗбҹ’бһҳбһёбҹ”

Example addition:
```python
DEPRECATED.append((r"\\bfoundry\\s+experimental\\s+foo\\b", "Remove experimental foo usage"))
```

### Allowing Explanatory Mentions
бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһҸбҹӮбһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбһҪбһҡбһҠбҹҒбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбһҖбһ…бҹҒбһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһҡбҹ„бһ‘бҹҚбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖ *бһҸбҹ’бһҡбһјбһңбһҸбҹӮ* бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһҪбһҖбһңбһ¶бһ—бһ¶бһӮбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ»бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶, бһ”бһјбһҖбһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһӣбһ»бһҖбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбһҪбһҡбһҠбҹҒбһҖ **бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“** triple backticks (бһ§. indent бһ¬ quote) бһ¬бһҹбһҡбһҹбҹҒбһҡбһЎбһҫбһ„бһңбһ·бһүбһ‘бҹ…бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ pseudoбҹ”

### Troubleshooting
| Issue | Cause | Resolution |
|-------|-------|-----------|
| Linter flags a line you updated | Regex too broad | Narrow pattern with additional word boundary (`\b`) or anchors |
| CI fails but local passes | Different Python version or uncommitted changes | Re-run locally, ensure clean working tree, check workflow Python version (3.11) |
| Need to temporarily bypass | Emergency hotfix | Apply fix immediately after; consider using a temporary branch and follow-up PR (avoid adding bypass switches) |

### Rationale
бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһҜбһҖбһҹбһ¶бһҡвҖӢбһІбҹ’бһҷбһҹбҹ’бһҡбһ”бһҮбһ¶бһҳбһҪбһҷбһ•бҹ’бһ‘бҹғ CLI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–бһ“бһ¶бһ–бҹҒбһӣбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһҮбһҪбһҷбһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢбһҖбһ¶бһҡбһҳбһҪбһҖбһ…бһ·бһҸбҹ’бһҸбһҖбҹ’бһ“бһ»бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶ бһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ—бһ¶бһ“бҹӢбһ…бҹ’бһҡбһЎбҹҶбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҡбҹҖбһ“ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҸбһ¶бһҳбһҹбҹ’бһӮбҹ’бһҡбһёбһ” Python бһҠбҹӮбһӣбһ”бһ¶бһ“бһҗбҹӮбһ‘бһ¶бҹҶ бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһҡбһҳбҹ’бһӣбҹҒбһ…бһ”бһүбҹ’бһҮбһ¶ CLIбҹ”

### What it checks
- вң… бһҹбһ»бһ–бһӣбһ—бһ¶бһ–бһҹбһүбҹ’бһүбһ¶бһҹбһҡбһҹбҹҒбһҡ Python
- вң… бһҳбһ¶бһ“бһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ
- вң… бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ (бһҳбҹүбһјбһҠбһӣбһҳбҹ’бһўбһ·бһҸ)
- вң… бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ type hints (бһҳбҹүбһјбһҠбһӣбһҳбҹ’бһўбһ·бһҸ)
- вң… docstrings бһҡбһ”бһҹбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ (бһҳбҹүбһјбһҠбһӣбһҳбҹ’бһўбһ·бһҸ)
- вң… бһҸбҹҶбһҺбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҷбҹ„бһ„ SDK (бһҳбҹүбһјбһҠбһӣбһҳбҹ’бһўбһ·бһҸ)

### Environment Variables
- `SKIP_IMPORT_CHECK=1` - бһҡбҹҶбһӣбһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ import
- `SKIP_SYNTAX_CHECK=1` - бһҡбҹҶбһӣбһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһүбҹ’бһүбһ¶бһҹбһҡбһҹбҹҒбһҡ

### Exit Codes

- `1` - бһӮбҹҶбһҡбһјбһҳбһҪбһҷбһ¬бһ…бҹ’бһҡбһҫбһ“бһ”бһҡбһ¶бһҮбҹҗбһҷ

## 3. бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸбһӮбҹҶбһҡбһј

`test_samples.py` бһҡбһҸбҹӢбһҸбҹҒбһҹбҹ’бһҸ smoke бһӣбһҫбһӮбҹҶбһҡбһјбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһ–бһҪбһҖбһңбһ¶бһ…бҹҒбһүбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹбҹ”

### Usage
```bash
# бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӮбҹҶбһҡбһјбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
python scripts/test_samples.py

# бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбһҳбҹҗбһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
python scripts/test_samples.py --session 01

# бһҡбһ”бҹҖбһ”бһӣбһҝбһ“ (бһ–бҹҒбһӣбһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢбһҒбҹ’бһӣбһёбһҮбһ¶бһ„)
python scripts/test_samples.py --quick

# бһҡбһ”бҹҖбһ”бһӣбһҳбҹ’бһўбһ·бһҸ (бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣ)
python scripts/test_samples.py --verbose
```

### Prerequisites
- бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Foundry Local бһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ: `foundry service start`
- бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖ: `foundry model run phi-4-mini`
- бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ–бһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„: `pip install -r requirements.txt`

### What it tests
- вң… бһӮбҹҶбһҡбһјбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ”бһ¶бһ“бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹ runtime
- вң… бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бһӣбһ·бһҸ
- вң… бһҳбһ¶бһ“бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ–бҹҒбһӣбһ”бһҡбһ¶бһҮбҹҗбһҷ
- вң… бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– (бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ)

### Environment Variables
- `FOUNDRY_LOCAL_ALIAS=phi-4-mini` - бһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ
- `TEST_TIMEOUT=30` - бһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹҶбһҡбһјбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ“бһ¶бһ‘бһё

### Expected Failures
бһҸбҹҒбһҹбҹ’бһҸбһҒбҹ’бһӣбҹҮбһ”бҹ’бһҡбһ бҹӮбһӣбһ”бһҡбһ¶бһҮбҹҗбһҷ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ–бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһҳбһ·бһ“бһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„ (бһ§. `ragas`, `sentence-transformers`)бҹ” бһҠбҹҶбһЎбһҫбһ„бһҮбһ¶бһҳбһҪбһҷ:

## 4. бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ¶бҹҶбһ…бҹҒбһү Benchmark бһ‘бҹ… Markdown

Script: `export_benchmark_markdown.py`

бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹ’бһҡбһ»бһҳбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһҪбһҷбҹ”

### Usage
```powershell
python Workshop\scripts\export_benchmark_markdown.py --models "qwen2.5-0.5b" --prompt "Explain retrieval augmented generation briefly." --rounds 3 --output benchmark_report.md
```

### Outputs
| File | Description |
|------|-------------|
| `benchmark_report.md` | бһҸбһ¶бһҡбһ¶бһ„ Markdown (avg, p95, tokens/sec, latency бһ“бҹғ token бһҠбҹҶбһ”бһјбһ„ бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) |
| `benchmark_report.json` | бһҹбҹҠбһ»бһҳбһҳбҹүбҹҒбһҸбҹ’бһҡбһ·бһҖбһҠбһҫбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹҖбһ” бһ“бһ·бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ |

### Options
| Flag | Description | Default |
|------|-------------|---------|
| `--models` | Comma-separated model aliases | (required) |
| `--prompt` | Prompt used each round | (required) |
| `--rounds` | Rounds per model | 3 |
| `--output` | Markdown output file | `benchmark_report.md` |
| `--json` | JSON output file | `benchmark_report.json` |
| `--fail-on-empty` | Non-zero exit if all benchmarks fail | off |

Environment variable `BENCH_STREAM=1` adds first token latency measurement.
