# Podcast Application

A console application for generating podcast scripts using AI agents.

## Usage

```bash
python podcast_app.py
```

## Workflow

1. **Welcome** - Application dey greet di user
2. **Topic Input** - User go provide topic for di podcast
3. **Search Agent** - E go search for correct information
4. **Generate Script Agent** - E go create podcast script
5. **Review** - User go check and approve or reject di script
6. **Save** - Di script wey dem approve go save for `podcast.md`

## Requirements

- Python 3.12+
- agent_framework
- All dependencies from 02.WorkflowDevUI
