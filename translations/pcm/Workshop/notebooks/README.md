## üöÄ How to Start

### Wetin You Need Before You Start

**System Requirements**:
- **OS**: Windows 10/11, macOS 11+, or Linux (Ubuntu 20.04+)
- **RAM**: At least 8GB, but 16GB+ go better
- **Storage**: At least 10GB free space for models
- **Hardware**: CPU wey get AVX2; GPU (CUDA, Qualcomm NPU) na optional

**Software Requirements**:
- **Python 3.8+** wey get pip
- **Jupyter Notebook** or **VS Code** wey get Jupyter extension
- **Microsoft Foundry Local** wey don dey install and configure
- **Git** (to clone repository)

### How to Install

#### 1. Install Foundry Local

**Windows**:
```cmd
winget install Microsoft.FoundryLocal
```

**macOS**:
```bash
brew tap microsoft/foundrylocal
brew install foundrylocal
```

**Verify Installation**:
```bash
foundry --version
```

#### 2. Arrange Python Environment

```bash
# Navigate to Workshop directory
cd Workshop

# Create virtual environment
python -m venv .venv

# Activate virtual environment
# Windows
.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

#### 3. Start Foundry Local

```bash
# Load a model (auto-downloads if needed)
foundry model run phi-4-mini

# Verify service is running
foundry service status
```

#### 4. Open Jupyter

```bash
# Start Jupyter Notebook
jupyter notebook notebooks/

# Or use VS Code with Jupyter extension
code notebooks/
```

### Quick Check

Run this one for Python cell to confirm setup:

```python
from foundry_local import FoundryLocalManager
import openai

# Initialize manager (auto-discovers service)
manager = FoundryLocalManager("phi-4-mini")

# Configure OpenAI client
client = openai.OpenAI(
    base_url=manager.endpoint,
    api_key=manager.api_key
)

# Test chat completion
response = client.chat.completions.create(
    model=manager.get_model_info("phi-4-mini").id,
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

**Wetin You Go See**: Greeting response from the local model.
