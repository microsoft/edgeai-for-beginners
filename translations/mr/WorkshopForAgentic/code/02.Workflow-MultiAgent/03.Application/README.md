# पॉडकास्ट अनुप्रयोग

एआय एजंट्स वापरून पॉडकास्ट स्क्रिप्ट तयार करण्यासाठी एक कन्सोल अनुप्रयोग.

## वापर

```bash
python podcast_app.py
```

## कार्यप्रवाह

1. **स्वागत आहे** - अनुप्रयोग वापरकर्ता स्वागत करतो
2. **विषय इनपुट** - वापरकर्ता पॉडकास्टसाठी विषय प्रदान करतो
3. **शोध एजंट** - संबंधित माहिती शोधतो
4. **स्क्रिप्ट तयार करणारा एजंट** - पॉडकास्ट स्क्रिप्ट तयार करतो
5. **पुनरावलोकन** - वापरकर्ता स्क्रिप्ट पुनरावलोकन करतो आणि मंजूर/नकारतो
6. **जतन करा** - मंजूर केलेली स्क्रिप्ट `podcast.md` मध्ये जतन केली जाते

## आवश्यकता

- Python 3.12+
- agent_framework
- 02.WorkflowDevUI मधील सर्व अवलंबित्वे
