## ഫംഗ്ഷൻ കോളിംഗ് എന്താണ്?

ഫംഗ്ഷൻ കോളിംഗ് എന്നത് സ്മോൾ ലാംഗ്വേജ് മോഡലുകൾ (SLMs) ബാഹ്യ ടൂളുകൾ, APIകൾ, സേവനങ്ങൾ എന്നിവയുമായി ഇടപഴകാൻ അനുവദിക്കുന്ന ശക്തമായ കഴിവാണ്. അവരുടെ പരിശീലന ഡാറ്റയിൽ മാത്രം പരിമിതപ്പെടാതെ, SLMകൾ ഇനി:

- **ബാഹ്യ APIകളുമായി ബന്ധപ്പെടുക** (വെതർ സർവീസുകൾ, ഡാറ്റാബേസുകൾ, സെർച്ച് എഞ്ചിനുകൾ)
- **ഉപയോക്തൃ അഭ്യർത്ഥനകളുടെ അടിസ്ഥാനത്തിൽ പ്രത്യേക ഫംഗ്ഷനുകൾ പ്രവർത്തിപ്പിക്കുക**
- **വിവിധ ഉറവിടങ്ങളിൽ നിന്നുള്ള യഥാർത്ഥ സമയ വിവരങ്ങൾ ലഭ്യമാക്കുക**
- **പ്രത്യേക ടൂളുകൾ വഴി കണക്കുകൂട്ടൽ പ്രവർത്തനങ്ങൾ നടത്തുക**
- **സങ്കീർണ്ണമായ വർക്ക്‌ഫ്ലോകൾക്കായി നിരവധി പ്രവർത്തനങ്ങൾ ചൈനിൽ ബന്ധിപ്പിക്കുക**

ഈ കഴിവ് SLMകളെ സ്ഥിരമായ ടെക്സ്റ്റ് ജനറേറ്ററുകളിൽ നിന്ന് യാഥാർത്ഥ്യ പ്രവർത്തനങ്ങൾ നടത്താൻ കഴിയുന്ന ഡൈനാമിക് AI ഏജന്റുകളായി മാറ്റുന്നു.

## ഫംഗ്ഷൻ കോളിംഗ് എങ്ങനെ പ്രവർത്തിക്കുന്നു

ഫംഗ്ഷൻ കോളിംഗ് പ്രക്രിയ ഒരു ക്രമബദ്ധമായ വർക്ക്‌ഫ്ലോ പിന്തുടരുന്നു:

### 1. ടൂൾ ഇന്റഗ്രേഷൻ
- **ബാഹ്യ ടൂളുകൾ**: SLMകൾ വെതർ APIകൾ, ഡാറ്റാബേസുകൾ, വെബ് സർവീസുകൾ, മറ്റ് ബാഹ്യ സിസ്റ്റങ്ങൾ എന്നിവയുമായി ബന്ധപ്പെടാൻ കഴിയും
- **ഫംഗ്ഷൻ നിർവചനങ്ങൾ**: ഓരോ ടൂളും പ്രത്യേക പാരാമീറ്ററുകൾ, ഇൻപുട്ട്/ഔട്ട്പുട്ട് ഫോർമാറ്റുകൾ, വിവരണങ്ങൾ എന്നിവയോടെ നിർവചിക്കപ്പെടുന്നു
- **API അനുയോജ്യത**: ടൂളുകൾ സ്റ്റാൻഡേർഡൈസ്ഡ് ഇന്റർഫേസുകൾ (REST APIകൾ, SDKകൾ തുടങ്ങിയവ) വഴി ഇന്റഗ്രേറ്റ് ചെയ്യപ്പെടുന്നു

### 2. ഫംഗ്ഷൻ നിർവചനം
ഫംഗ്ഷനുകൾ മൂന്ന് പ്രധാന ഘടകങ്ങളോടെ നിർവചിക്കപ്പെടുന്നു:
```json
{
  "name": "function_name",
  "description": "Clear description of what the function does",
  "parameters": {
    "parameter_name": {
      "description": "What this parameter represents",
      "type": "data_type",
      "default": "default_value"
    }
  }
}
```

### 3. ഉദ്ദേശ്യം കണ്ടെത്തൽ
- **നാച്ചുറൽ ലാംഗ്വേജ് പ്രോസസ്സിംഗ്**: ഉപയോക്തൃ ഇൻപുട്ട് വിശകലനം ചെയ്ത് ഉദ്ദേശ്യം മനസ്സിലാക്കുന്നു
- **ഫംഗ്ഷൻ മാച്ചിംഗ്**: അഭ്യർത്ഥന പൂർത്തിയാക്കാൻ ആവശ്യമായ ഫംഗ്ഷൻ(കൾ) നിർണയിക്കുന്നു
- **പാരാമീറ്റർ എക്സ്ട്രാക്ഷൻ**: ഉപയോക്തൃ സന്ദേശത്തിൽ നിന്നുള്ള ആവശ്യമായ പാരാമീറ്ററുകൾ തിരിച്ചറിയുകയും എടുക്കുകയും ചെയ്യുന്നു

### 4. JSON ഔട്ട്പുട്ട് ജനറേഷൻ
SLM ഘടനാപരമായ JSON സൃഷ്ടിക്കുന്നു, അതിൽ ഉൾപ്പെടുന്നു:
- വിളിക്കേണ്ട ഫംഗ്ഷന്റെ പേര്
- അനുയോജ്യമായ മൂല്യങ്ങളോടെയുള്ള ആവശ്യമായ പാരാമീറ്ററുകൾ
- എക്സിക്യൂഷൻ കോൺടെക്സ്റ്റും മെറ്റാഡാറ്റയും

### 5. ബാഹ്യ എക്സിക്യൂഷൻ
- **പാരാമീറ്റർ പരിശോധന**: എല്ലാ ആവശ്യമായ പാരാമീറ്ററുകളും സാന്നിധ്യമുണ്ടെന്നും ശരിയായ ഫോർമാറ്റിലാണെന്നും ഉറപ്പാക്കുന്നു
- **ഫംഗ്ഷൻ എക്സിക്യൂഷൻ**: നൽകിയ പാരാമീറ്ററുകളോടെ ആപ്ലിക്കേഷൻ നിർദ്ദിഷ്ട ഫംഗ്ഷൻ പ്രവർത്തിപ്പിക്കുന്നു
- **പിശക് കൈകാര്യം**: പരാജയങ്ങൾ, ടൈംഔട്ടുകൾ, അസാധുവായ പ്രതികരണങ്ങൾ എന്നിവ കൈകാര്യം ചെയ്യുന്നു

### 6. പ്രതികരണം ഇന്റഗ്രേഷൻ
- **ഫലം പ്രോസസ്സിംഗ്**: ഫംഗ്ഷൻ ഔട്ട്പുട്ട് SLM-ലേക്ക് തിരികെ നൽകുന്നു
- **കോൺടെക്സ്റ്റ് ഇന്റഗ്രേഷൻ**: ഫലം SLM-ന്റെ പ്രതികരണത്തിൽ ഉൾപ്പെടുത്തുന്നു
- **ഉപയോക്തൃ ആശയവിനിമയം**: സ്വാഭാവിക, സംഭാഷണാത്മക ഫോർമാറ്റിൽ വിവരങ്ങൾ അവതരിപ്പിക്കുന്നു

## അപ്ലിക്കേഷൻ സീനാരിയോകൾ

### ഡാറ്റ റിട്രീവൽ
സ്വാഭാവിക ഭാഷാ ചോദനകൾ ഘടനാപരമായ API കോളുകളായി മാറ്റുക:
- **"എന്റെ പുതിയ ഓർഡറുകൾ കാണിക്കൂ"** → ഉപയോക്തൃ ഐഡി, തീയതി ഫിൽട്ടറുകളോടെയുള്ള ഡാറ്റാബേസ് ക്വറി
- **"ടോക്കിയോയിൽ കാലാവസ്ഥ എന്താണ്?"** → സ്ഥലം പാരാമീറ്ററോടെയുള്ള വെതർ API കോളിംഗ്
- **"ജോൺ കഴിഞ്ഞ ആഴ്ച അയച്ച ഇമെയിലുകൾ കണ്ടെത്തൂ"** → അയച്ചവനും തീയതിയും ഫിൽട്ടർ ചെയ്ത ഇമെയിൽ സർവീസ് ക്വറി

### പ്രവർത്തന നിർവഹണം
ഉപയോക്തൃ അഭ്യർത്ഥനകൾ പ്രത്യേക ഫംഗ്ഷൻ കോളുകളായി മാറ്റുക:
- **"നാളെ 2 മണിക്ക് മീറ്റിംഗ് ഷെഡ്യൂൾ ചെയ്യൂ"** → കലണ്ടർ API ഇന്റഗ്രേഷൻ
- **"ടീമിന് ഒരു സന്ദേശം അയയ്ക്കൂ"** → കമ്മ്യൂണിക്കേഷൻ പ്ലാറ്റ്ഫോം API
- **"എന്റെ ഫയലുകളുടെ ബാക്കപ്പ് സൃഷ്ടിക്കൂ"** → ഫയൽ സിസ്റ്റം ഓപ്പറേഷൻ

### കണക്കുകൂട്ടൽ പ്രവർത്തനങ്ങൾ
സങ്കീർണ്ണമായ ഗണിതമോ ലജിക്കൽ ഓപ്പറേഷനുകളോ കൈകാര്യം ചെയ്യുക:
- **"$10,000-ൽ 5% പലിശ 10 വർഷത്തേക്ക് കണക്കാക്കൂ"** → സാമ്പത്തിക കണക്കുകൂട്ടൽ ഫംഗ്ഷൻ
- **"ഈ ഡാറ്റാസെറ്റ് ട്രെൻഡുകൾക്കായി വിശകലനം ചെയ്യൂ"** → സ്റ്റാറ്റിസ്റ്റിക്കൽ അനാലിസിസ് ടൂളുകൾ
- **"ഡെലിവറിയ്ക്ക് ഈ റൂട്ടു ഒപ്റ്റിമൈസ് ചെയ്യൂ"** → റൂട്ടു ഒപ്റ്റിമൈസേഷൻ ആൽഗോരിതങ്ങൾ

### ഡാറ്റ പ്രോസസ്സിംഗ് വർക്ക്‌ഫ്ലോകൾ
സങ്കീർണ്ണ പ്രവർത്തനങ്ങൾക്ക് നിരവധി ഫംഗ്ഷൻ കോളുകൾ ചൈനിൽ ബന്ധിപ്പിക്കുക:
1. **വിവിധ ഉറവിടങ്ങളിൽ നിന്നുള്ള ഡാറ്റ റിട്രീവ് ചെയ്യുക**
2. **വിവരം പാഴ്‌സ് ചെയ്ത് പരിശോധന നടത്തുക**
3. **ഡാറ്റ ആവശ്യമായ ഫോർമാറ്റിലേക്ക് മാറ്റുക**
4. **ഫലങ്ങൾ അനുയോജ്യമായ സിസ്റ്റങ്ങളിൽ സംഭരിക്കുക**
5. **റിപ്പോർട്ടുകൾ അല്ലെങ്കിൽ ദൃശ്യവൽക്കരണങ്ങൾ സൃഷ്ടിക്കുക**

### UI/UX ഇന്റഗ്രേഷൻ
ഡൈനാമിക് ഇന്റർഫേസ് അപ്‌ഡേറ്റുകൾ സജ്ജമാക്കുക:
- **"ഡാഷ്ബോർഡിൽ സെയിൽസ് ഡാറ്റ കാണിക്കൂ"** → ചാർട്ട് ജനറേഷൻ, പ്രദർശനം
- **"പുതിയ സ്ഥലങ്ങളോടെ മാപ്പ് അപ്‌ഡേറ്റ് ചെയ്യൂ"** → ജിയോസ്പേഷ്യൽ ഡാറ്റ ഇന്റഗ്രേഷൻ
- **"ഇൻവെന്ററി പ്രദർശനം റിഫ്രഷ് ചെയ്യൂ"** → യഥാർത്ഥ സമയ ഡാറ്റ സിങ്ക്രണൈസേഷൻ

## Phi-4-mini ഉം Ollama ഉം ഉപയോഗിച്ച് ഫംഗ്ഷൻ കോളിംഗ് സജ്ജമാക്കൽ

Microsoft-ന്റെ Phi-4-mini ഒറ്റത്തവണയും സമാന്തര ഫംഗ്ഷൻ കോളിംഗും Ollama വഴി പിന്തുണയ്ക്കുന്നു. ഇതു സജ്ജമാക്കുന്നത് ഇങ്ങനെ:

### മുൻകൂട്ടി ആവശ്യമായവ
- Ollama പതിപ്പ് 0.5.13 അല്ലെങ്കിൽ അതിനുമുകളിൽ
- Phi-4-mini മോഡൽ (ശുപാർശ: `phi4-mini:3.8b-fp16`)

### ഇൻസ്റ്റലേഷൻ ഘട്ടങ്ങൾ

#### 1. Phi-4-mini ഇൻസ്റ്റാൾ ചെയ്ത് പ്രവർത്തിപ്പിക്കുക
```bash
# മോഡൽ ഡൗൺലോഡ് ചെയ്യുക (ഇതിനകം ഉണ്ടായില്ലെങ്കിൽ)
ollama run phi4-mini:3.8b-fp16

# മോഡൽ ലഭ്യമാണ് എന്ന് സ്ഥിരീകരിക്കുക
ollama list
```

#### 2. കസ്റ്റം ModelFile ടെംപ്ലേറ്റ് സൃഷ്ടിക്കുക
Ollamaയുടെ നിലവിലെ ഡിഫോൾട്ട് ടെംപ്ലേറ്റുകളിൽ ഉള്ള പരിമിതികൾ കാരണം, താഴെ കൊടുത്തിരിക്കുന്ന ടെംപ്ലേറ്റ് ഉപയോഗിച്ച് കസ്റ്റം ModelFile സൃഷ്ടിക്കേണ്ടതാണ്:

from qwen_agent.tools.base import BaseTool

class WeatherTool(BaseTool):
    description = 'Get weather information for a specific location'
    parameters = [
        {
            'name': 'location',
            'type': 'string', 
            'description': 'City or location name',
            'required': True
        },
        {
            'name': 'units',
            'type': 'string',
            'description': 'Temperature units (celsius or fahrenheit)',
            'required': False,
            'default': 'celsius'
        }
    ]
    
    def call(self, params: str, **kwargs) -> str:
        """Execute the weather lookup"""
        params_dict = json.loads(params)
        location = params_dict.get('location')
        units = params_dict.get('units', 'celsius')
        
        # കാലാവസ്ഥ API കോൾ സിമുലേറ്റ് ചെയ്യുക

            'location': location,
            'temperature': '22°C' if units == 'celsius' else '72°F',
            'condition': 'Partly cloudy',
            'humidity': '65%'
        }
        
        return json.dumps(weather_data)

# കസ്റ്റം ടൂൾ ഉപയോഗിക്കുക

import asyncio
import aiohttp

async def batch_function_calls(function_calls):
    """Execute multiple function calls concurrently"""
    async with aiohttp.ClientSession() as session:
        tasks = []
        for call in function_calls:
            if call["name"] == "fetch_url":
                task = fetch_url_async(session, call["parameters"]["url"])
                tasks.append(task)
        
        results = await asyncio.gather(*tasks)
        return results

async def fetch_url_async(session, url):
    async with session.get(url) as response:
        return await response.text()

    def register_function(self, name: str, description: str, parameters: Dict, 
                         handler: callable, rate_limit: int = 60):
        """Register a new function with the agent"""
        self.functions[name] = {
            'description': description,
            'parameters': parameters,
            'handler': handler,
            'rate_limit': rate_limit,
            'call_count': 0,
            'last_reset': datetime.now()
        }
    
    async def execute_function(self, function_name: str, parameters: Dict) -> FunctionResult:
        """Execute a function with comprehensive error handling and logging"""
        start_time = datetime.now()
        
        try:
            # ഫംഗ്ഷൻ നിലവിലുണ്ടോ എന്ന് സ്ഥിരീകരിക്കുക
            if function_name not in self.functions:
                return FunctionResult(
                    success=False,
                    error=f"Function '{function_name}' not found",
                    timestamp=start_time
                )
            
            # നിരക്ക് പരിധികൾ പരിശോധിക്കുക
            if not self._check_rate_limit(function_name):
                return FunctionResult(
                    success=False,
                    error=f"Rate limit exceeded for function '{function_name}'",
                    timestamp=start_time
                )
            
            # പാരാമീറ്ററുകൾ സ്ഥിരീകരിക്കുക
            validation_result = self._validate_parameters(function_name, parameters)
            if not validation_result.success:
                return validation_result
            
            # ഫംഗ്ഷൻ നടപ്പാക്കുക
            func_info = self.functions[function_name]
            handler = func_info['handler']
            
            if asyncio.iscoroutinefunction(handler):
                result_data = await handler(**parameters)
            else:
                result_data = handler(**parameters)
            
            execution_time = (datetime.now() - start_time).total_seconds()
            
            result = FunctionResult(
                success=True,
                data=result_data,
                execution_time=execution_time,
                timestamp=start_time
            )
            
            # വിജയകരമായ നടപ്പാക്കൽ ലോഗ് ചെയ്യുക
            self._log_function_call(function_name, parameters, result)
            
            return result
            
        except Exception as e:
            execution_time = (datetime.now() - start_time).total_seconds()
            result = FunctionResult(
                success=False,
                error=str(e),
                execution_time=execution_time,
                timestamp=start_time
            )
            
            # പരാജയപ്പെട്ട നടപ്പാക്കൽ ലോഗ് ചെയ്യുക
            self._log_function_call(function_name, parameters, result)
            
            return result
    
    def _check_rate_limit(self, function_name: str) -> bool:
        """Check if function call is within rate limits"""
        func_info = self.functions[function_name]
        now = datetime.now()
        
        # ഒരു മിനിറ്റ് കഴിഞ്ഞാൽ കൗണ്ടർ റീസെറ്റ് ചെയ്യുക
        if (now - func_info['last_reset']).seconds >= 60:
            func_info['call_count'] = 0
            func_info['last_reset'] = now
        
        # പരിധിക്കുള്ളിൽ ആണോ എന്ന് പരിശോധിക്കുക
        if func_info['call_count'] >= func_info['rate_limit']:
            return False
        
        func_info['call_count'] += 1
        return True
    
    def _validate_parameters(self, function_name: str, parameters: Dict) -> FunctionResult:
        """Validate function parameters"""
        func_params = self.functions[function_name]['parameters']
        
        # ആവശ്യമായ പാരാമീറ്ററുകൾ പരിശോധിക്കുക
        for param_name, param_info in func_params.items():
            if param_info.get('required', False) and param_name not in parameters:
                return FunctionResult(
                    success=False,
                    error=f"Missing required parameter: {param_name}"
                )
        
        # പാരാമീറ്റർ തരംകളും നിയന്ത്രണങ്ങളും സ്ഥിരീകരിക്കുക
        for param_name, value in parameters.items():
            if param_name in func_params:
                param_info = func_params[param_name]
                
                # തരം സ്ഥിരീകരണം
                expected_type = param_info.get('type')
                if expected_type == 'string' and not isinstance(value, str):
                    return FunctionResult(
                        success=False,
                        error=f"Parameter '{param_name}' must be a string"
                    )
                elif expected_type == 'number' and not isinstance(value, (int, float)):
                    return FunctionResult(
                        success=False,
                        error=f"Parameter '{param_name}' must be a number"
                    )
                elif expected_type == 'boolean' and not isinstance(value, bool):
                    return FunctionResult(
                        success=False,
                        error=f"Parameter '{param_name}' must be a boolean"
                    )
                
                # എൻയം സ്ഥിരീകരണം
                if 'enum' in param_info and value not in param_info['enum']:
                    return FunctionResult(
                        success=False,
                        error=f"Parameter '{param_name}' must be one of: {param_info['enum']}"
                    )
        
        return FunctionResult(success=True)
    
    def _log_function_call(self, function_name: str, parameters: Dict, result: FunctionResult):

- **സുരക്ഷാ ഫീച്ചറുകൾ വർദ്ധിപ്പിക്കൽ**: മെച്ചപ്പെട്ട ഓതന്റിക്കേഷൻ, ഓതറൈസേഷൻ സംവിധാനങ്ങൾ
- **വ്യാപകമായ ഇക്കോസിസ്റ്റം**: മുൻകൂട്ടി നിർമ്മിച്ച ഫംഗ്ഷനുകളും ഇന്റഗ്രേഷനുകളും വളർച്ച

### ആരംഭിക്കാം

നിങ്ങളുടെ പ്രോജക്ടുകളിൽ ഫംഗ്ഷൻ കോളിംഗ് നടപ്പിലാക്കാൻ:

1. **സിമ്പിൾ ആയി തുടങ്ങുക**: അടിസ്ഥാന ഒറ്റ ഫംഗ്ഷൻ സീനാരിയോകളിൽ നിന്ന് ആരംഭിക്കുക
2. **ഫ്രെയിംവർക്ക് തിരഞ്ഞെടുക്കുക**: നേരിട്ട് നടപ്പിലാക്കൽ (Ollama/Phi-4) അല്ലെങ്കിൽ ഫ്രെയിംവർക്ക് സഹായം (Qwen-Agent) തിരഞ്ഞെടുക്കുക
3. **ഫംഗ്ഷനുകൾ ശ്രദ്ധാപൂർവ്വം ഡിസൈൻ ചെയ്യുക**: വ്യക്തവും നന്നായി രേഖപ്പെടുത്തിയ ഫംഗ്ഷൻ നിർവചനങ്ങളിൽ ശ്രദ്ധ കേന്ദ്രീകരിക്കുക
4. **പിശക് കൈകാര്യം നടപ്പിലാക്കുക**: തുടക്കത്തിൽ തന്നെ ശക്തമായ പിശക് കൈകാര്യം സംവിധാനം നിർമ്മിക്കുക
5. **നിരന്തരം സ്കെയിൽ ചെയ്യുക**: പരിചയം നേടുമ്പോൾ ലളിതത്തിൽ നിന്ന് സങ്കീർണ്ണത്തിലേക്ക് നീങ്ങുക

ഫംഗ്ഷൻ കോളിംഗ് SLMകളെ പ്രശംസനീയമായ ടെക്സ്റ്റ് ജനറേറ്ററുകളിൽ നിന്ന് യാഥാർത്ഥ്യ പ്രശ്നങ്ങൾ പരിഹരിക്കാൻ കഴിയുന്ന പ്രായോഗിക AI ഏജന്റുകളായി മാറ്റുന്നു. ഈ ട്യൂട്ടോറിയലിൽ നൽകിയ മാതൃകകളും പ്രാക്ടീസുകളും പിന്തുടർന്ന്, നിങ്ങൾക്ക് ശക്തമായ, വിശ്വസനീയമായ AI സിസ്റ്റങ്ങൾ നിർമ്മിക്കാം, പരമ്പരാഗത ചാറ്റ് ഇന്റർഫേസുകൾക്കപ്പുറം വ്യാപിച്ചുകൊണ്ടുള്ളവ.
