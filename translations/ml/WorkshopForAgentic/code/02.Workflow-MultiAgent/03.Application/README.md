# പോഡ്കാസ്റ്റ് അപ്ലിക്കേഷന്‍

AI എജന്റുമാരുടെ സഹായത്തോടെ പോഡ്കാസ്റ്റ് സ്‌ക്രിപ്പ്റ്റുകൾ സൃഷ്‌ടിക്കാൻ کنസോൾ അപ്ലിക്കേഷൻ.

## ഉപയോഗം

```bash
python podcast_app.py
```

## പ്രവൃത്തി പ്രക്രിയ

1. **സ്വാഗതം** - അപ്ലിക്കേഷൻ ഉപയോക്താവിനെ സ്വാഗതം ചെയ്യുന്നു
2. **വിഷയ ഇൻപുട്ട്** - ഉപയോക്താവ് പോഡ്കാസ്റ്റിനായി ഒരു വിഷയം നൽകുന്നു
3. **സേർച്ച് എജന്റ്** - ബന്ധപ്പെട്ട വിവരങ്ങൾ തിരയുന്നു
4. **സ്‌ക്രിപ്റ്റ് സൃഷ്ടിക്കൽ എജന്റ്** - ഒരു പോഡ്കാസ്റ്റ് സ്‌ക്രിപ്റ്റ് സൃഷ്ടിക്കുന്നു
5. **സമീക്ഷണം** - ഉപയോക്താവ് സ്‌ക്രിപ്റ്റ് അവലോകനം ചെയ്ത് അംഗീകരിക്കുന്നു/നിഷേധിക്കുന്നു
6. **സേവ്** - അംഗീകരിച്ച സ്‌ക്രിപ്റ്റ് `podcast.md` എന്ന ഫയലിൽ സംരക്ഷിക്കുന്നു

## ആവശ്യകതകൾ

- Python 3.12+
- agent_framework
- 02.WorkflowDevUI-ലെ എല്ലാ ആശ്രിതങ്ങളും
