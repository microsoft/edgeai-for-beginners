## സുരക്ഷ

മൈക്രോസോഫ്റ്റ് നമ്മുടെ സോഫ്റ്റ്വെയർ ഉൽപ്പന്നങ്ങളും സേവനങ്ങളും ഉൾപ്പെടെ, നമ്മുടെ GitHub സംഘടനകളിലെ എല്ലാ സോഴ്‌സ് കോഡ് റിപോസിറ്ററികളും ഉൾപ്പെടെ, സുരക്ഷയെ ഗൗരവത്തോടെ കാണുന്നു.

**സുരക്ഷാ ദുർബലതകൾ പൊതുജന GitHub പ്രശ്നങ്ങളിലൂടെ റിപ്പോർട്ട് ചെയ്യരുത്.**

സുരക്ഷാ റിപ്പോർട്ടിംഗ് വിവരങ്ങൾ, സ്ഥലങ്ങൾ, ബന്ധപ്പെടാനുള്ള വിവരങ്ങൾ, നയങ്ങൾ എന്നിവയ്ക്ക്, ദയവായി മൈക്രോസോഫ്റ്റ് റിപോസിറ്ററികൾക്കുള്ള ഏറ്റവും പുതിയ മാർഗ്ഗനിർദ്ദേശം [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md) പരിശോധിക്കുക.
