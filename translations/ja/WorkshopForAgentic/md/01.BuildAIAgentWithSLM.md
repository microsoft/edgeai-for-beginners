# 第1幕: あなたのAIリサーチアシスタントと出会う 🤖

![lab1](../../../../translated_images/ja/lab1.1dd5210a7d114e2e.webp)

## 挑戦

あなたは新しいテックポッドキャスト「Future Bytes」を始めます。第1話は最新のAIの突破口についてですが、24時間以内に以下をこなさなければなりません：
1. トピックを調査する
2. 信頼できる情報源を見つける
3. 魅力的なスクリプトを書く
4. 自然な話し方にする

**どんでん返し**：一人でやる必要はありません。これからあなたのためにこれらすべてを手伝う最初のAIアシスタントを作ります。名前はAlexとしましょう — 眠らなくても疲れ知らずのリサーチパートナーです。

## なぜ小型言語モデルなのか？（ネタバレ: それらは素晴らしい）

小型言語モデル（SLM）を、*あなたの*コンピューター上にいるパーソナルAIと思ってください。クラウド不要、月額料金なし、怪しいデータ共有もなし。

**SLMが🔥な理由:**
- **🏠 自分の機械で動く**: ノートPC、デスクトップ、さらにはパワフルなRaspberry Piでも
- **💸 継続コストゼロ**: API料金で「昼食代」が減らない
- **🔒 プライバシー最優先**: あなたのデータはデバイスから出ない
- **⚡ 超高速**: インターネットの遅延なし、即時応答
- **🪦 軽量**: 1B〜10Bパラメーターで、大規模なモデルの100B+よりずっと軽い

**人気のSLM**: Qwen 3、Phi-4、Gemma 3（このワークショップではQwenを使います）

## あなたのツールキット

### Ollama: あなたのAIモデルマネージャー

[Ollama](https://ollama.com/)はAIモデルのためのSteamのようなものです。モデルのダウンロード、実行、管理を簡単なコマンドで行えます。

**クールなポイント:**
- コマンド１つでモデルをダウンロード＆実行
- Mac、Windows、Linuxで動作
- GPUがあれば自動で使用
- 超メモリー効率的

### Microsoft Agent Framework: 魔法が起こる場所

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework)は、以下のことができるAIエージェントを作るための遊び場です：

- 💬 会話し、話した内容を覚える
- 🛠️ カスタムツール（ウェブ検索や天気確認など）を使える
- 🧠 複雑な問題を段階的に考える
- 🤝 他のエージェントとチームとして働く
- 🔌 複数のAIプロバイダーに接続（OpenAI、Ollama、Azureなど）

**構成要素:**
- **Agents**: 特定の仕事を持つAIアシスタント
- **Tools**: 彼らに与える特別な能力
- **Memory**: 会話を忘れないための記憶機能
- **Reasoning**: ただ応答するだけでなく考える力を教える

## あなたのトレーニングモンタージュ: 4つのミッション

### ミッション1: 最初のエージェントを作ろう

📓 [ノートブックを開く](../code/01.BasicAgent/00.BasicAgent-agent.ipynb)

**クエスト**: あなたのポッドキャストのスクリプトライターAI、Alexを作ります。Alexはテックトピックについて２人のホストが話す対話を生成します。

**学べること**:
- AIエージェントを起動させる方法（月曜の朝起きるより簡単です）
- 個性と指示を与える方法
- 実際のポッドキャストスクリプトを生成させる方法
- 返答内容を理解する方法

**勝利条件**: AlexがAIについての「Future Bytes」パイロットエピソードのスクリプトを作成！ 🎯

### ミッション2: Alexにスーパーパワー（ツール！）を与える

📓 [ノートブックを開く](../code/01.BasicAgent/01.BasicAgent-tools.ipynb)

**クエスト**: Alexは賢いけれど、今日の天気や時間はわかりません。ツールを与えてそれを解決しましょう！

**学べること**:
- カスタムPython関数を「ツール」として作る
- いつどのツールを使うかAlexに判断させる
- 自律的に問題を解決する様子を観察する
- 複数のツールを組み合わせて複雑なタスクをこなす

**勝利条件**: 「東京の天気は？」と聞いてAlexが自分で調べて答える！ ☁️

### ミッション3: Alexに考えさせる

📓 [ノートブックを開く](../code/01.BasicAgent/02.BasicAgent-reasoning.ipynb)

**クエスト**: Alexに答えだけでなく、考えた過程も見せましょう。問題解決中の思考過程が見たいのです。

**学べること**:
- 「推論モード」をオンにする（数学の授業で途中式を見せる感じ）
- Alexのステップバイステップの思考過程を見る
- チェイン・オブ・ソート推論を理解する
- Alexが混乱したときのデバッグ方法

**勝利条件**: 難しい数学の問題を出してAlexがじっくり考える様子を見る！ 🧠

### ミッション4: Alexをインターネットに接続する

📓 [ノートブックを開く](../code/01.BasicAgent/03.BasicAgent-websearch.ipynb)

**クエスト**: Alexの知識はある時点で止まっています。リアルタイム情報のためにウェブに繋ぎましょう！

**学べること**:
- カスタムウェブ検索ツールの構築
- 外部APIの統合
- ネットワークエラーの優雅な処理
- Alexの訓練データを超えた情報取得

**勝利条件**: 今日のテックニュースを聞いて新鮮な結果を得る！ 📰

## トラブルが起きたら（とその直し方）🔧

### 「Alexが起動しない！メモリ不足！」
**対処法**: あなたのPCが苦しんでいます。他のアプリを閉じるか、より小さいモデルに変えてください。8GBは最低限です。

### 「Alexがすごく遅い」
**対処法**: Ollamaの設定でGPUアクセラレーションを有効にしましょう。あるいはコンテキストウィンドウサイズを減らす。スピードモード発動！ 🏎️

### 「ツールが動かない！」
**対処法**: 関数のシグネチャを再確認。Alexにはツールが何をするか正確な型ヒントが必要です。はっきりした指示を与えるイメージで。
