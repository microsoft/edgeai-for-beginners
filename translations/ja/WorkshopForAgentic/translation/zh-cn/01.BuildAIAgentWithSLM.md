# 第一幕：あなたのAI研究アシスタントを知ろう 🤖

![lab1](../../../../../translated_images/ja/lab1.1dd5210a7d114e2e.png)

## チャレンジ

あなたは「未来バイト」という新しいテクノロジーポッドキャストを立ち上げています。第一回は最新のAIブレイクスルーについてですが、あなたには24時間しかありません：
1. テーマを調査する
2. 信頼できる情報源を見つける
3. 引き込まれる脚本を書く
4. 自然な聞こえにする

**プロットツイスト**：一人でやる必要はありません。あなたの初めてのAIアシスタントを構築します。彼の名前はAlex――疲れ知らずの研究パートナーで、眠る必要なし。

## なぜ小型言語モデル？（ネタバレ：最高だから）

小型言語モデル（SLM）を*あなた*のコンピュータ上で動く、プライベートなAIと考えてください。クラウドなし、月額なし、怪しいデータ共有もなし。

**SLMがすごい理由🔥：**
- **🏠 自分のマシンで動作**：ノートPC、デスクトップ、性能の良いラズベリーパイでも
- **💸 ランニングコストゼロ**：API料金でお小遣いが減らされない
- **🔒 プライバシー優先**：あなたのデータは決してデバイスから出ない
- **⚡ 超高速**：ネットの遅延なし、即応答
- **🪦 軽量**：10億～100億パラメータ vs 大型モデルの1000億以上

**人気のSLM**：Qwen 3、Phi-4、Gemma 3（本ワークショップではQwenを使用）

## あなたのツールキット

### Ollama：あなたのAIモデルマネージャー

[Ollama](https://ollama.com/) はAIモデルのSteamのようなもの。簡単なコマンドでモデルをダウンロードし、実行、管理できる。

**すごいところ：**
- 1コマンドでどんなモデルでもダウンロード＆実行可能
- Mac、Windows、Linux対応
- GPUがあれば自動で利用
- メモリ非常に節約

### Microsoft Agent Framework：魔法が起きる場所

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework) はAIエージェントを作る遊び場。彼らは：

- 💬 チャットし、話した内容を覚える
- 🛠️ カスタムツールを使う（Web検索や天気確認など）
- 🧠 複雑な問題を段階的に考える
- 🤝 チームとして他のエージェントと協力する
- 🔌 様々なAIプロバイダー（OpenAI、Ollama、Azure）に接続する

**構成モジュール：**
- **エージェント（Agents）**：特定の仕事を持つAIアシスタント
- **ツール（Tools）**：彼らに授ける特別な能力
- **メモリ（Memory）**：会話を忘れないようにする仕組み
- **推論（Reasoning）**：ただ答えるだけでなく、考えることを教える

## 問題が起きたとき（対処法）🔧

### 「Alexが読み込めない！メモリ不足！」
**対処法**：コンピュータが苦戦中。他のアプリを閉じるか、より小さいモデルに切り替えましょう。8GBは最低限のメモリ要件。

### 「Alexが遅い」
**対処法**：Ollamaの設定でGPUアクセラレーションを有効に。もしくはコンテキストウィンドウサイズを減らす。スピードモード発動！🏎️

### 「ツールが動かない！」
**対処法**：関数のシグネチャをよくチェック。Alexがツールの役割を理解するには正しい型ヒントが必要。明確な指示を与えるイメージで。
