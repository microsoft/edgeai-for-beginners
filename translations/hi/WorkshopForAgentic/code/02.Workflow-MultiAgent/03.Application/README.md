# पॉडकास्ट एप्लीकेशन

एआई एजेंट्स का उपयोग करके पॉडकास्ट स्क्रिप्ट बनाने के लिए एक कंसोल एप्लीकेशन।

## उपयोग

```bash
python podcast_app.py
```

## कार्यप्रवाह

1. **स्वागत** - एप्लीकेशन उपयोगकर्ता का अभिवादन करता है
2. **विषय इनपुट** - उपयोगकर्ता पॉडकास्ट के लिए एक विषय प्रदान करता है
3. **सर्च एजेंट** - प्रासंगिक जानकारी खोजता है
4. **स्क्रिप्ट जनरेट एजेंट** - पॉडकास्ट स्क्रिप्ट बनाता है
5. **समीक्षा** - उपयोगकर्ता स्क्रिप्ट की समीक्षा करता है और उसे मंजूरी/अस्वीकृति देता है
6. **सहेजें** - स्वीकृत स्क्रिप्ट `podcast.md` में सहेजी जाती है

## आवश्यकताएँ

- पायथन 3.12+
- agent_framework
- 02.WorkflowDevUI से सभी निर्भरताएँ
