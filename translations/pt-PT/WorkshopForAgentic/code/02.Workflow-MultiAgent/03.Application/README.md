# Aplicação de Podcast

Uma aplicação de consola para gerar roteiros de podcast usando agentes de IA.

## Utilização

```bash
python podcast_app.py
```

## Fluxo de Trabalho

1. **Boas-vindas** - A aplicação cumprimenta o utilizador
2. **Entrada de Tema** - O utilizador fornece um tema para o podcast
3. **Agente de Pesquisa** - Pesquisa informação relevante
4. **Agente de Geração de Roteiro** - Cria um roteiro para o podcast
5. **Revisão** - O utilizador revê e aprova/rejeita o roteiro
6. **Guardar** - Roteiro aprovado é guardado em `podcast.md`

## Requisitos

- Python 3.12+
- agent_framework
- Todas as dependências de 02.WorkflowDevUI
