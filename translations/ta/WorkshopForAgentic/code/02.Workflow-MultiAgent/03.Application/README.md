# பாக்காஸ்ட் பயன்பாடானது

AI முகவர்கள் பயன்படுத்தி பாக்காஸ்ட் கதைசெய்திகள் உருவாக்கும் ஒரு கான்சோல் பயன்பாடு.

## பயன்பாடு

```bash
python podcast_app.py
```

## பணியாற்றல் முறைகள்

1. **வரவேற்க்கல்** - பயன்பாடு பயனருக்கு வணக்கம் கூறுகிறது
2. **தலைப்பு உள்ளீடு** - பயனர் பாக்காஸ்டுக்கான தலைப்பை வழங்குகிறான்
3. **தேடல் முகவர்** - தொடர்புடைய தகவல்களை தேடுகிறது
4. **கதைசெய்தி உருவாக்கும் முகவர்** - பாக்காஸ்ட் கதைசெய்தியை உருவாக்குகிறது
5. **மதிப்பாய்வு** - பயனர் கதைசெய்தியை மதிப்பாய்வு செய்து ஒப்புதல்/நிராகரிப்பு செய்கிறான்
6. **சேமி** - ஒப்புக் கொள்ளப்பட்ட கதைசெய்தி `podcast.md` என்ற கோப்பில் சேமிக்கப்படுகிறது

## தேவைகள்

- Python 3.12+
- agent_framework
- 02.WorkflowDevUI இல் உள்ள அனைத்து சார்புகளும்
