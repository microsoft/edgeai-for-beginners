# אפליקציית פודקאסט

אפליקציית קונסולה ליצירת תסריטי פודקאסט באמצעות סוכני בינה מלאכותית.

## שימוש

```bash
python podcast_app.py
```

## תהליך עבודה

1. **ברוכים הבאים** - האפליקציה מברכת את המשתמש
2. **הזנת נושא** - המשתמש מזין נושא לפודקאסט
3. **סוכן חיפוש** - מחפש מידע רלוונטי
4. **סוכן יצירת תסריט** - יוצר תסריט לפודקאסט
5. **סקירה** - המשתמש בודק ומאשר/דוחה את התסריט
6. **שמירה** - התסריט המאושר נשמר בקובץ `podcast.md`

## דרישות

- Python 3.12+
- agent_framework
- כל התלויות מ- 02.WorkflowDevUI
