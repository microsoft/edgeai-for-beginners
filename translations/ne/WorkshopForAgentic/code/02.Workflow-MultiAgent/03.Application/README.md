# पोडकास्ट अनुप्रयोग

एआई एजेन्टहरूको प्रयोग गरेर पोडकास्ट स्क्रिप्टहरू उत्पादन गर्नका लागि कन्सोल अनुप्रयोग।

## प्रयोग

```bash
python podcast_app.py
```

## कार्यप्रवाह

1. **स्वागत छ** - अनुप्रयोगले प्रयोगकर्तालाई स्वागत गर्दछ
2. **विषय इनपुट** - प्रयोगकर्ताले पोडकास्टको विषय प्रदान गर्दछ
3. **खोज एजेन्ट** - सान्दर्भिक जानकारी खोज्छ
4. **स्क्रिप्ट उत्पादन गर्ने एजेन्ट** - पोडकास्ट स्क्रिप्ट बनाउँछ
5. **समीक्षा** - प्रयोगकर्ताले स्क्रिप्ट समीक्षा गरी स्वीकृत/अस्वीकृत गर्छ
6. **सेभ गर्नुहोस्** - स्वीकृत स्क्रिप्ट `podcast.md` मा सेभ गरिन्छ

## आवश्यकताहरू

- Python 3.12+
- agent_framework
- 02.WorkflowDevUI बाट सबै निर्भरता
