## ಭದ್ರತೆ

Microsoft ನಮ್ಮ ಸಾಫ್ಟ್‌ವೇರ್ ಉತ್ಪನ್ನಗಳು ಮತ್ತು ಸೇವೆಗಳ ಭದ್ರತೆಯನ್ನು ಗಂಭೀರವಾಗಿ ತೆಗೆದುಕೊಳ್ಳುತ್ತದೆ, ಇದರಲ್ಲಿ ನಮ್ಮ GitHub ಸಂಸ್ಥೆಗಳಲ್ಲಿನ ಎಲ್ಲಾ ಮೂಲ ಕೋಡ್ ಸಂಗ್ರಹಣೆಗಳು ಸೇರಿವೆ.

**ದಯವಿಟ್ಟು ಸಾರ್ವಜನಿಕ GitHub ಸಮಸ್ಯೆಗಳ ಮೂಲಕ ಭದ್ರತಾ ದುರ್ಬಲತೆಗಳನ್ನು ವರದಿ ಮಾಡಬೇಡಿ.**

ಭದ್ರತಾ ವರದಿ ಮಾಹಿತಿ, ಸ್ಥಳಗಳು, ಸಂಪರ್ಕ ಮಾಹಿತಿ ಮತ್ತು ನೀತಿಗಳಿಗಾಗಿ, ದಯವಿಟ್ಟು Microsoft ಸಂಗ್ರಹಣೆಗಳ ಇತ್ತೀಚಿನ ಮಾರ್ಗದರ್ಶನವನ್ನು ಪರಿಶೀಲಿಸಿ [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
