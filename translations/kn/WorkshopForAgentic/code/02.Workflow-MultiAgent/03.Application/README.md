# ಪಾಡ್‌ಕಾಸ್ಟ್ ಅಪ್ಲಿಕೇಶನ್

ಎಐ ಏಜೆಂಟ್ಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಪಾಡ್‌ಕಾಸ್ಟ್ ಸ್ಕ್ರಿಪ್ಟ್‌ಗಳನ್ನು ಉತ್ಪಾದಿಸುವ ಕನ್ಸೋಲ್ ಅಪ್ಲಿಕೇಶನ್.

## ಬಳಕೆ

```bash
python podcast_app.py
```

## ವರ್ಕ್‌ಫ್ಲೋ

1. **ಸ್ವಾಗತ** - ಅಪ್ಲಿಕೇಶನ್ ಬಳಕೆದಾರರನ್ನು ಸ್ವಾಗತಿಸುತ್ತದೆ  
2. **ವಿಷಯ ನವೀಕರಣೆ** - ಬಳಕೆದಾರ ಪಾಡ್‌ಕಾಸ್ಟ್‌ಗೆ ವಿಷಯ ಒದಗಿಸುತ್ತಾರೆ  
3. **ಹುಡುಕು ಏಜೆಂಟ್** - ಸಂಬಂಧಿತ ಮಾಹಿತಿಯನ್ನು ಹುಡುಕುತ್ತದೆ  
4. **ಸ್ಕ್ರಿಪ್ಟ್ ರಚನೆ ಏಜೆಂಟ್** - ಪಾಡ್‌ಕಾಸ್ಟ್ ಸ್ಕ್ರಿಪ್ಟ್ ರಚಿಸುತ್ತದೆ  
5. **ಮೌಲ್ಯಮಾಪನ** - ಬಳಕೆದಾರ ಸ್ಕ್ರಿಪ್ಟ್ ಪರಿಶೀಲಿಸಿ ಅಂಗೀಕಾರ/ನಿರಾಕರಣೆ ಮಾಡುತ್ತಾರೆ  
6. **ಸಂಗ್ರಹಣೆ** - ಅಂಗೀಕೃತ ಸ್ಕ್ರಿಪ್ಟ್ ಅನ್ನು `podcast.md` ಗೆ ಉಳಿಸಲಾಗುತ್ತದೆ  

## ಅವಶ್ಯಕತೆಗಳು

- Python 3.12+  
- agent_framework  
- 02.WorkflowDevUI ಯಲ್ಲಿನ ಎಲ್ಲ ನಿರ্ভರತೆಗಳು
