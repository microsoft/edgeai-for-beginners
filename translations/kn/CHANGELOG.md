# ಬದಲಾವಣೆಗಳ ಚರಿತ್ರೆ

EdgeAI for Beginners ಗೆ ಸಂಬಂಧಿಸಿದ ಎಲ್ಲಾ ಪ್ರಮುಖ ಬದಲಾವಣೆಗಳನ್ನು ಇಲ್ಲಿ ದಾಖಲಿಸಲಾಗಿದೆ. ಈ ಯೋಜನೆ ದಿನಾಂಕ ಆಧಾರಿತ ಎಂಟ್ರಿಗಳನ್ನು ಮತ್ತು Keep a Changelog ಶೈಲಿಯನ್ನು (ಸೇರಿಸಲಾಗಿದೆ, ಬದಲಾಯಿಸಲಾಗಿದೆ, ಸರಿಪಡಿಸಲಾಗಿದೆ, ತೆಗೆದುಹಾಕಲಾಗಿದೆ, ಡಾಕ್ಸ್, ಸ್ಥಳಾಂತರಿಸಲಾಗಿದೆ) ಬಳಸುತ್ತದೆ.

## 2025-10-30

### ಸೇರಿಸಲಾಗಿದೆ - Module06 AI ಏಜೆಂಟ್ಸ್ ಸಮಗ್ರ ಸುಧಾರಣೆ
- **Microsoft Agent Framework ಏಕೀಕರಣ** (`Module06/01.IntroduceAgent.md`):
  - ಉತ್ಪಾದನೆಗೆ ಸಿದ್ಧ ಏಜೆಂಟ್ ಅಭಿವೃದ್ಧಿಗಾಗಿ Microsoft Agent Framework ಕುರಿತು ಸಂಪೂರ್ಣ ವಿಭಾಗ
  - ಎಡ್ಜ್ ನಿಯೋಜನೆಗಾಗಿ Foundry Local ಜೊತೆಗೆ ವಿವರವಾದ ಏಕೀಕರಣ ಮಾದರಿಗಳು
  - ವಿಶೇಷ SLM ಮಾದರಿಗಳೊಂದಿಗೆ ಬಹು-ಏಜೆಂಟ್ ಸಂಯೋಜನೆ ಉದಾಹರಣೆಗಳು
  - ಸಂಪನ್ಮೂಲ ನಿರ್ವಹಣೆ ಮತ್ತು ಮೇಲ್ವಿಚಾರಣೆಯೊಂದಿಗೆ ಎಂಟರ್‌ಪ್ರೈಸ್ ನಿಯೋಜನೆ ಮಾದರಿಗಳು
  - ಎಡ್ಜ್ ಏಜೆಂಟ್ ವ್ಯವಸ್ಥೆಗಳಿಗಾಗಿ ಭದ್ರತೆ ಮತ್ತು ಅನುಕೂಲತೆ ವೈಶಿಷ್ಟ್ಯಗಳು
  - ವಾಸ್ತವಿಕ ಜಾಗತಿಕ ಅನುಷ್ಠಾನ ಉದಾಹರಣೆಗಳು (ರಿಟೇಲ್, ಆರೋಗ್ಯಸೇವೆ, ಗ್ರಾಹಕ ಸೇವೆ)

- **ಉತ್ಪಾದನಾ SLM ಏಜೆಂಟ್ ನಿಯೋಜನೆ ತಂತ್ರಗಳು**:
  - **Foundry Local**: ಸ್ಥಾಪನೆ, ಸಂರಚನೆ ಮತ್ತು ಉತ್ಪಾದನಾ ಮಾದರಿಗಳೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಎಂಟರ್‌ಪ್ರೈಸ್-ಗ್ರೇಡ್ ಎಡ್ಜ್ AI ರನ್‌ಟೈಮ್ ಡಾಕ್ಯುಮೆಂಟೇಶನ್
  - **Ollama**: ಸಮುದಾಯ-ಕೇಂದ್ರೀಕೃತ ನಿಯೋಜನೆಗಾಗಿ ಸುಧಾರಿತ ಮೇಲ್ವಿಚಾರಣೆ ಮತ್ತು ಮಾದರಿ ನಿರ್ವಹಣೆ
  - **VLLM**: ಉನ್ನತ ಕಾರ್ಯಕ್ಷಮತೆಯ ಇನ್ಫರೆನ್ಸ್ ಎಂಜಿನ್, ಸುಧಾರಿತ ಆಪ್ಟಿಮೈಜೆಷನ್ ತಂತ್ರಗಳು ಮತ್ತು ಎಂಟರ್‌ಪ್ರೈಸ್ ವೈಶಿಷ್ಟ್ಯಗಳು
  - ಎಲ್ಲಾ ಮೂರು ವೇದಿಕೆಗಳಿಗಾಗಿ ಉತ್ಪಾದನಾ ನಿಯೋಜನೆ ಪರಿಶೀಲನಾ ಪಟ್ಟಿಗಳು ಮತ್ತು ಹೋಲಿಕೆ ಪಟ್ಟಿಗಳು

- **ಎಡ್ಜ್-ಆಪ್ಟಿಮೈಸ್ ಮಾಡಿದ SLM ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳ ಸುಧಾರಣೆ**:
  - **ONNX Runtime**: ಕ್ರಾಸ್-ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ SLM ಏಜೆಂಟ್ ನಿಯೋಜನೆಗಾಗಿ ಹೊಸ ಸಂಪೂರ್ಣ ವಿಭಾಗ
  - Windows, Linux, macOS, iOS ಮತ್ತು Android ನಲ್ಲಿ ಸರ್ವತ್ರ ನಿಯೋಜನೆ ಮಾದರಿಗಳು
  - ಸ್ವಯಂಚಾಲಿತ ಪತ್ತೆಗೊಳಿಸುವಿಕೆ ಜೊತೆಗೆ ಹಾರ್ಡ್‌ವೇರ್ ವೇಗವರ್ಧನೆ ಆಯ್ಕೆಗಳು (CPU, GPU, NPU)
  - ಉತ್ಪಾದನೆಗೆ ಸಿದ್ಧ ವೈಶಿಷ್ಟ್ಯಗಳು ಮತ್ತು ಏಜೆಂಟ್-ನಿರ್ದಿಷ್ಟ ಆಪ್ಟಿಮೈಜೆಷನ್‌ಗಳು
  - Microsoft Agent Framework ಏಕೀಕರಣದೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಅನುಷ್ಠಾನ ಉದಾಹರಣೆಗಳು

- **ಸೂಚನೆಗಳು ಮತ್ತು ಮುಂದಿನ ಓದು**:
  - 100+ ಪ್ರಾಧಿಕೃತ ಮೂಲಗಳೊಂದಿಗೆ ಸಮಗ್ರ ಸಂಪನ್ಮೂಲ ಗ್ರಂಥಾಲಯ
  - AI ಏಜೆಂಟ್‌ಗಳು ಮತ್ತು ಸಣ್ಣ ಭಾಷಾ ಮಾದರಿಗಳ ಕುರಿತ ಪ್ರಮುಖ ಸಂಶೋಧನಾ ಪತ್ರಿಕೆಗಳು
  - ಎಲ್ಲಾ ಪ್ರಮುಖ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳು ಮತ್ತು ಉಪಕರಣಗಳ ಅಧಿಕೃತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್
  - ಕೈಗಾರಿಕಾ ವರದಿಗಳು, ಮಾರುಕಟ್ಟೆ ವಿಶ್ಲೇಷಣೆ ಮತ್ತು ತಾಂತ್ರಿಕ ಬೆಂಚ್ಮಾರ್ಕ್‌ಗಳು
  - ಶೈಕ್ಷಣಿಕ ಸಂಪನ್ಮೂಲಗಳು, ಸಮ್ಮೇಳನಗಳು ಮತ್ತು ಸಮುದಾಯ ವೇದಿಕೆಗಳು
  - ಮಾನದಂಡಗಳು, ವಿಶೇಷಣಗಳು ಮತ್ತು ಅನುಕೂಲತೆ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳು

### ಬದಲಾಯಿಸಲಾಗಿದೆ - Module06 ವಿಷಯ ಆಧುನೀಕರಣ
- **ಸುಧಾರಿತ ಕಲಿಕೆ ಗುರಿಗಳು**: Microsoft Agent Framework ನ ಪರಿಣತಿ ಮತ್ತು ಎಡ್ಜ್ ನಿಯೋಜನೆ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- **ಉತ್ಪಾದನಾ ಕೇಂದ್ರೀಕರಣ**: ಕಲ್ಪನೆಗಿಂತ ಅನುಷ್ಠಾನಕ್ಕೆ ಸಿದ್ಧ ಮಾರ್ಗದರ್ಶನಕ್ಕೆ ಬದಲಾಯಿಸಲಾಗಿದೆ, ಉತ್ಪಾದನಾ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ
- **ಕೋಡ್ ಉದಾಹರಣೆಗಳು**: ಎಲ್ಲಾ ಉದಾಹರಣೆಗಳನ್ನು ಆಧುನಿಕ SDK ಮಾದರಿಗಳು ಮತ್ತು ಉತ್ತಮ ಅಭ್ಯಾಸಗಳಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳು**: ಹೈರಾರ್ಕಿಕಲ್ ಏಜೆಂಟ್ ವಾಸ್ತುಶಿಲ್ಪ ಮತ್ತು ಎಡ್ಜ್-ಟು-ಕ್ಲೌಡ್ ಸಂಯೋಜನೆ ಸೇರಿಸಲಾಗಿದೆ
- **ಕಾರ್ಯಕ್ಷಮತೆ ಆಪ್ಟಿಮೈಜೆಷನ್**: ಸಂಪನ್ಮೂಲ ನಿರ್ವಹಣೆ ಮತ್ತು ಸ್ವಯಂ-ಸ್ಕೇಲಿಂಗ್ ಶಿಫಾರಸುಗಳೊಂದಿಗೆ ಸುಧಾರಿಸಲಾಗಿದೆ

### ಡಾಕ್ಸ್ - Module06 ರಚನೆ ಸುಧಾರಣೆ
- **ಸಮಗ್ರ ಏಜೆಂಟ್ ಫ್ರೇಮ್ವರ್ಕ್ ವ್ಯಾಪ್ತಿ**: ಮೂಲ ಕಲ್ಪನೆಗಳಿಂದ ಎಂಟರ್‌ಪ್ರೈಸ್ ನಿಯೋಜನೆವರೆಗೆ
- **ಉತ್ಪಾದನಾ ನಿಯೋಜನೆ ತಂತ್ರಗಳು**: Foundry Local, Ollama ಮತ್ತು VLLM ಗೆ ಸಂಪೂರ್ಣ ಮಾರ್ಗದರ್ಶಿಗಳು
- **ಕ್ರಾಸ್-ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಆಪ್ಟಿಮೈಜೆಷನ್**: ಸರ್ವತ್ರ ನಿಯೋಜನೆಗಾಗಿ ONNX Runtime ಸೇರಿಸಲಾಗಿದೆ
- **ಸಂಪನ್ಮೂಲ ಗ್ರಂಥಾಲಯ**: ಮುಂದುವರಿದ ಕಲಿಕೆ ಮತ್ತು ಅನುಷ್ಠಾನಕ್ಕಾಗಿ ವ್ಯಾಪಕ ಸೂಚನೆಗಳು

### ಸೇರಿಸಲಾಗಿದೆ - Module06 ಮಾದರಿ ಸನ್ನಿವೇಶ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ನವೀಕರಣ
- **MCP ಪರಿಚಯ ಆಧುನೀಕರಣ** (`Module06/03.IntroduceMCP.md`):
  - modelcontextprotocol.io ನಿಂದ ಇತ್ತೀಚಿನ MCP ವಿಶೇಷಣಗಳೊಂದಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ (2025-06-18 ಆವೃತ್ತಿ)
  - ಮಾನಕೃತ AI ಅಪ್ಲಿಕೇಶನ್ ಸಂಪರ್ಕಗಳಿಗಾಗಿ ಅಧಿಕೃತ USB-C ಉಪಮೆಯನ್ನು ಸೇರಿಸಲಾಗಿದೆ
  - ಅಧಿಕೃತ ಎರಡು-ಮಟ್ಟದ ವಿನ್ಯಾಸ (ಡೇಟಾ ಲೇಯರ್ + ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಲೇಯರ್) ಜೊತೆಗೆ ವಾಸ್ತುಶಿಲ್ಪ ವಿಭಾಗ ನವೀಕರಿಸಲಾಗಿದೆ
  - ಸರ್ವರ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು (ಉಪಕರಣಗಳು, ಸಂಪನ್ಮೂಲಗಳು, ಪ್ರಾಂಪ್ಟ್‌ಗಳು) ಮತ್ತು ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು (ಸ್ಯಾಂಪ್ಲಿಂಗ್, ಎಲಿಸಿಟೇಶನ್, ಲಾಗಿಂಗ್) ಜೊತೆಗೆ ಕೋರ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಸುಧಾರಿಸಲಾಗಿದೆ

- **ಸಮಗ್ರ MCP ಸೂಚನೆಗಳು ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳು**:
  - **MCP for Beginners** ಲಿಂಕ್ ಸೇರಿಸಲಾಗಿದೆ (https://aka.ms/mcp-for-beginners)
  - ಅಧಿಕೃತ MCP ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮತ್ತು ವಿಶೇಷಣಗಳು (modelcontextprotocol.io)
  - MCP ಇನ್ಸ್‌ಪೆಕ್ಟರ್ ಮತ್ತು ಉಲ್ಲೇಖ ಅನುಷ್ಠಾನಗಳನ್ನು ಒಳಗೊಂಡ ಅಭಿವೃದ್ಧಿ ಸಂಪನ್ಮೂಲಗಳು
  - ತಾಂತ್ರಿಕ ಮಾನದಂಡಗಳು (JSON-RPC 2.0, JSON Schema, OpenAPI, Server-Sent Events)

### ಸೇರಿಸಲಾಗಿದೆ - Module04 Qualcomm QNN ಏಕೀಕರಣ
- **ಹೊಸ ವಿಭಾಗ 7: Qualcomm QNN ಆಪ್ಟಿಮೈಜೆಷನ್ ಸೂಟ್** (`Module04/05.QualcommQNN.md`):
  - Qualcomm ನ ಏಕೀಕೃತ AI ಇನ್ಫರೆನ್ಸ್ ಫ್ರೇಮ್ವರ್ಕ್ ಕುರಿತು 400+ ಸಾಲುಗಳ ಸಮಗ್ರ ಮಾರ್ಗದರ್ಶಿ
  - ವಿಭಿನ್ನ ಗಣನೆ (Hexagon NPU, Adreno GPU, Kryo CPU) ಕುರಿತು ವಿವರವಾದ ವ್ಯಾಪ್ತಿ
  - Snapdragon ವೇದಿಕೆಗಳಿಗೆ ಹಾರ್ಡ್‌ವೇರ್-ಜಾಗರೂಕ ಆಪ್ಟಿಮೈಜೆಷನ್ ಮತ್ತು ಬುದ್ಧಿವಂತ ಕೆಲಸ ಹಂಚಿಕೆ
  - ಮೊಬೈಲ್ ನಿಯೋಜನೆಗಾಗಿ ಉನ್ನತ ಮಟ್ಟದ ಕ್ವಾಂಟೈಜೆಷನ್ ತಂತ್ರಗಳು (INT8, INT16, ಮಿಶ್ರ-ಪ್ರಿಸಿಷನ್)
  - ಬ್ಯಾಟರಿ ಚಾಲಿತ ಸಾಧನಗಳು ಮತ್ತು ರಿಯಲ್-ಟೈಮ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳಿಗೆ ವಿದ್ಯುತ್-ಕಾರ್ಯಕ್ಷಮ ಇನ್ಫರೆನ್ಸ್ ಆಪ್ಟಿಮೈಜೆಷನ್
  - QNN SDK ಸೆಟ್‌ಅಪ್ ಮತ್ತು ಪರಿಸರ ಸಂರಚನೆಯೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಸ್ಥಾಪನೆ ಮಾರ್ಗದರ್ಶಿ
  - ಪ್ರಾಯೋಗಿಕ ಉದಾಹರಣೆಗಳು: PyTorch ನಿಂದ QNN ಪರಿವರ್ತನೆ, ಬಹು-ಬ್ಯಾಕ್‌ಎಂಡ್ ಆಪ್ಟಿಮೈಜೆಷನ್, ಸನ್ನಿವೇಶ ಬೈನರಿ ತಯಾರಿಕೆ
  - ಸುಧಾರಿತ ಬಳಕೆ ಮಾದರಿಗಳು: ಕಸ್ಟಮ್ ಬ್ಯಾಕ್‌ಎಂಡ್ ಸಂರಚನೆ, ಡೈನಾಮಿಕ್ ಕ್ವಾಂಟೈಜೆಷನ್, ಕಾರ್ಯಕ್ಷಮತೆ ಪ್ರೊಫೈಲಿಂಗ್
  - ಸಮಗ್ರ ತೊಂದರೆ ಪರಿಹಾರ ವಿಭಾಗ ಮತ್ತು ಸಮುದಾಯ ಸಂಪನ್ಮೂಲಗಳು

- **ಸುಧಾರಿತ Module04 ರಚನೆ**:

  - ಮೊಬೈಲ್ AI ನಿಯೋಜನೆ ಮತ್ತು ವಿದ್ಯುತ್ ಆಪ್ಟಿಮೈಜೆಷನ್ ಗುರಿಗಳ ಸಮಗ್ರ ಕಲಿಕೆ ಫಲಿತಾಂಶಗಳು

### ಡಾಕ್ಸ್ - ಕಾರ್ಯಾಗಾರ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಸುಧಾರಣೆ

  - ಕಲಿಕೆ ಗುರಿಗಳು ಮತ್ತು ಅಧ್ಯಯನ ಕೇಂದ್ರೀಕೃತ ಪ್ರದೇಶಗಳು
  - ಸ್ವಯಂ-ಮೌಲ್ಯಮಾಪನ ಪ್ರಶ್ನೆಗಳು
  - ಸಮಯ ಅಂದಾಜುಗಳೊಂದಿಗೆ ಕೈಗೊಳ್ಳುವ ಅಭ್ಯಾಸಗಳು
  - ಕೇಂದ್ರೀಕೃತ ಮತ್ತು ಭಾಗಕಾಲಿಕ ಅಧ್ಯಯನಕ್ಕೆ ಸಮಯ ಹಂಚಿಕೆ

- README ಗೆ ಕಾರ್ಯಾಗಾರ ಮಾದರಿ ವಿವರಣೆಗಳು ಮತ್ತು ಕಲಿಕೆ ಫಲಿತಾಂಶಗಳು ಸೇರಿಸಲಾಗಿದೆ

### ಸರಿಪಡಿಸಲಾಗಿದೆ
