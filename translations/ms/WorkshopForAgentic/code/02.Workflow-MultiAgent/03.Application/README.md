# Aplikasi Podcast

Satu aplikasi konsol untuk menjana skrip podcast menggunakan ejen AI.

## Penggunaan

```bash
python podcast_app.py
```

## Aliran Kerja

1. **Selamat Datang** - Aplikasi menyapa pengguna
2. **Input Topik** - Pengguna memberikan topik untuk podcast
3. **Ejen Carian** - Mencari maklumat yang berkaitan
4. **Ejen Jana Skrip** - Membuat skrip podcast
5. **Semak** - Pengguna menyemak dan meluluskan/menghapuskan skrip
6. **Simpan** - Skrip yang diluluskan disimpan ke dalam `podcast.md`

## Keperluan

- Python 3.12+
- agent_framework
- Semua kebergantungan dari 02.WorkflowDevUI
