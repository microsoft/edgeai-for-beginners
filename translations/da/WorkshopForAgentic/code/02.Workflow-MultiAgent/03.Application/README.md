# Podcast-applikation

En konsolapplikation til generering af podcastmanuskripter ved hjælp af AI-agenter.

## Brug

```bash
python podcast_app.py
```

## Arbejdsgang

1. **Velkomst** - Applikationen byder brugeren velkommen
2. **Emneindgang** - Brugeren angiver et emne til podcasten
3. **Søgeagent** - Søger efter relevant information
4. **Generér manuskript-agent** - Opretter et podcastmanuskript
5. **Gennemgang** - Brugeren gennemgår og godkender/afviser manuskriptet
6. **Gem** - Godkendt manuskript gemmes i `podcast.md`

## Krav

- Python 3.12+
- agent_framework
- Alle afhængigheder fra 02.WorkflowDevUI
