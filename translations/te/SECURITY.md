## భద్రత

మైక్రోసాఫ్ట్ మా సాఫ్ట్‌వేర్ ఉత్పత్తులు మరియు సేవల భద్రతను గంభీరంగా తీసుకుంటుంది, దీనిలో మా GitHub సంస్థలలోని అన్ని సోర్స్ కోడ్ రిపాజిటరీలు కూడా ఉన్నాయి.

**దయచేసి భద్రతా లోపాలను పబ్లిక్ GitHub ఇష్యూల ద్వారా నివేదించవద్దు.**

భద్రతా నివేదిక సమాచారం, ప్రదేశాలు, సంప్రదింపు సమాచారం మరియు విధానాల కోసం, దయచేసి మైక్రోసాఫ్ట్ రిపాజిటరీల తాజా మార్గదర్శకాలను [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md) వద్ద సమీక్షించండి.
