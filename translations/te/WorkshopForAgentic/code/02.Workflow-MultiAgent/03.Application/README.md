# పోడ్కాస్ట్ అప్లికేషన్

AI ఏజెంట్లు ఉపయోగించి పోడ్కాస్ట్ స్క్రిప్ట్‌లను తయారు చేసే ఒక కన్సోల్ అప్లికేషన్.

## ఉపయోగం

```bash
python podcast_app.py
```

## పనివిధానం

1. **స్వాగతం** - అప్లికేషన్ వినియోగదారుని స్వాగతిస్తుంది  
2. **విషయ ఇన్పుట్** - వినియోగదారు పోడ్కాస్ట్‌కు ఒక విషయం ఇస్తాడు  
3. **సేర్చ్ ఏజెంట్** - సంబంధించిన సమాచారాన్ని వెతుకుతుంది  
4. **జెనరేట్ స్క్రిప్ట్ ఏజెంట్** - పోడ్కాస్ట్ స్క్రిప్ట్‌ను సృష్టిస్తుంది  
5. **సమీక్ష** - వినియోగదారు స్క్రిప్ట్‌ను సమీక్షించి ఆమోదించు/అంగీకారించకపోవచ్చు  
6. **సేవ్** - ఆమోదించిన స్క్రిప్ట్ `podcast.md`లో సేవ్ అవుతుంది  

## అవసరాలు

- Python 3.12+  
- agent_framework  
- 02.WorkflowDevUI నుండి అన్ని డిపెండెన్సీలు
