# ‡∞´‡±à-4-‡∞Æ‡∞ø‡∞®‡±Ä-‡∞á‡∞®‡±ç‚Äå‡∞∏‡±ç‡∞ü‡±ç‡∞∞‡∞ï‡±ç‡∞ü‡±ç‚Äå‡∞®‡±Å ‡∞≤‡±ã‡∞°‡±ç ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø

    "microsoft/Phi-4-mini-instruct",
    torch_dtype=torch.bfloat16,
    trust_remote_code=True,
    attn_implementation="flash_attention_2"  # ‡∞Æ‡±Ü‡∞∞‡±Å‡∞ó‡±à‡∞® ‡∞™‡∞®‡∞ø‡∞§‡±Ä‡∞∞‡±Å ‡∞ï‡±ã‡∞∏‡∞Ç

# ‡∞ö‡∞æ‡∞ü‡±ç ‡∞ü‡±Ü‡∞Ç‡∞™‡±ç‡∞≤‡±á‡∞ü‡±ç ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞Æ‡±Ä ‡∞™‡±ç‡∞∞‡∞æ‡∞Ç‡∞™‡±ç‡∞ü‡±ç‚Äå‡∞®‡±Å ‡∞´‡∞æ‡∞∞‡±ç‡∞Æ‡∞æ‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø
messages = [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Explain quantum computing in simple terms."}

# ‡∞ö‡∞æ‡∞ü‡±ç ‡∞ü‡±Ü‡∞Ç‡∞™‡±ç‡∞≤‡±á‡∞ü‡±ç‚Äå‡∞®‡±Å ‡∞µ‡∞∞‡±ç‡∞§‡∞ø‡∞Ç‡∞™‡∞ú‡±á‡∞Ø‡∞Ç‡∞°‡∞ø

```

)

# ‡∞∏‡∞æ‡∞Æ‡∞∞‡±ç‡∞•‡±ç‡∞Ø‡∞æ‡∞®‡±ç‡∞®‡∞ø ‡∞Æ‡±Ü‡∞∞‡±Å‡∞ó‡±Å‡∞™‡∞∞‡∞ö‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞ï‡∞æ‡∞®‡±ç‡∞´‡∞ø‡∞ó‡∞∞‡±á‡∞∑‡∞®‡±ç

    num_train_epochs=1,
    bf16=True,
    gradient_checkpointing=True,
    warmup_ratio=0.2,
    save_steps=100
)

# ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞ü‡±ã‡∞ï‡∞®‡±à‡∞ú‡∞∞‡±ç ‡∞≤‡±ã‡∞°‡±ç ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø
model = AutoModelForCausalLM.from_pretrained(
    "microsoft/Phi-4-mini-instruct",
    torch_dtype=torch.bfloat16,
    trust_remote_code=True
)

# ‡∞°‡±á‡∞ü‡∞æ‡∞∏‡±Ü‡∞ü‡±ç ‡∞≤‡±ã‡∞°‡±ç ‡∞ö‡±á‡∞∏‡∞ø ‡∞™‡±ç‡∞∞‡∞æ‡∞∏‡±Ü‡∞∏‡±ç ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø

# ‡∞ü‡±ç‡∞∞‡±à‡∞®‡∞∞‡±ç ‡∞™‡±ç‡∞∞‡∞æ‡∞∞‡∞Ç‡∞≠‡∞ø‡∞Ç‡∞ö‡∞Ç‡∞°‡∞ø

    model=model,
    args=training_config,

    packing=True

```
