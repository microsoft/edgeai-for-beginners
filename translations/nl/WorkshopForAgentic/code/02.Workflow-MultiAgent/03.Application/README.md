# Podcast Applicatie

Een console-applicatie voor het genereren van podcastscripts met behulp van AI-agenten.

## Gebruik

```bash
python podcast_app.py
```

## Workflow

1. **Welkom** - Applicatie begroet de gebruiker
2. **Onderwerp Invoer** - Gebruiker geeft een onderwerp voor de podcast op
3. **Zoek Agent** - Zoekt naar relevante informatie
4. **Genereer Script Agent** - Maakt een podcastscriptt
5. **Beoordeling** - Gebruiker beoordeelt en keurt het script goed/af
6. **Opslaan** - Goedgekeurd script wordt opgeslagen in `podcast.md`

## Vereisten

- Python 3.12+
- agent_framework
- Alle afhankelijkheden van 02.WorkflowDevUI
