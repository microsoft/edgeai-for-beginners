# Podcast Application

Консольное приложение для создания сценариев подкастов с использованием AI-агентов.

## Usage

```bash
python podcast_app.py
```

## Workflow

1. **Welcome** - Приложение приветствует пользователя
2. **Topic Input** - Пользователь вводит тему для подкаста
3. **Search Agent** - Производит поиск релевантной информации
4. **Generate Script Agent** - Создает сценарий подкаста
5. **Review** - Пользователь проверяет и одобряет/отклоняет сценарий
6. **Save** - Одобренный сценарий сохраняется в `podcast.md`

## Requirements

- Python 3.12+
- agent_framework
- Все зависимости из 02.WorkflowDevUI
