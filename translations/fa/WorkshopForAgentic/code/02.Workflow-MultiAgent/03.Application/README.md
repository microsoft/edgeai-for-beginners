# برنامه پادکست

یک برنامه کنسولی برای تولید اسکریپت‌های پادکست با استفاده از عامل‌های هوش مصنوعی.

## نحوه استفاده

```bash
python podcast_app.py
```

## روند کاری

1. **خوش‌آمدگویی** - برنامه به کاربر خوش‌آمد می‌گوید
2. **وارد کردن موضوع** - کاربر موضوعی برای پادکست ارائه می‌دهد
3. **عامل جستجو** - به دنبال اطلاعات مرتبط می‌گردد
4. **عامل تولید اسکریپت** - اسکریپت پادکست را ایجاد می‌کند
5. **بررسی** - کاربر اسکریپت را بررسی و تأیید یا رد می‌کند
6. **ذخیره‌سازی** - اسکریپت تأیید شده در `podcast.md` ذخیره می‌شود
