# Podcastová aplikácia

Konzolová aplikácia na generovanie podcastových scénárov pomocou AI agentov.

## Použitie

```bash
python podcast_app.py
```

## Postup práce

1. **Uvítanie** - Aplikácia pozdraví používateľa
2. **Vstup témy** - Používateľ zadá tému pre podcast
3. **Agent na vyhľadávanie** - Vyhľadáva relevantné informácie
4. **Agent na generovanie scenára** - Vytvára podcastový scenár
5. **Kontrola** - Používateľ si scenár prezrie a schváli/odmietne
6. **Uloženie** - Schválený scenár je uložený do `podcast.md`

## Požiadavky

- Python 3.12+
- agent_framework
- Všetky závislosti z 02.WorkflowDevUI
