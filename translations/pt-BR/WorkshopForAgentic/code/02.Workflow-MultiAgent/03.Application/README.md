# Aplicativo de Podcast

Um aplicativo de console para gerar roteiros de podcast usando agentes de IA.

## Uso

```bash
python podcast_app.py
```

## Fluxo de trabalho

1. **Boas-vindas** - Aplicativo cumprimenta o usuário
2. **Entrada de Tópico** - Usuário fornece um tópico para o podcast
3. **Agente de Pesquisa** - Pesquisa informações relevantes
4. **Agente de Geração de Roteiro** - Cria um roteiro de podcast
5. **Revisão** - Usuário revisa e aprova/rejeita o roteiro
6. **Salvar** - Roteiro aprovado é salvo em `podcast.md`

## Requisitos

- Python 3.12+
- agent_framework
- Todas as dependências de 02.WorkflowDevUI
