# Πράξη 1: Γνωρίστε τον βοηθό AI για έρευνα 🤖

![lab1](../../../../../translated_images/lab1.1dd5210a7d114e2e.el.png)

## Πρόκληση

Ετοιμάζετε το «Μελλοντικά Bytes», το καινούργιο σας τεχνολογικό podcast. Το πρώτο επεισόδιο αφορά τις τελευταίες εξελίξεις της AI, αλλά έχετε μόνο 24 ώρες να:
1. Ερευνήσετε το θέμα
2. Βρείτε αξιόπιστες πηγές
3. Γράψετε ένα συναρπαστικό σενάριο
4. Το κάνετε να ακούγεται φυσικό

**Στροφή στην πλοκή**: δεν χρειάζεται να το κάνετε μόνοι σας. Θα δημιουργήσετε τον πρώτο σας βοηθό AI, που μπορεί να βοηθήσει σε όλα αυτά. Ας τον ονομάσουμε Alex — ο ακούραστος σύντροφος έρευνας που δεν χρειάζεται ποτέ ύπνο.

## Γιατί να επιλέξετε μικρά γλωσσικά μοντέλα; (Υπόδειξη: Είναι υπέροχα)

Φανταστείτε τα μικρά γλωσσικά μοντέλα (SLM) ως προσωπικό AI που τρέχει στον *υπολογιστή σας*. Χωρίς cloud, χωρίς μηνιαία συνδρομή, χωρίς αμφίβολη κοινοποίηση δεδομένων.

**Γιατί τα SLM είναι τόσο 🔥:**
- **🏠 Τρέχουν στον υπολογιστή σας**: φορητό, desktop ή ακόμα και ισχυρό Raspberry Pi
- **💸 Μηδενικό συνεχές κόστος**: χωρίς έξοδα API να καταβροχθίζουν το πορτοφόλι σας
- **🔒 Ιδιωτικότητα πρώτα**: τα δεδομένα σας ποτέ δεν φεύγουν από τη συσκευή σας
- **⚡ Αστραπιαία γρήγορα**: χωρίς καθυστερήσεις δικτύου, άμεση απόκριση
- **🪦 Ελαφριά**: 1-10 δισεκατομμύρια παράμετροι έναντι 100+ δισ. στους μεγάλους

**Δημοφιλή SLM**: Qwen 3, Phi-4, Gemma 3 (σε αυτό το εργαστήριο χρησιμοποιούμε Qwen)

## Τα εργαλεία σου

### Ollama: ο διαχειριστής AI μοντέλων

[Ollama](https://ollama.com/) είναι σαν το Steam για AI μοντέλα. Κατεβάστε, τρέξτε και διαχειριστείτε μοντέλα με απλές εντολές.

**Τα ωραία του:**
- Μια εντολή για να κατεβάσετε και να τρέξετε οποιοδήποτε μοντέλο
- Υποστηρίζει Mac, Windows, Linux
- Αν έχετε GPU, τη χρησιμοποιεί αυτόματα
- Εξαιρετικά οικονομικό σε μνήμη

### Microsoft Agent Framework: το μέρος όπου συμβαίνει η μαγεία

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework) είναι το πάρκο αναψυχής για να χτίσετε AI πράκτορες που μπορούν:

- 💬 Να συνομιλούν και να θυμούνται τι λέτε
- 🛠️ Να χρησιμοποιούν προσαρμοσμένα εργαλεία (π.χ. αναζήτηση στο διαδίκτυο ή πρόβλεψη καιρού)
- 🧠 Να σκέφτονται βήμα προς βήμα πολύπλοκα προβλήματα
- 🤝 Να συνεργάζονται ως ομάδα με άλλους πράκτορες
- 🔌 Να συνδέονται με διαφορετικούς παρόχους AI (OpenAI, Ollama, Azure)

**Δομικά στοιχεία:**
- **Πράκτορες (Agents)**: βοηθοί AI με συγκεκριμένη αποστολή
- **Εργαλεία (Tools)**: ειδικές ικανότητες που τους δίνετε
- **Μνήμη (Memory)**: για να μην ξεχνούν τις συνομιλίες σας
- **Λογική (Reasoning)**: για να τους μάθετε να σκέφτονται και όχι απλά να απαντούν

## Το εκπαιδευτικό σας μοντάζ: 4 αποστολές

### Αποστολή 1: Δημιουργήστε τον πρώτο σας πράκτορα

📓 [Άνοιγμα Notebook](../../code/01.BasicAgent/00.BasicAgent-agent.ipynb)

**Αποστολή**: Δημιουργήστε τον Alex, τον AI σεναριογράφο για το podcast σας. Ο Alex πρέπει να δημιουργήσει μια συζήτηση μεταξύ δύο οικοδεσποτών για τεχνολογικά θέματα.

**Θα μάθετε:**
- Πώς να ξυπνήσετε έναν πράκτορα AI (πιο εύκολο από το να σηκωθείς Δευτέρα πρωί)
- Πώς να του δώσετε προσωπικότητα και οδηγίες
- Πώς να δημιουργήσετε ρεαλιστικά σενάρια podcast
- Πώς να κατανοήσετε τις απαντήσεις του

**Στόχος**: Ο Alex να γράψει το σενάριο για το πιλοτικό επεισόδιο του «Μελλοντικά Bytes» για AI! 🎯

### Αποστολή 2: Δώστε υπερδυνάμεις στον Alex (Εργαλεία!)

📓 [Άνοιγμα Notebook](../../code/01.BasicAgent/01.BasicAgent-tools.ipynb)

**Αποστολή**: Ο Alex είναι έξυπνος, αλλά δεν ξέρει τον καιρό ή τι ώρα είναι τώρα. Ας λύσουμε αυτό το πρόβλημα δίνοντάς του εργαλεία!

**Θα μάθετε:**
- Πώς να φτιάχνετε προσαρμοσμένες συναρτήσεις Python ως "εργαλεία"
- Πώς ο Alex αποφασίζει *πότε* να χρησιμοποιήσει κάθε εργαλείο
- Να τον παρακολουθείτε να λύνει προβλήματα μόνος του
- Συνδυασμό πολλών εργαλείων για σύνθετες εργασίες

**Στόχος**: Ρωτήστε "Πώς είναι ο καιρός στο Τόκιο;" και ο Alex να βρει την απάντηση μόνος του! ☁️

### Αποστολή 3: Διδάξτε τον Alex να σκέφτεται

📓 [Άνοιγμα Notebook](../../code/01.BasicAgent/02.BasicAgent-reasoning.ipynb)

**Αποστολή**: Αφήστε τον Alex να δείξει πώς δουλεύει. Όταν λύνει πρόβλημα, θέλετε να δείτε *πώς* σκέφτεται, όχι μόνο την απάντηση.

**Θα μάθετε:**
- Να ενεργοποιείτε τη "λογική" (σαν να δείχνετε τη λύση στα μαθηματικά)
- Να βλέπετε τη σταδιακή σκέψη του Alex
- Να κατανοείτε τα prompts αλυσίδας σκέψης
- Πώς να κάνετε debugging όταν ο Alex μπερδεύεται

**Στόχος**: Κάντε μια δύσκολη μαθηματική ερώτηση και δείτε τον Alex να το επεξεργάζεται! 🧠

### Αποστολή 4: Συνδέστε τον Alex στο διαδίκτυο

📓 [Άνοιγμα Notebook](../../code/01.BasicAgent/03.BasicAgent-websearch.ipynb)

**Αποστολή**: Οι γνώσεις του Alex έχουν ημερομηνία λήξης. Ας τον συνδέσουμε στο Διαδίκτυο για ζωντανές πληροφορίες!

**Θα μάθετε:**
- Πώς να φτιάχνετε προσαρμοσμένα εργαλεία αναζήτησης web
- Πώς να ενσωματώνετε εξωτερικά API
- Πώς να χειρίζεστε κομψά σφάλματα δικτύου
- Πώς να λαμβάνετε πληροφορίες πέραν των δεδομένων εκπαίδευσης του Alex

**Στόχος**: Ρωτήστε για τα σημερινά νέα τεχνολογίας και λάβετε ενημερωμένα αποτελέσματα! 📰

## Όταν κάτι πάει στραβά (και πώς να το διορθώσετε) 🔧

### "Ο Alex δεν φορτώνει! Λείπει μνήμη!"
**Λύση**: Ο υπολογιστής σας δυσκολεύεται. Δοκιμάστε να κλείσετε άλλες εφαρμογές ή να χρησιμοποιήσετε πιο μικρό μοντέλο. 8GB μνήμη είναι το ελάχιστο.

### "Ο Alex είναι πολύ αργός"
**Λύση**: Ενεργοποιήστε GPU acceleration στις ρυθμίσεις του Ollama. Ή μειώστε το μέγεθος του context window. Ενεργοποιήστε το mode ταχύτητας! 🏎️

### "Τα εργαλεία δεν δουλεύουν!"
**Λύση**: Ελέγξτε προσεκτικά την υπογραφή της συνάρτησης. Ο Alex χρειάζεται σωστές τύπου ενδείξεις για να καταλάβει κάθε εργαλείο. Σκεφτείτε το ως σαφείς οδηγίες.
