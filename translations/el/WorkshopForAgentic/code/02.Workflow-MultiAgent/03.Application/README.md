# Εφαρμογή Podcast

Μια κονσόλα εφαρμογή για τη δημιουργία σεναρίων podcast χρησιμοποιώντας AI πράκτορες.

## Χρήση

```bash
python podcast_app.py
```

## Ροή εργασίας

1. **Καλωσόρισμα** - Η εφαρμογή χαιρετά τον χρήστη
2. **Εισαγωγή Θέματος** - Ο χρήστης παρέχει ένα θέμα για το podcast
3. **Πράκτορας Αναζήτησης** - Αναζητά σχετικές πληροφορίες
4. **Πράκτορας Δημιουργίας Σεναρίου** - Δημιουργεί το σενάριο του podcast
5. **Ανασκόπηση** - Ο χρήστης ελέγχει και εγκρίνει/απορρίπτει το σενάριο
6. **Αποθήκευση** - Το εγκεκριμένο σενάριο αποθηκεύεται στο `podcast.md`

## Απαιτήσεις

- Python 3.12+
- agent_framework
- Όλες οι εξαρτήσεις από 02.WorkflowDevUI
