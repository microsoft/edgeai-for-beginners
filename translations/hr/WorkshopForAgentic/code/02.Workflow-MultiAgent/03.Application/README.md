# Podcast aplikacija

Konzolna aplikacija za generiranje podcast skripti koristeći AI agente.

## Upute za korištenje

```bash
python podcast_app.py
```

## Radni tijek

1. **Dobrodošlica** - Aplikacija pozdravlja korisnika
2. **Unos teme** - Korisnik unosi temu za podcast
3. **Agent za pretraživanje** - Traži relevantne informacije
4. **Agent za generiranje skripte** - Stvara podcast skriptu
5. **Pregled** - Korisnik pregledava i odobrava/odbacuje skriptu
6. **Spremanje** - Odobrena skripta se sprema u `podcast.md`

## Zahtjevi

- Python 3.12+
- agent_framework
- Sve ovisnosti iz 02.WorkflowDevUI
