# Ứng dụng Podcast

Một ứng dụng console để tạo kịch bản podcast bằng cách sử dụng các tác nhân AI.

## Cách sử dụng

```bash
python podcast_app.py
```

## Quy trình làm việc

1. **Chào mừng** - Ứng dụng chào hỏi người dùng
2. **Nhập chủ đề** - Người dùng cung cấp chủ đề cho podcast
3. **Tác nhân Tìm kiếm** - Tìm kiếm thông tin liên quan
4. **Tác nhân Tạo kịch bản** - Tạo kịch bản podcast
5. **Xem lại** - Người dùng xem xét và duyệt/từ chối kịch bản
6. **Lưu** - Kịch bản được duyệt sẽ được lưu vào `podcast.md`

## Yêu cầu

- Python 3.12+
- agent_framework
- Tất cả các phụ thuộc từ 02.WorkflowDevUI
