# Aplikacja Podcastowa

Konsolowa aplikacja do generowania scenariuszy podcastów za pomocą agentów AI.

## Użycie

```bash
python podcast_app.py
```

## Przebieg pracy

1. **Powitanie** - Aplikacja wita użytkownika  
2. **Wprowadzenie tematu** - Użytkownik podaje temat podcastu  
3. **Agent wyszukiwania** - Wyszukuje odpowiednie informacje  
4. **Agent generujący scenariusz** - Tworzy scenariusz podcastu  
5. **Przegląd** - Użytkownik przegląda i zatwierdza/odrzuca scenariusz  
6. **Zapis** - Zatwierdzony scenariusz jest zapisywany do `podcast.md`

## Wymagania

- Python 3.12+  
- agent_framework  
- Wszystkie zależności z 02.WorkflowDevUI
