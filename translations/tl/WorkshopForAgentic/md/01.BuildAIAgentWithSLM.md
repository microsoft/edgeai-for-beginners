## Bakit Small Language Models? (Spoiler: Ang Galing Nila)

Isipin ang Small Language Models (SLMs) bilang iyong personal na AI na nasa *iyong* computer. Walang ulap, walang buwanang bayad, walang kahalayan sa pagbabahagi ng data.

**Bakit patok ang SLMs:**
- **ğŸ  Tumatakbo sa Iyong Machine**: Laptop, desktop, pati ang malakas na Raspberry Pi
- **ğŸ’¸ Walang Patuloy na Gastos**: Walang API fees na kumakain ng pera mo
- **ğŸ”’ Pribadong Una**: Hindi lalabas ang iyong data sa device mo
- **âš¡ Napakabilis**: Walang internet lag, instant na sagot
- **ğŸª¦ Magaan**: 1B-10B parameters kumpara sa 100B+ ng mga malalaking modelo

**Mga Popular na SLMs**: Qwen 3, Phi-4, Gemma 3 (gagamitin natin ang Qwen para sa workshop na ito)

## Ang Iyong Mga Kasangkapan

### Ollama: Ang Iyong AI Model Manager

Ang [Ollama](https://ollama.com/) ay parang Steam para sa AI models. I-download, patakbuhin, at pamahalaan ang mga modelo gamit ang simple na mga utos.

**Mga dahilan kung bakit ito astig:**
- Isang utos lang para i-download at patakbuhin ang kahit anong modelo
- Gumagana sa Mac, Windows, Linux
- Awtomatikong ginagamit ang iyong GPU kung mayroon ka nito
- Super memory-efficient

### Microsoft Agent Framework: Dito Nangyayari ang Mahika

Ang [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) ay ang iyong palaruan para gumawa ng AI agents na kayang:

- ğŸ’¬ Makipag-chat at tandaan ang pinag-usapan ninyo
- ğŸ› ï¸ Gumamit ng mga custom na tools (gaya ng paghahanap sa web o pagsuri ng panahon)
- ğŸ§  Mag-isip ng masalimuot na mga problema nang hakbang-hakbang
- ğŸ¤ Makipagtulungan sa iba pang mga agent bilang isang koponan
- ğŸ”Œ Kumonekta sa iba't ibang AI providers (OpenAI, Ollama, Azure)

**Mga Pangunahing Bahagi:**
- **Agents**: Ang iyong mga AI assistant na may tiyak na gawain
- **Tools**: Mga espesyal na kakayahan na ibinibigay mo sa kanila
- **Memory**: Para hindi makalimutan ang inyong usapan
- **Reasoning**: Pagtuturo sa kanila na mag-isip, hindi lang sumagot
