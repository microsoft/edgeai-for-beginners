## Iyong Toolkit

### Ollama: Iyong AI Model Manager

[Ollama](https://ollama.com/) ay parang Steam para sa AI models. Gumamit ng simpleng commands para mag-download, mag-run, at mag-manage ng mga modelo.

**Ang Astig Nito:**
- Isang command lang para i-download at patakbuhin ang kahit anong model
- Available sa Mac, Windows, Linux
- Awtomatikong gumagamit ng GPU kung meron
- Super memory efficient

### Microsoft Agent Framework: Dito Nangyayari ang Magic

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework) ay isang playground para bumuo ng AI agents na kaya:

- ğŸ’¬ Makipag-chat at tandaan ang pinag-usapan

- ğŸ”Œ Kumonekta sa iba't ibang AI providers (OpenAI, Ollama, Azure)

**Mga building block:**
