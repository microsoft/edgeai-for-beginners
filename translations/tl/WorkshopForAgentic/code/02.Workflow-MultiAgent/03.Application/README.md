# Podcast Application

Isang console application para sa pagbuo ng mga script ng podcast gamit ang mga AI agent.

## Usage

```bash
python podcast_app.py
```

## Workflow

1. **Welcome** - Binabati ng application ang user
2. **Topic Input** - Nagbibigay ang user ng paksa para sa podcast
3. **Search Agent** - Naghahanap ng kaugnay na impormasyon
4. **Generate Script Agent** - Gumagawa ng script ng podcast
5. **Review** - Sine-review at ine-approve/reject ng user ang script
6. **Save** - Ang na-approve na script ay sine-save sa `podcast.md`

## Requirements

- Python 3.12+
- agent_framework
- Lahat ng dependencies mula sa 02.WorkflowDevUI
