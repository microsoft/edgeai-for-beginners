# 第一막：당신의 AI 연구 조수를 만나보세요 🤖

![lab1](../../../../../translated_images/ko/lab1.1dd5210a7d114e2e.webp)

## 도전 과제

당신은 새 기술 팟캐스트 "미래 바이트"를 출시하려고 합니다. 첫 번째 에피소드는 최신 AI 혁신에 관한 것이지만, 당신에게 주어진 시간은 24시간뿐입니다:
1. 주제 조사
2. 신뢰할 수 있는 출처 찾기
3. 흥미로운 대본 작성
4. 자연스러운 음성 만들기

**플롯 반전**: 혼자 할 필요가 없습니다. 당신은 곧 모든 것을 도와줄 첫 번째 AI 조수를 만들 것입니다. 이름을 Alex라고 합시다 — 절대 피곤하지 않고 절대 잠들지 않는 당신의 연구 파트너입니다.

## 왜 소형 언어 모델인가? (스포일러: 정말 훌륭합니다)

소형 언어 모델(SLM)을 *당신* 컴퓨터에서 실행되는 개인 AI로 생각하세요. 클라우드도, 월 요금도, 수상한 데이터 공유도 없습니다.

**SLM이 이렇게 🔥인 이유:**
- **🏠 당신의 기기에서 실행**: 노트북, 데스크탑, 심지어 성능 좋은 라즈베리 파이에서도
- **💸 지속 비용 0**: API 비용 때문에 지갑이 얇아질 일 없음
- **🔒 개인정보 우선**: 당신의 데이터는 절대 기기를 벗어나지 않음
- **⚡ 번개처럼 빠름**: 네트워크 지연 없이 즉시 반응
- **🪦 가벼움**: 10억~100억 매개변수 vs 대형 모델의 1000억+

**인기 있는 SLM**: Qwen 3, Phi-4, Gemma 3 (본 워크숍은 Qwen 사용)

## 당신의 도구 키트

### Ollama: 당신의 AI 모델 관리자

[Ollama](https://ollama.com/)는 AI 모델들의 Steam과 같습니다. 간단한 명령어로 모델을 다운로드하고 실행하며 관리할 수 있습니다.

**멋진 점:**
- 한 줄 명령으로 모델 다운로드 및 실행
- Mac, Windows, Linux 전부 지원
- GPU가 있으면 자동 사용
- 메모리 대폭 절약

### Microsoft Agent Framework: 마법이 일어나는 곳

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework)는 AI 에이전트를 만들기 위한 놀이터로, 다음 기능을 갖췄습니다:

- 💬 대화하며 당신과 나눈 내용을 기억
- 🛠️ 맞춤 툴(웹 검색, 날씨 조회 등) 사용
- 🧠 복잡한 문제를 단계별로 생각
- 🤝 팀으로서 다른 에이전트와 협력
- 🔌 다양한 AI 공급자 연결(OpenAI, Ollama, Azure)

**모듈 구성:**
- **에이전트(Agents)**: 특정 작업에 특화된 AI 조수
- **도구(Tools)**: 특수 기능을 부여하는 수단
- **기억(Memory)**: 대화를 잊지 않도록 저장
- **추론(Reasoning)**: 단순 답변이 아니라 생각하도록 교육

## 당신의 훈련 몽타주: 4가지 과제

### 과제 1: 첫 번째 AI 에이전트 만들기

📓 [노트북 열기](../../code/01.BasicAgent/00.BasicAgent-agent.ipynb)

**과제**: 팟캐스트 대본 작성 AI인 Alex 만들기. Alex는 두 진행자가 기술 주제에 관해 토론하는 대화를 생성해야 합니다.

**배울 것:**
- AI 에이전트를 깨우는 방법 (월요일 아침보다 쉬움)
- 성격과 지침 부여하기
- 실제 같은 팟캐스트 대본 생성
- 응답 내용 이해하기

**성공 조건**: Alex가 당신의 "미래 바이트" AI 테스트 에피소드용 대본을 만듭니다! 🎯

### 과제 2: Alex에게 초능력을 주다 (도구!)

📓 [노트북 열기](../../code/01.BasicAgent/01.BasicAgent-tools.ipynb)

**과제**: Alex는 똑똑하지만 오늘 날씨가 어떤지, 지금 몇 시인지 모릅니다. 도구를 줘서 이 문제를 해결해 봅시다!

**배울 것:**
- 맞춤 Python 함수를 "도구"로 만들기
- Alex가 스스로 *언제* 어떤 도구를 쓸지 결정하게 하기
- 문제를 스스로 해결하는 모습을 관찰
- 여러 도구를 조합해 복잡한 과제 수행

**성공 조건**: "도쿄 날씨 어때?"라고 물으면 Alex가 직접 답을 찾습니다! ☁️

### 과제 3: Alex에게 생각하는 법 가르치기

📓 [노트북 열기](../../code/01.BasicAgent/02.BasicAgent-reasoning.ipynb)

**과제**: 문제 해결 과정을 Alex가 보여주게 하기. 당신은 단순 답이 아니라 *어떻게* 생각하는지 보고 싶습니다.

**배울 것:**
- "추론 모드" 활성화 (수학 시간에 풀이 보여주는 것처럼)
- Alex의 단계별 사고 과정 보기
- 사고 체인 프롬프트 이해
- Alex가 혼란스러울 때 디버깅

**성공 조건**: 어려운 수학 문제를 물어보고 Alex가 생각하는 모습을 지켜보기! 🧠

### 과제 4: Alex를 인터넷에 연결하기

📓 [노트북 열기](../../code/01.BasicAgent/03.BasicAgent-websearch.ipynb)

**과제**: Alex의 지식은 유효기간이 있습니다. 실시간 정보를 얻도록 네트워크에 연결해 봅시다!

**배울 것:**
- 맞춤 웹 검색 도구 만들기
- 외부 API 통합
- 네트워크 오류 우아하게 처리
- Alex 학습 데이터 범위 밖 정보 얻기

**성공 조건**: 오늘의 기술 뉴스를 질문하고 최신 결과 획득! 📰

## 당신이 익힐 것

첫 막을 마친 후, 당신은:

- ✅ 자신의 하드웨어에서 AI 모델 실행 (클라우드 필요 없음)
- ✅ 맞춤 개성과 기능이 있는 에이전트 구축
- ✅ 실제 문제 해결용 도구를 에이전트에 제공
- ✅ 에이전트가 자신의 추론 과정을 보여주도록 하기
- ✅ 에이전트를 외부 데이터 소스에 연결
- ✅ 문제 발생 시 디버깅

## 문제 발생 시 (및 해결 방법)🔧

### "Alex가 로드되지 않아요! 메모리가 부족해요!"
**해결 방법**: 컴퓨터가 버거워합니다. 다른 앱을 닫거나 더 작은 모델로 변경해 보세요. 8GB는 최소 요구량입니다.

### "Alex가 너무 느려요"
**해결 방법**: Ollama 설정에서 GPU 가속 켜기. 또는 컨텍스트 창 크기를 줄이세요. 속도 마스터 모드 활성화! 🏎️

### "도구가 작동하지 않아요!"
**해결 방법**: 함수 서명을 꼼꼼히 확인하세요. Alex가 도구 기능을 이해하려면 올바른 타입 힌트가 필요합니다. 명확한 지시를 준다고 생각하세요.
