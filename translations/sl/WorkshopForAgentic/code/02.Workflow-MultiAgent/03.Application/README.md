# Podcast aplikacija

Konzolna aplikacija za ustvarjanje scenarijev za podcaste z uporabo AI agentov.

## Uporaba

```bash
python podcast_app.py
```

## Potek dela

1. **Pozdrav** - Aplikacija pozdravi uporabnika
2. **Vnos teme** - Uporabnik vnese temo za podcast
3. **Iskalni agent** - Išče relevantne informacije
4. **Agent za generiranje scenarija** - Ustvari scenarij za podcast
5. **Pregled** - Uporabnik pregleda in potrdi/prekliče scenarij
6. **Shrani** - Potrjen scenarij se shrani v `podcast.md`

## Zahteve

- Python 3.12+
- agent_framework
- Vse odvisnosti iz 02.WorkflowDevUI
