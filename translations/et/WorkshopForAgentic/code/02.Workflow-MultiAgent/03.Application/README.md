# Podcasti Rakendus

Käsurearakendus podcasti stsenaariumite genereerimiseks AI agentide abil.

## Kasutus

```bash
python podcast_app.py
```

## Töövoog

1. **Tere tulemast** - Rakendus tervitab kasutajat
2. **Teema sisestus** - Kasutaja annab podcasti teema
3. **Otsinguagent** - Otsib asjakohast teavet
4. **Stsenaariumi genereerimise agent** - Loob podcasti stsenaariumi
5. **Ülevaatus** - Kasutaja vaatab stsenaariumi üle ja kinnitab/hülgab selle
6. **Salvesta** - Kinnitatud stsenaarium salvestatakse faili `podcast.md`

## Nõuded

- Python 3.12+
- agent_framework
- Kõik sõltuvused failist 02.WorkflowDevUI
