# تطبيق البودكاست

تطبيق وحدة تحكم لإنشاء نصوص بودكاست باستخدام وكلاء الذكاء الاصطناعي.

## الاستخدام

```bash
python podcast_app.py
```

## سير العمل

1. **الترحيب** - يرحب التطبيق بالمستخدم
2. **إدخال الموضوع** - يقدم المستخدم موضوعًا للبودكاست
3. **وكيل البحث** - يبحث عن معلومات ذات صلة
4. **وكيل إنشاء النص** - ينشئ نص البودكاست
5. **المراجعة** - يراجع المستخدم النص ويوافق عليه/يرفضه
6. **الحفظ** - يتم حفظ النص الموافق عليه في `podcast.md`

## المتطلبات

- بايثون 3.12+
- agent_framework
- جميع التبعيات من 02.WorkflowDevUI
