# پوڈکاسٹ ایپلیکیشن

AI ایجنٹس کا استعمال کرتے ہوئے پوڈکاسٹ اسکرپٹس بنانے کے لیے کنسول ایپلیکیشن۔

## استعمال

```bash
python podcast_app.py
```

## ورک فلو

1. **خوش آمدید** - ایپلیکیشن صارف کو سلام کرتی ہے  
2. **موضوع داخل کریں** - صارف پوڈکاسٹ کے لیے موضوع فراہم کرتا ہے  
3. **سرچ ایجنٹ** - متعلقہ معلومات تلاش کرتا ہے  
4. **اسکرپٹ تیار کرنے والا ایجنٹ** - پوڈکاسٹ اسکرپٹ بناتا ہے  
5. **جائزہ** - صارف اسکرپٹ کا جائزہ لیتا ہے اور منظوری یا مسترد کرتا ہے  
6. **محفوظ کریں** - منظور شدہ اسکرپٹ `podcast.md` میں محفوظ کی جاتی ہے  

## تقاضے

- Python 3.12+  
- agent_framework  
- 02.WorkflowDevUI سے تمام انحصارات
