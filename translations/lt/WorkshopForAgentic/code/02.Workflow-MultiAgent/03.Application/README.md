# Podcast Programėlė

Konsolinė programėlė podcastų scenarijų generavimui naudojant AI agentus.

## Naudojimas

```bash
python podcast_app.py
```

## Darbo eiga

1. **Sveikinimas** - Programėlė pasveikina vartotoją
2. **Temos įvedimas** - Vartotojas pateikia podcasto temą
3. **Paieškos agentas** - Ieško susijusios informacijos
4. **Scenarijaus generavimo agentas** - Kuria podcasto scenarijų
5. **Peržiūra** - Vartotojas peržiūri ir patvirtina/atmėto scenarijų
6. **Išsaugojimas** - Patvirtintas scenarijus išsaugomas į `podcast.md`

## Reikalavimai

- Python 3.12+
- agent_framework
- Visos priklausomybės iš 02.WorkflowDevUI
