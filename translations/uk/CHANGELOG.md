- **Покращення периферійно-оптимізованих SLM фреймворків**:

  - Універсальні шаблони розгортання для Windows, Linux, macOS, iOS та Android
  - Опції апаратного прискорення (CPU, GPU, NPU) з автоматичним визначенням
