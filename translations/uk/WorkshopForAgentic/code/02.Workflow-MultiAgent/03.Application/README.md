# Podcast Application

Консольний додаток для створення сценаріїв подкастів за допомогою AI-агентів.

## Usage

```bash
python podcast_app.py
```

## Workflow

1. **Welcome** - Додаток вітає користувача
2. **Topic Input** - Користувач задає тему подкасту
3. **Search Agent** - Шукає релевантну інформацію
4. **Generate Script Agent** - Створює сценарій подкасту
5. **Review** - Користувач переглядає та схвалює/відхиляє сценарій
6. **Save** - Схвалений сценарій зберігається у `podcast.md`

## Requirements

- Python 3.12+
- agent_framework
- Всі залежності з 02.WorkflowDevUI
