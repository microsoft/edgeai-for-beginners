# 第一幕：認識你的 AI 研究助理 🤖

![lab1](../../../../translated_images/tw/lab1.1dd5210a7d114e2e.webp)

## 挑戰

你正在啟動「未來元件」，你的新科技播客。第一集主題是最新的 AI 突破，但你只有 24 小時來：
1. 研究主題
2. 找到可信的資訊來源
3. 撰寫引人入勝的腳本
4. 讓它聽起來自然流暢

**劇情轉折**：你不必獨自完成這些任務。你即將建造你的第一個 AI 助理，可以幫助你完成所有這些工作。讓我們稱他為 Alex —— 你不需睡眠的勤奮研究夥伴。

## 為什麼選擇小型語言模型？（劇透：它們超讚）

把小型語言模型（SLMs）想像成住在*你*電腦裡的個人 AI。沒有雲端、沒有每月費用、沒有可疑的資料共享。

**SLMs 的優點：**
- **🏠 在你本機運行**：筆電、桌機，甚至是性能不錯的 Raspberry Pi
- **💸 無持續成本**：不用擔心 API 費用吃掉你的午餐錢
- **🔒 隱私至上**：你的資料絕不會離開你的設備
- **⚡ 速度極快**：沒有網路延遲，即時回應
- **🪦 輕量級**：1B-10B 參數，對比大模型的 100B 以上

**熱門 SLMs**：Qwen 3、Phi-4、Gemma 3（本工作坊使用的是 Qwen）

## 你的工具箱

### Ollama：你的 AI 模型管理器

[Ollama](https://ollama.com/) 就像 AI 模型界的 Steam。用簡單指令下載、執行及管理模型。

**特色：**
- 一條指令即可下載並執行任意模型
- 支援 Mac、Windows、Linux
- 有 GPU 的話會自動使用
- 超級節省記憶體

### Microsoft Agent Framework：魔法發生的地方

[Microsoft Agent Framework](https://github.com/microsoft/agent-framework) 是你打造 AI 代理的遊樂場，他們可以：

- 💬 聊天且記得你們的對話內容
- 🛠️ 使用自訂工具（像是網路搜尋或查天氣）
- 🧠 一步步思考複雜問題
- 🤝 與其他代理合作組隊
- 🔌 連接不同 AI 供應商（OpenAI、Ollama、Azure）

**構建基石：**
- **Agents**：有特定任務的 AI 助理
- **Tools**：你賦予的特殊能力
- **Memory**：讓它們不忘對話內容
- **Reasoning**：教它們不只是回應，還能思考

## 出問題了該怎麼辦 🔧

### 「Alex 無法啟動！記憶體不足！」
**解決辦法**：電腦負擔過重。試著關閉其他應用，或換用較小的模型。8GB RAM 是最低標準。

### 「Alex 好慢」
**解決辦法**：在 Ollama 設定中啟用 GPU 加速。或縮小上下文視窗大小。超速模式啟動！🏎️

### 「工具沒反應！」
**解決辦法**：確認你的函數簽名。Alex 需要正確的型別提示才能理解工具功能，就像給它清楚的指令。
