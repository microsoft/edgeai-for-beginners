# Podcast апликација

Конзолна апликација за генерисање пoдкаст скрипти коришћењем AI агената.

## Употреба

```bash
python podcast_app.py
```

## Радни ток

1. **Добродошлица** - Апликација поздравља корисника
2. **Унос теме** - Корисник уноси тему за подкаст
3. **Агент за претрагу** - Тражи релевантне информације
4. **Агент за генерисање скрипте** - Креира скрипту за подкаст
5. **Преглед** - Корисник прегледа и одобрава/одбија скрипту
6. **Снимање** - Одобрена скрипта се чува у `podcast.md`

## Захтеви

- Python 3.12+
- agent_framework
- Све зависности из 02.WorkflowDevUI
